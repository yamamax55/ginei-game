# 預言者＝啓示宗教の創始者 参考設計（EPIC #PRPH）

> 参照元：ムハンマドの生涯（イスラムの預言者・7世紀）。一個人が**啓示を受けたと称し、一代で宗教・共同体・国家を同時に創設**し、迫害からの移住を経て拠点で共同体国家を樹立、政教軍を一身に担い、死後に**後継問題から宗派が分裂**した——という史的構造。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋宗教/カリスマ/継承の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#PRPH` として issue 化する提案。
> 著作権・倫理注意：**特定の現存宗教・実在の預言者を再現・論評・断定・揶揄しない**。固有名・神名・教義・歴史的固有名詞は一切使わず、**「架空の啓示宗教の創始者（使徒型指導者）」をめぐる正統性・建国・継承の構造パターンのみ**を、敬意をもって機構へ翻案する（既存メタEPIC #305／コーラン風 #313 の「風」モチーフ方針を踏襲）。

## 0. 位置づけ — コーラン #313 との関係・なぜ別EPICか

本作には既に宗教ファミリー（親メタ #305）があり、その中の**コーラン風 #313（QUR-1〜6）が「啓典・教義・信徒共同体・審判」＝信仰の"内容"**を扱っている。本EPICはそれと**別軸＝「啓示宗教を立ち上げた創始者という"人物・政治家"の歩み」**を扱う。

| 軸 | 既存（コーラン #313） | 本EPIC（PRPH） |
|---|---|---|
| 何を扱うか | 啓典・教義・ウンマ・聖法・審判（**信仰の内容**） | 創始者の召命・建国・継承分裂（**創始の過程**） |
| 時間 | 確立後の宗教の構造 | 立ち上げの一代（と死の瞬間） |
| 主体 | 信徒共同体 | カリスマ的個人とその後継 |

さらに本EPICは、既存 `SuccessionRules`/`Organization`（#812 カリスマの日常化＝**英雄死後の組織存続**）の**前段＝「カリスマ的創始者が組織を立ち上げる創始フェーズ」**を埋める。#812 は「死後どう続くか」、PRPH は「**どう生まれ、なぜ死の瞬間に分裂するか**」——一対をなす。

| 既存（カリスマ・宗教・継承） | カバー範囲 |
|---|---|
| `Organization`/`SuccessionRules`#812 | 英雄死後の組織存続（制度化 vs 個人カリスマ）——**死後のみ** |
| `ReligionRules`#172-175・コーラン #313 | 改宗圧力・異端・聖戦・啓典・ウンマ |
| `DynastyRules`#867・`Regime` | 王朝サイクル・天命・正統性 |
| `ColonizationRules`#129 | 1惑星→建国（入植） |
| ZUO-5 亡命公子（#1198）・LEAR-6（#1181） | 請求権つき亡命・変名帰参 |
| `OfficeRules`#142・文民統制#145・三権分立#171 | 役職の分立・統制 |

**結論**：PRPH は当プロジェクトに**「カリスマ的個人が無から正統性を創り、一代で宗教＋共同体＋国家を立ち上げ、その死が分裂の起源になる」**という創始ダイナミクスを与える。世襲（#646）でも実力（#169）でもない**第三の正統性の源泉＝啓示的カリスマ**、亡命からの建国、政教軍一体の指導、そして**カリスマの非制度性ゆえの継承危機**。銀英伝の「一代の英雄が興した国は誰が継ぐのか」という問いの、宗教版・最大スケールの答え。

## 1. 役に立つ視点（要約）

1. **正統性は無から創れる**。世襲でも実力でもなく、超越的権威を主張する個人のカリスマが第三の正統性源泉になる。
2. **建国は亡命から始まりうる**。迫害された創始者が新天地へ移り、そこで共同体国家を樹立する（追放が建国の起点）。
3. **法は状況に応じて事後的に形成される**。教義・法が一括でなく出来事に応じて段階的に「下り」、既成事実を聖別する。
4. **創始者は政教軍を一身に担う**。役職分立の対極＝一人格に権威が集中するほど強いが、継承で破綻する。
5. **盟約は信仰を超える**。異なる部族/信仰を含む憲章型の共同体盟約が、初期国家の骨格になる。
6. **カリスマの死は分裂の起源**。後継を制度化できなかった創始者の死が、正統 vs 血統の宗派分裂を生む。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`ReligionRules`#172・コーラン #313・#812・#646・`ColonizationRules` を作り直さない**。PRPH はそれらに**「啓示的カリスマによる創始と継承分裂」**を足し、接続するだけ（additive）。タイクン化回避＝召命・移住建国・後継指名はいずれも高位の決断と創発的帰結。**架空の「風」宗教＝固有名・教義なし**。

### ★★★ 最優先（真の欠落・PRPH の signature）

#### PRPH 啓示的カリスマと正統性の創出（第三の正統性源泉）
- `PropheticAuthorityRules`（static・test-first）＝一個人が**超越的権威を主張**し、信奉者を獲得して正統性を**無から創出**する（世襲#646・実力#169 と並ぶ第三の源泉）。権威は信奉者数×奇跡的成功×言行一致で、矛盾・敗北・偽預言で失墜。`Organization`#812 の**個人カリスマの源泉側**を与える。
- 接続：`Organization`#812（カリスマの蓄積元）・`DynastyRules`#867（正統性）・`ReligionRules`#172（新宗教の発生）・支持#113。

#### PRPH 後継未指名と継承分裂（カリスマの死＝分裂の起源）
- `ProphetSuccessionRules`（static）＝カリスマ的創始者が後継を**制度化できないまま死ぬ**と、正統（指名/合議）vs 血統（縁者）vs 実力（高弟）の継承候補が並立し**宗派分裂**（`ReligionRules` の宗派へ）。制度化投資（#812 `InvestInstitution`）が高いほど分裂を抑える。
- 接続：#812 `SuccessionRules`・`SuccessionLawRules`#646・LEAR（継承危機）・`ReligionRules`（宗派の発生）・SGZ-1（分裂後の多極）。

### ★★ 高（建国・統治形態）

#### PRPH 迫害と移住建国（追放が建国の起点）
- 迫害された創始者集団が**移住して新天地で共同体国家を樹立**＝亡命建国（ヒジュラ型）。出発点の迫害が結束を高め、移住先で `ColonizationRules`/共同体樹立に接続。元の地への帰還が後の目標になる。
- 接続：`ColonizationRules`#129（建国）・ZUO-5 #1198（亡命の請求権）・LEAR-6（帰還）・#160（都市国家化）。

#### PRPH 政教軍一体の指導と段階的啓示
- 創始者が**宗教＋軍事＋立法を一身に担う**（役職分立 #142/#145/#171 の対極＝最大集中）＝強力だが継承で破綻。法・教義は**状況対応で段階的に形成**（事後的正当化・既成事実の聖別）＝一括制定でない柔軟さと、後の解釈論争の種。
- 接続：`OfficeRules`#142/文民統制#145/三権分立#171（の対極係数）・コーラン #313（啓典形成の過程側）・VEN-3 文言主義（後の解釈論争）。

### ★ 中（盟約・拡張・lore）

#### PRPH 盟約共同体（信仰を超える憲章型同盟）
- 異なる部族/信仰を含む**憲章型の共同体盟約**（メディナ憲章型）が初期国家の骨格＝信仰一致でなく相互防衛/裁定の政治契約。ZUO 会盟の**創始国家版**。
- 接続：ZUO-2 会盟（#1195）・`DiplomacyRules`・VEN 契約・`CovenantRules`。

#### PRPH（lore）啓示宗教の爆発的拡張・認識データ
- 宗教的動員による**一代～二代の急速な征服拡張**（結束×大義×機会の窓）＝爆発的だが制度が追いつかず後に分裂。世界観の開示データ（カリスマ建国の栄光と脆さ）。
- 接続：`ReligionRules`（聖戦動員）・`DisclosureLedger`（FND-4）・SGZ-1（拡張後の多極化）。コード新設は最小。

### ❌ 不採用（重複・倫理・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 啓典・教義・ウンマ・審判 | **コーラン #313（QUR-1〜6）がカバー**。PRPH は創始者の人物軸のみ |
| 宗教システムそのもの | **`ReligionRules`#172 がカバー**。PRPH は新宗教の発生と分裂を接続 |
| 特定の現存宗教・実在預言者の再現/論評/揶揄 | 倫理方針（#305/#313 踏襲）。**架空の「風」宗教の構造のみ** |
| カリスマの日常化（死後の存続）そのもの | **#812 がカバー**。PRPH は創始フェーズ（前段）を足す |
| 宗教実践（祈り/施し/巡礼）の数値ファーム | **QUR-6 がカバー**＋タイクン化回避 |

### 推奨着手順
`PRPH-1 → PRPH-2`（カリスマの創出＋死による分裂＝signature・#812 と一対）→ `PRPH-3`（移住建国）→ `PRPH-4`（政教軍一体）→ 残り（PRPH-5/6）。
