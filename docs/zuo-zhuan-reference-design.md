# 春秋左氏伝 参考設計（EPIC #ZUO）

> 参照元：『春秋左氏伝』（春秋時代の編年史・左丘明伝）。**周王室の権威が空洞化**した分裂期に、最強諸侯が**「尊王攘夷」を掲げて諸侯を束ねる覇者**となり、**会盟（盟誓）**の儀礼で同盟を結び、**礼と名分**が国の威信と正統性を測る——下剋上の萌芽（世卿が君権を侵食）と亡命公子の復位を描く。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋外交・継承・正統性の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#ZUO` として issue 化する提案。
> 著作権注意：固有名・文章・人物は流用せず、**外交儀礼・覇権・正統性の構造パターンのみ**を参考にする。

## 0. なぜ「春秋左氏伝」が本システムに役立つか

当プロジェクトは外交・継承・正統性の純ロジックを保有している（[CLAUDE.md] 参照）：

| 既存（外交・継承・正統性） | カバー範囲 |
|---|---|
| `DiplomacyRules`/`TreatyRules`（DIP-1/2 #190/#191） | 二国間の外交状態・条約・opinion |
| SGZ-1 多極均衡（#1103） | 最強台頭→弱小連衡（バランサー側） |
| `DynastyRules`#867・`Regime` | 王朝サイクル・天命・正統性 |
| `SuccessionLawRules`#646・LEAR（#1175） | 継承法・生前退位・亡命忠臣の帰参 |
| `FeudalRules`/`Fief`#168 | 封建・徴募・反乱リスク |
| `PowerRules`#164 | 実権/傀儡/影の支配者 |
| VEN（#1166）契約・裁判 | 私人間の約束の執行 |

**しかし、春秋左氏伝が固有に描く「分裂期の秩序維持装置」**は無い：

| 春秋左氏伝が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **覇者（覇権）**＝最強諸侯が空洞化した王室を**奉じて**諸侯を束ねる（尊王攘夷） | DIP は二国間。SGZ は連衡（最強を**抑える**）。**最強が正統を奉じて束ねるヘゲモン秩序**が無い |
| **会盟（盟誓）**＝多国間の同盟儀礼（歃血・誓約・違背の罰） | `TreatyRules` は二国間条約。**多国間の盟約＋盟主＋違背者の制裁**が無い |
| **礼と名分**＝外交儀礼の作法が国の威信・正統性を測る | opinion はあるが、**儀礼の正否が威信に直結する**回路が無い |
| **春秋の筆法＝大義名分の言説闘争**（誰が「正しい戦争」か） | `WarGoalRules` は厭戦/講和。**名分（征伐の正当性ラベル）が支持と参戦動機を左右**しない |
| **世卿世禄→下剋上**（家臣の家が代を重ね君権を侵食＝三桓/六卿） | #168 は反乱リスク。**重臣の家が世代を超えて実権を簒奪する漸進過程**が無い |
| **亡命公子の復位**＝他国に逃れた王族が請求権を持ち帰国して君主に | LEAR/Captivity は亡命の退場/投靠。**請求権つき亡命王族→外国の支援で復位**が無い |
| **小国の両属生存**＝強国の間で朝貢を使い分け生き延びる | #160 都市国家・属国はあるが、**両属（二重朝貢）の綱渡り**が無い |

**結論**：春秋左氏伝は当プロジェクトに**「中央権威が壊れた後、武力一辺倒でない秩序がどう保たれるか」**を与える。SGZ-1（最強を抑える連衡）の**対**＝覇者（最強が正統を奉じて束ねる）、二国間外交（DIP）の**上**＝多国間の会盟、武力（WarGoal）の**前**＝名分の言説闘争。**王道で銀河を統べる**本作のテーマに、**「徳と礼と名分による覇権」という非武力の秩序軸**を加える。

## 1. 役に立つ視点（要約）

1. **空位の権威は奉じる者に力を与える**。空洞化した王室を「奉じる」最強諸侯が覇者になる＝正統性を独占せず**借りて**号令する。
2. **同盟は儀礼で固まり、違背は制裁される**。会盟＝多国間の盟約は歃血の儀礼で結ばれ、破れば諸侯連合の討伐対象になる。
3. **礼は力の代替であり威信の通貨**。正しい儀礼は戦わずに格を示し、非礼は開戦事由になる。
4. **戦争には名分が要る**。同じ侵攻でも「征伐（正義）」か「侵略（不義）」かで支持と参戦が変わる＝言説が戦力。
5. **臣下の家は静かに主君を超える**。世襲の重臣が代を重ねて実権を吸い上げ、ある日主家を乗り越える（下剋上）。
6. **亡命王族は時限爆弾**。請求権を持つ公子を匿うことは、相手国への内政干渉カードになる。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**DIP#190/191・SGZ-1・`DynastyRules`・#168・#164 を作り直さない**。ZUO はそれらに**「分裂期の秩序装置」（覇権・会盟・名分・下剋上・亡命王族）**を足し、接続するだけ（additive）。タイクン化回避＝覇権争い・会盟・名分はいずれも高位の決断と創発的帰結。

### ★★★ 最優先（真の欠落・ZUO の signature）

#### ZUO 覇者システム（空洞化した権威を奉じる覇権）
- `HegemonyRules`（static・test-first）＝中央権威（`Regime`/盟主）が衰えた状況で、最強勢力が**正統を奉じて**覇者を称し、諸侯へ号令（参戦要請・貢納・裁定）できる。覇者の権威は**実力×正統の奉戴×礼の遵守**で、専横（正統を僭称）すると失墜。SGZ-1（最強を抑える連衡）と**動的に拮抗**＝覇者が専横すれば連衡が起きる。
- 接続：`DynastyRules`#867（奉じる対象＝空洞化した王権）・SGZ-1 #1103・`PowerRules`#164・`DiplomacyRules`。

#### ZUO 会盟（多国間の盟約・盟主・違背の制裁）
- `CovenantRules`（static）＝**多国間の盟約**（二国間 `TreatyRules` の上位）：盟主＋参加国＋誓約条項＋違背罰。違背者は**諸侯連合の討伐対象**（参戦義務の発火）。会盟の招集力＝覇者の威信。
- 接続：`TreatyRules`（DIP-2＝二国間はそちら・住み分け）・`HegemonyRules`・`WarGoalRules`（討伐の名分）。

### ★★ 高（正統性・言説・秩序侵食）

#### ZUO 名分と春秋の筆法（征伐の正当性ラベル＝言説が戦力）
- 戦争・行動に**名分**（正当性ラベル：征伐/自衛/会盟義務/簒奪討伐 ↔ 侵略/下剋上/盟約違背）を付与。名分の正否が**自国支持・第三国の参戦/中立・戦後正統性**を左右＝言説が外交資源。`WarGoalRules` の casus belli に正統性の歯を入れる。
- 接続：`WarGoalRules`（DIP-3）・`DynastyRules`（正統性）・`JusticeRules`#918（正義観で名分の是認が変わる）・支持#113。

#### ZUO 世卿世禄と下剋上（重臣の家が君権を侵食）
- `RetainerHouseRules`（static）＝世襲の重臣（門閥）が**代を重ねて実権・封・人脈を蓄積**し、閾値を超えると主家を**侵食→簒奪**（三桓/六卿/下剋上の漸進過程）。君主は分封で報いるほど将来の侵食を育てる（封建のジレンマ）。
- 接続：`FeudalRules`#168・`PowerRules`#164（`IsShadowRuler` の家版）・LEAR-1（生前退位の二重権力と同系）・継承#152（家の世代継承）。

### ★ 中（亡命・小国・lore）

#### ZUO 亡命公子と請求権外交
- 請求権を持つ**亡命王族**（継承争いに敗れて他国へ）を匿う＝相手国への**内政干渉カード**（支援して帰国・復位させ傀儡化、または交渉の質草）。LEAR/Captivity の亡命を**請求権つき**へ拡張。
- 接続：`SuccessionLawRules`#646（請求権）・LEAR-6（変名帰参）・`CaptivityRules`・`DiplomacyRules`。

#### ZUO 小国の両属生存
- 弱小勢力が強国の間で**両属（二重朝貢）**して生き延びる綱渡り＝どちらにも貢ぐが、覇権交代や踏み絵で破綻。都市国家#160・属国の生存戦略。
- 接続：#160・`DiplomacyRules`（属国状態）・`CovenantRules`（会盟への両属）。

#### ZUO（lore）礼の威信・卜占の解釈
- 「礼を失えば国が乱れる」「卜占は天意の言説」＝礼の遵守が威信通貨、卜占が決断の正統化に使われる開示データ。礼の正否は会盟・外交の威信係数に薄く接続。
- 接続：`DisclosureLedger`（FND-4）・`EventEngine`・会盟/外交の威信。コード新設は最小。

### ❌ 不採用（重複・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 二国間条約そのもの | **`TreatyRules`（DIP-2）がカバー**。会盟は多国間の上位として足す |
| 王朝サイクル・天命そのもの | **`DynastyRules`#867 がカバー**。覇者は天命の奉戴として接続 |
| 封建の徴募/反乱そのもの | **`FeudalRules`#168 がカバー**。下剋上は世代侵食の過程だけ足す |
| 編年史UI・春秋経文の自動生成 | 史書編纂は別系統（クロニクル候補）。ここは名分ラベルのみ |
| 儀礼ミニゲーム化 | タイクン化。礼は威信係数として背景的に効かせる |
