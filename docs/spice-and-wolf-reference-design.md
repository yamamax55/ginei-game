# 狼と香辛料 参考設計（EPIC #SAW）

> 参照元：『狼と香辛料』（支倉凍砂）。中世欧州風世界の**行商人クラフト・ロレンスと豊穣の狼神ホロ**の旅。
> 戦争でなく**商売・通貨・情報**で世界が動く——「賢狼」の知恵と行商の駆け引きを描く経済譚。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に巨大な経済純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#SAW` として issue 化する提案。
> 著作権注意：固有名・文章・キャラクター・固有設定は流用せず、**経済メカニクス／世界観の構造パターンのみ**を参考にする。

## 0. なぜ「狼と香辛料」が本システムに役立つか

当プロジェクトは経済の**マクロ純ロジックを大量に保有**している（[CLAUDE.md] 参照）：

| 既存（マクロ・抽象） | カバー範囲 |
|---|---|
| `MarketRules`/`Good`/`Market`（#179-182） | 市場ごとの需給均衡価格・生活水準→支持 |
| `BankRules`/`Bank`（#186） | 信用創造・取付け・債務超過 |
| `FiscalRules`/`FiscalState`（#161/162） | 国債/金利/為替・税/社会保障 |
| `StockMarketRules`/`Company`（#185） | 株価・配当・暴落リスク |
| 商社/企業バリューチェーン（FRM #1022） | 調達+物流+金融+販売の結節点（フェザーン=商社国家#160） |
| `ReligionRules`（#172-175） | 改宗圧力・異端・聖戦 |

**しかし、これらは「国家・市場・銀行」という抽象主体のマクロ均衡**であり、狼と香辛料が固有に描く以下が**欠けている**：

| 狼と香辛料が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **通貨改鋳・コインの品位（銀含有量）** | `FiscalRules.ExchangeRateFactor` は法定通貨的なマクロ為替。**正貨の品位・改鋳政策・改鋳投機**（トレニー銀貨アーク）が無い |
| **情報の非対称＝情報を制する者が商売を制す** | 市場は全員同条件で均衡。**情報優位の商人が儲ける／噂で相場が動く**回路が無い |
| **空間裁定（安く仕入れ高く売る）＝行商の核** | `Market` は単一市場の均衡。**星系間の価格差をトレードで埋める**裁定の動学が薄い |
| **買い占め・コーナリング・バブル**（商品の） | 株の暴落はあるが**商品の意図的コーナリング**が無い |
| **教会の経済力＋旧神/異教の経済的退場** | `ReligionRules` は信仰均衡。**宗教×経済の融合**（十分の一税・贖宥・交易支配・異教地の経済萎縮）が無い |
| **収穫の神＝豊凶が穀物相場を揺らす** | 豊凶ショック→穀物相場→民心の連鎖が薄い |
| **ネームド商人の帳簿・信用・破産** | `Bank` はマクロ。**個人商人のレバレッジ・担保・破産**（人間スケール）が無い |

**結論**：狼と香辛料は当プロジェクトのマクロ経済に**「地上の商人視点」という人間スケールの顔**と、**①通貨改鋳 ②情報の非対称 ③信仰×経済 ④ネームド商人の信用**という4つの欠落軸を与える。そして**フェザーン（#160 商社国家）にこそ最も似合うテクスチャ**を供給する。

## 1. 役に立つ視点（要約）

狼と香辛料の世界観を、**本システムに効く形**で1行ずつ：

1. **「戦争より商売」＝経済が歴史の真のエンジン**。剣で終わらぬ対立を交易が終わらせる。→ 当プロジェクトの**王道路線**（武力でなく和で銀河を統べる）と完全に共鳴。フェザーンの戦略的意義を裏打ち。
2. **通貨は信用の塊で、品位を弄ると国が動く**。改鋳（混ぜ物）と改鋳投機＝戦わずに敵国経済を揺さぶる武器。→ 通商破壊#95・経済制裁(PRC-8)と並ぶ**第三の経済戦**。
3. **情報を制する者が商売を制す**。どの町が何を欲し、どこが改鋳するかを先に知る者が勝つ。→ 諜報`EspionageRules`に**経済的価値**を与える。
4. **市場は均衡へ向かうが、その途上に儲けがある**。裁定・コーナリング・バブルは均衡の隙間。→ マクロ均衡に**動学（時間差・人為）の遊び**を足す。
5. **信仰と経済は不可分**。教会は最大の経済主体（税・贖宥・交易網）、旧神の退場は近代化＝市場合理性の浸透。→ `ReligionRules`に経済の歯を入れ、世界観EPIC（啓蒙/ニーチェ「神は死んだ」/秘史）に接続。
6. **賢者（ホロ）の長寿＝制度を超えて続くものと滅ぶもの**。不死者の孤独は、英雄死後の組織存続（#812 カリスマの日常化）と同じ問い。→ 開示エンジンのlore。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**市場#179/銀行#186/商社FRM#1022/SCM#982 を作り直さない**。SAW はそれらに**欠落軸を足し、接続する**だけ（additive）。

### ★★★ 最優先（真の欠落・狼と香辛料の signature）

#### SAW 通貨改鋳・品位・改鋳投機（トレニー銀貨アーク）
- **正貨モデル**：コインに**品位（銀含有量）**。改鋳（混ぜ物で品位を下げる）＝短期に鋳造益（シニョリッジ）を得るが信用が落ち実質為替が下落。
- **改鋳投機**：品位改定の**噂**を先に掴んだ者が旧貨/新貨を売買して儲ける。情報（SAW-4）と直結。
- 接続：`FiscalRules.ExchangeRateFactor` の**micro/正貨版**＝新 `CoinageRules`/`Specie`（純ロジック・test-first）。**戦わずに敵経済を揺さぶる経済戦**＝通商破壊#95・制裁PRC-8 と三層。

#### SAW 情報の非対称と風説の相場（情報を制する者が商売を制す）
- 市場価格に**情報優位**を導入：先に知る商人/勢力が裁定で儲け、**噂（真偽不明）で相場が動く**。
- 接続：`EspionageRules`（情報収集に**経済的ペイオフ**を付与）×`MarketRules`×`EventEngine`（風説イベント）。**諜報が儲けに直結**＝情報戦の動機づけ。

### ★★ 高（マクロ均衡に動学の遊びを足す）

#### SAW 空間裁定（星系間の価格差をトレードで埋める）
- 銀河グラフ上で**安い星系で仕入れ高い星系で売る**。裁定が**価格差を収束**させる（裁定するほど儲けが減る自己消滅）。
- 接続：`MarketRules` を `GalaxyMap` 上で連結＝**価格収束ルール**。商社FRM#1022/調達PRC#1002 の取引動機に。

#### SAW 買い占め・投機・バブル（商品コーナリング）
- ある商品を**買い占めて価格を吊り上げる**／投機で**バブル→暴落**。
- 接続：`StockMarketRules.CrashRisk` の**商品版**＝`MarketRules` にコーナリング/バブル動学を追加。

#### SAW ネームド商人の信用・為替手形・破産（人間スケールの帳簿）
- ネームド商人（`Person`/`ICharacter`）が**帳簿・担保・レバレッジ**を持ち、読み違えると**破産**（ロレンスの危機）。
- 接続：`BankRules`（信用）×資産所有権OWN#1032×会計#973（B/S）。マクロ銀行に**個人スケールの顔**を与える。

#### SAW 教会の経済力と異教/旧神の経済的圧迫
- 教会＝最大の経済主体（**十分の一税・贖宥状・交易網支配**）。**異教地の経済萎縮**（旧神＝ホロの退場）＝改宗圧力の経済的帰結。
- 接続：`ReligionRules`（改宗圧力/異端）×経済＝**宗教×経済の融合ルール**。世界観EPIC（啓蒙/秘史）へ。

### ★ 中（豊凶・行商の物理・世界観lore）

#### SAW 収穫信仰と豊凶サイクル（収穫の神ホロ）
- **豊作保証の信仰**→豊凶が**穀物相場をショック**→民心（`GovernanceRules`）。旧神の退場で豊凶が荒れる、という信仰×農本経済。
- 接続：`ReligionRules`×`GovernanceRules`×穀物`Market`。

#### SAW（lore）世界観の開示データ
- 「経済が戦争を終わらせる（王道）」「旧神の退場＝近代化」「不死者の孤独＝制度を超えて続くもの」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。CCX-6（世界観codex退避）方針に一貫。

### ❌ 不採用（重複・既存で十分）

| 不採用 | 理由 |
|---|---|
| 商社/取引先ネットワーク/契約 | **FRM#1022・PRC#1002 が既にカバー**。SAW は作り直さず接続 |
| 多段加工/BOM/サプライチェーン | **SCM#982 がカバー** |
| 行商の隊商物理（腐敗/関税/路上リスク）を新規実装 | **SupplyRules/通商#94/SCM** に薄く接続するだけ（新EPIC化しない） |
| 株式市場そのもの | **#185 がカバー**。SAW は商品コーナリングのみ足す |
