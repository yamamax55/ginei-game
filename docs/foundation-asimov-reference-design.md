# ファウンデーション 参考設計（EPIC #FOUN）

> 参照元：アイザック・アシモフ『ファウンデーション』シリーズ。**心理歴史学**（psychohistory）で銀河帝国の崩壊を統計的に予測し、暗黒時代を3万年から1千年へ**短縮**する計画を描くSF。
> 個人の行動は予測不能だが、十分大きな人口集団の振る舞いは**統計的に予測できる**——というマクロ予測と、それを攪乱する個（ミュール＝予測外の変異原）の対立が核。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に巨大な社会・政治シミュ純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#FOUN` として issue 化する提案。
> 著作権注意：固有名・文章・キャラクター・固有設定（心理歴史学の数式、セルダン、ファウンデーション組織、ミュール等の固有名）は流用せず、**予測メカニクス／世界観の構造パターンのみ**を参考にする。

## 0. なぜ「ファウンデーション」が本システムに役立つか

当プロジェクトは社会・政治の**マクロ状態を大量に保有**している（[CLAUDE.md] 参照）。それらは「**今の状態**」を精密にシミュレートする：

| 既存（マクロ状態シミュ） | カバー範囲 |
|---|---|
| `CampaignState`/`CampaignRules`（最上層統合） | 盤面×社会の現在状態を毎Tick進める |
| `FactionState`/`FactionStateRules` | 正統性/合意/結束/希望の合成・`IsCollapsing`（崩壊**判定**） |
| `DynastyRules`/`Regime`（孔子・易姓革命） | 腐敗→正統性低下→天命喪失の王朝サイクル |
| `GovernanceRules`/`Province` | 安定度の目標値収束・反乱リスク |
| `ResearchRules`/`ResearchProject`（#123-127） | 研究進捗・政体偏り |
| `DiplomacyRules`/`DiplomacyState`（#189） | 外交状態×opinion |
| `EspionageRules`/`SpyNetwork` | 諜報の成功/情報獲得（決定論roll） |
| `DisclosureRules`/`DisclosureLedger`（FND-4） | 条件→開示の物語背骨 |
| `Person`/`LoyaltyRules`（関ヶ原） | 個人の役割・寝返り |

**しかし、これらはすべて「現在状態を進める」シミュレーション**であり、ファウンデーションが固有に描く以下が**欠けている**：

| ファウンデーションが固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **心理歴史学＝マクロ状態から未来を統計予測する** | `IsCollapsing` は今が崩壊「中」かの真偽判定。**「この勢力はNヶ月後に崩壊する」と先に予報する**予測エンジンが無い。状態は進むが**未来を読まない** |
| **個人は予測不能・集団は予測可能（大数の法則）** | `LoyaltyRules`/`EspionageRules` は個のroll。**「個の攪乱は予測を外す／集団規模が大きいほど予測が当たる」という予測信頼度の構造**が無い |
| **予測を外す変異原（ミュール）＝ブラックスワン** | イベントは条件発火（`EventRules`）。**「予測モデルそのものを破綻させる規格外の個」**＝予測外イベントの系統が無い |
| **帝国崩壊の短縮計画＝介入で暗黒時代を最小化する** | 崩壊は起きるが、**「崩壊は不可避と受け入れ、その後の暗黒時代を短縮する」長期介入計画**の枠組みが無い |
| **周期的危機＝予定された分岐点で選択を迫られる** | `EventEngine` は重み付き抽選。**予測が事前に告げる「この時期に必ず岐路が来る」周期的クライシス**が無い |
| **隠れた第二の計画＝予測を秘匿し誘導する（心理操作の超長期戦）** | 諜報は短期情報戦。**予測を伏せて社会を望む軌道へ静かに誘導する**メタ戦略が無い |
| **予測の自己成就/自己破壊（観測が結果を変える）** | 予測結果は誰も見ない。**予報を公開すると人々が反応して予測がずれる**（反射性）回路が無い |

**結論**：ファウンデーションは当プロジェクトの「現在を進めるシミュ」に**「未来を読む層」という新しい軸**を与える。具体的には**①マクロ崩壊予測 ②予測信頼度（集団規模×個の攪乱）③予測外の変異原 ④暗黒時代短縮の長期介入計画 ⑤周期的クライシス**という5つの欠落軸。既存の `FactionState`/`CampaignState`/`DynastyRules` を**入力**として読む**予測レイヤー**であり、既存を作り直さず**その上に乗る**（additive）。

## 1. 役に立つ視点（要約）

ファウンデーションの世界観を、**本システムに効く形**で1行ずつ：

1. **心理歴史学＝マクロ状態から未来の軌道を統計予測する**。崩壊は判定でなく**予報**できる。→ `FactionStateRules.IsCollapsing` を「現状判定」から「**Nステップ先の崩壊確率**」へ拡張する予測エンジン。
2. **個は予測不能、集団は予測可能**。人口が大きいほど予測は当たり、少数勢力ほど個の攪乱で外れる。→ `Person`/`Population`（#153）の規模が**予測信頼度**を決める構造。
3. **予測を外す変異原（規格外の個）＝ブラックスワン**。モデルの前提を壊す英雄/異常が予測を無効化する。→ `EventEngine`に**「予測破綻イベント」**の系統を追加（傑物#544・主人公#735と共鳴）。
4. **崩壊は不可避——ならば暗黒時代を短縮せよ**。崩壊を止めるでなく、**崩壊後の回復を最小化する長期介入**。→ `DynastyRules`の易姓革命（崩壊→再起動）に**「介入で再起動を早める」計画レイヤー**を接続。
5. **周期的危機＝予定された分岐点**。予測が「この時期に必ず岐路が来る」と告げ、選択を迫る。→ `EventEngine`に**暦境界で発火する予報駆動クライシス**（`CalendarDispatcher`相乗り）。
6. **隠れた計画＝予測を秘匿し社会を静かに誘導する**。→ `DisclosureLedger`（予報の段階開示）×`EspionageRules`（誘導工作）。武力でなく予測で銀河を統べる**王道路線**の極北。
7. **予測の反射性＝予報を公開すると結果が変わる**。観測が対象を変える社会科学の宿命。→ 予報の公開可否そのものが戦略選択になる。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`FactionState`/`CampaignState`/`DynastyRules`/`EventEngine` を作り直さない**。FOUN はそれらを**入力として読み、未来予測という新しい層を上に乗せる**だけ（additive）。予測は read-only に既存状態を観測する＝既存シミュを後退させない。

### ★★★ 最優先（真の欠落・ファウンデーションの signature）

#### FOUN 心理歴史学＝マクロ崩壊予測エンジン
- **予測モデル**：`FactionState`（正統性/合意/結束/希望＋腐敗ドリフト）の**現在の変化率**から、Nステップ先の `Stability`/`IsCollapsing` を外挿し、**崩壊までの予測時間（ETA）と崩壊確率**を返す。`DynastyRules` の腐敗→正統性低下のトレンドを延長して天命喪失時期を予報する。
- 純粋な前向き積分（既存 `Tick` を内部で空回しする/トレンド外挿する）で、**実状態は一切変えない read-only 予測**。
- 接続：新 `PsychohistoryRules`/`Forecast`（純ロジック・test-first）。入力＝`FactionState`/`Regime`/`CampaignState`。`FactionStateRules.IsCollapsing`（現状判定）の**未来版**。

#### FOUN 予測信頼度＝集団規模×個の攪乱（大数の法則）
- **信頼度モデル**：予測の確度は**対象集団の規模**（`Population` の総人口・勢力数・星系数）が大きいほど高く、少数勢力ほど低い。さらに**個の攪乱要因**（突出した能力の `Person`／低結束）が信頼度を下げる。
- 予測は点でなく**幅（信頼区間）**で返す＝「68%の確率でN±M月後に崩壊」。
- 接続：`PsychohistoryRules.Confidence(population, factionState)`。`Population`/`DemographicsRules`（#153）×`Person`（突出個の検出）。

### ★★ 高（予測を攪乱する／介入する層）

#### FOUN 予測外の変異原（ブラックスワン・規格外の個）
- 予測モデルの**前提を壊す規格外の個/異常事象**＝予測を一時的に無効化する。発生すると当該勢力の `Forecast` 信頼度が急落し、予報が外れる。
- 既存の傑物（自律分散 `AutonomyRules.IsFunctional` の高能力個）・主人公（#735）・英雄登場イベントと共鳴。
- 接続：`EventEngine`に**「予測破綻」カテゴリ**のイベント定義（`GameEventDef`）＋`PsychohistoryRules.DisruptedBy(person)`（攪乱度判定）。新 state は最小（既存 `Person`/`EventEngine` を読む）。

#### FOUN 暗黒時代短縮の長期介入計画
- **崩壊は不可避と受容**し、崩壊後の**回復（再起動）を早める介入**を計画として持つ。`DynastyRules.Revolution`（易姓革命＝王朝交代で再起動）の**回復速度**に介入投資が効く。
- 「いま崩壊を止める（短期・高コスト・先送り）」vs「崩壊を受け入れ回復を仕込む（長期・低コスト）」の**高位の決断**＝タイクン化回避の核。
- 接続：新 `InterventionPlanRules`/`RecoveryPlan`（純ロジック・test-first）。`DynastyRules`（崩壊→再起動）×`PsychohistoryRules`（予報）。

#### FOUN 周期的クライシス（予報駆動の予定された分岐点）
- 予報が「この暦境界で必ず岐路が来る」と事前に告げ、到来時に**選択を迫る**クライシスを発火。重み付き抽選でなく**予測が日時を指定する**点が既存イベントと違う。
- 接続：`EventEngine`×`CalendarDispatcher`（暦境界Tick）×`PsychohistoryRules`（クライシス時期の予報）。`EventRules` に予報駆動の発火経路を**追加**（既存の重み抽選は不変）。

### ★ 中（予測の社会性・世界観lore）

#### FOUN 予測の秘匿と段階開示（隠れた計画・反射性）
- 予報を**公開するか秘匿するか**が戦略選択：公開すると人々が反応して予測がずれる（自己成就/自己破壊＝反射性）。秘匿して静かに誘導するのが「隠れた計画」。
- 接続：**コード新設を最小化** ＝ `DisclosureLedger`（FND-4）への予報の段階開示データ＋`PsychohistoryRules.ReflexiveShift(published)`（公開で予測がずれる補正）。`EspionageRules`（誘導工作）に接続。

#### FOUN（lore）世界観の開示データ
- 「個は不可能でも歴史は予測できる」「崩壊は不可避だが暗黒時代は短縮できる」「規格外の個が必然を覆す」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。世界観codex退避方針（CCX-6）に一貫。

### ❌ 不採用（重複・既存で十分・スコープ外）

| 不採用 | 理由 |
|---|---|
| 崩壊の現状判定そのもの | **`FactionStateRules.IsCollapsing`・`DynastyRules.MandateLost` が既にカバー**。FOUN は作り直さず「未来版」を上に乗せる |
| 王朝サイクル/易姓革命の再起動 | **`DynastyRules`（#867）がカバー**。FOUN は回復**速度**への介入のみ接続 |
| 諜報・情報収集の基盤 | **`EspionageRules`/`SpyNetwork` がカバー**。FOUN は誘導工作として接続のみ |
| 人口動態（ボーナス/オーナス） | **`DemographicsRules`/`Population`（#153）がカバー**。FOUN は規模を予測信頼度の入力に読むだけ |
| イベントエンジン基盤 | **`EventEngine`/`EventRules`（#116）がカバー**。FOUN は予測破綻/予報駆動の発火経路を**追加**するのみ |
| 銀河規模の「未来年表」UI/可視化を新規構築 | 予測は **観測層**（`CoreStateInspector`）に追従表示で足りる。専用UIは新EPIC化しない |
| 個人の精神操作/メンタリック等の超能力描写 | 固有設定（著作権）かつタイクン化＝不採用。攪乱は**集団統計の予測信頼度**として抽象化 |

### 推奨着手順
`FOUN-1`（崩壊予測エンジン＝最も固有で欠落の大きい signature）→ `FOUN-2`（予測信頼度＝集団規模で当たり外れが決まる構造）→ `FOUN-3`（変異原＝予報を外す攪乱）→ `FOUN-4`（暗黒時代短縮＝崩壊受容の長期介入＝高位の決断）→ `FOUN-5`（周期的クライシス）→ `FOUN-6`（盤面・観測層へ配線）→ `FOUN-7`（秘匿・反射性・lore）。

> いずれも既存の社会・政治シミュを**後退させず、read-only 予測層として上に乗せる** additive 設計。`CampaignState`（最上層統合）に最も効き、武力でなく予測で銀河を統べる王道路線の極北を与える。
