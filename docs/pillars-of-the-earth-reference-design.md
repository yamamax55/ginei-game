# 大聖堂 参考設計（EPIC #PIL）

> 参照元：『大聖堂』（ケン・フォレット／The Pillars of the Earth）。12世紀イングランド、**一つの大聖堂の建設に生涯を捧げた人々**の群像劇。
> 石工の棟梁・伯爵令嬢・野心の大貴族・理想の修道院長——交わるはずのない運命が「巨大プロジェクト」を背骨に絡み合う。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋群像・政治・経済の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#PIL` として issue 化する提案。
> 著作権注意：固有名・文章・キャラクター・固有設定は流用せず、**メカニクス／物語構造のパターンのみ**を参考にする。

## 0. なぜ「大聖堂」が本システムに役立つか

当プロジェクトは群像・政治・経済の**純ロジックを大量に保有**している（[CLAUDE.md] 参照）：

| 既存（関連する土台） | カバー範囲 |
|---|---|
| `Shipyard`/`ShipyardRules`（#884） | 建艦キュー・生産力連動（**数日〜数月スケール**） |
| `GameClock`/`CalendarDispatcher`（TIME #946） | 統一時間・暦の日/月/年境界フック＝**数十年を刻む暦は既にある** |
| `Organization`/`SuccessionRules`（#812） | 英雄死後の**組織**存続（制度化 vs 属人） |
| `LoyaltyRules`（#817 関ヶ原） | 忠誠/調略/趨勢→寝返りカスケード（**会戦・戦役スケール**） |
| `SuccessionLawRules`（PDX-1 #646） | 継承法→**継承危機リスク値**（リスク算出どまり） |
| `MagnaCartaRules`/`Charter`（#624） | 憲章＝**権力への一般的制約**（個別利権ではない） |
| `ResearchRules`（#123-127・未配線） | 研究出力・政体偏り（**抽象的な研究点**） |
| `Person`/`CareerPipelineRules`（LIFE-5/6/7） | 武/官/技の三系統・席次・学閥 |
| `FiscalRules`（#161/162） | 国債/金利/税/社会保障（単年度フロー中心） |

**しかし、これらは個々のモジュールであり、大聖堂が固有に描く以下が欠けている**：

| 大聖堂が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **世代を超えるメガプロジェクト＝物語の背骨** | 造船#884 は数月スケール。**数十年・指導者の死を跨ぐ国家的大事業**（要塞建設・大シップヤード・遷都）が無い。要塞 #76-78/C-7 は「在るもの」で「**建てるもの**」ではない |
| **急げば崩れる（工期と品質のトレードオフ）** | 建設に**リスク動学**（崩落・火災・襲撃で進捗後退）が無い |
| **技術は人に宿る**（棟梁・職人・新工法の伝来） | `ResearchRules` は抽象点。**ネームド技術者が工法を保持し、人の移動＝技術伝播、人の死＝技術喪失**の回路が無い |
| **利権の争奪**（採石権・市場開設権・聖俗の管轄争い） | `Charter` は一般的制約。**個別の特許状＝授与・取消・争奪できる利権資産**が無い |
| **大事業が町を育てる**（聖堂→市→都市） | プロジェクト→人口流入→市場成立→`Province` 成長の連鎖が無い |
| **継承戦争＝国家規模の旗幟**（The Anarchy・スティーブン vs モード） | `SuccessionCrisisRisk` はリスク値どまり。**請求者が並立し諸侯が旗幟を選ぶ内戦の盤面化**（＝関ヶ原カスケードの国家規模適用）が無い |
| **理想の指導者 vs 出世主義者**（修道院長 vs 司教） | 聖職＝宗教組織の**キャリアラダー**が無い（`CareerTrack` は武/官/技のみ） |
| **数十年前の不正が世代を超えて露見**（沈没船の秘密） | 開示エンジン（FND-4）への**世代スケールの秘史**データ入力が薄い |

**結論**：大聖堂は当プロジェクトに**「世代を超えるメガプロジェクト」という背骨**を与える。これは①暦（TIME）②財政（#163）③継承（#812）④群像（LIFE）⑤秘史（FND-4）という**既存モジュールを数十年スケールで一本に束ねる結節点**であり、銀英伝世界の象徴（イゼルローン的要塞・大シップヤード）に「**建てた者たちの物語**」を与える。

## 1. 役に立つ視点（要約）

大聖堂の物語構造を、**本システムに効く形**で1行ずつ：

1. **巨大プロジェクトが群像の背骨になる**。一つの建設が石工・貴族・聖職者・庶民の運命を数十年束ねる。→ 当プロジェクトの群像純ロジック（LIFE/GOV/列伝#784）に**「全員が関わる一本の軸」**を与える。勝敗ではなく「完成」が目標になる王道路線。
2. **事業は指導者より長生きする**。発起人が死んでも後継が継ぐ——制度化されていれば（#812 カリスマの日常化と同型）。→ 属人事業は発起人と頓挫し、制度化された事業は世代を超える。
3. **急げば崩れる**。早さを買えば品質を失い、崩落が数年分の進捗を奪う。→ 高位の決断（工期優先/品質優先）に**創発的帰結**（タイクン回避に一貫）。
4. **技術は人に宿る**。新工法は書物でなく**人と共に**旅をする。棟梁の死は技術の死。→ 諜報・引き抜き・亡命（#194）に**技術的価値**を与える。
5. **利権が争いの火種**。採石権・市場開設権をめぐり修道院・司教・伯爵の管轄が衝突する。→ 戦争でなく**特許状の授与・取消**で進む権力闘争（OWN#1032 の利権版）。
6. **大事業は都市を育てる**。職人が集まり、市が立ち、村が町になる。→ 内政（#109）に**投資→成長の物語**を与える。
7. **継承戦争＝国家規模の関ヶ原**。王位請求者が並立し、諸侯は日和見・寝返りで旗幟を選ぶ。→ `LoyaltyRules` のカスケードを**内戦**へ拡張する最短路。
8. **埋もれた不正は世代を超えて露見する**。一隻の沈没船の秘密が数十年後に王権を揺るがす。→ 開示エンジン（FND-4）の**世代スケール lore** の型。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**建築マイクロは作らない**（タイクン回避＝CLAUDE.md「建設マイクロ/通貨経済は持たない」）。プレイヤーの操作は「着工・中断・資金水準・方針」の**高位の決断のみ**、進捗はエンジン駆動・帰結は創発。
> 既存モジュール（造船#884/財政#163/継承#812/関ヶ原#817/Charter#624）は**作り直さず接続**する（additive）。

### ★★★ 最優先（真の欠落・大聖堂の signature）

#### PIL メガプロジェクト基盤（世代を跨ぐ国家的大事業）
- **`MegaProject`/`MegaProjectRules`**（純ロジック・test-first）：種別（要塞建設/大シップヤード/遷都/記念碑）×段階建設（基礎→構造→完成の phase）×**数十年スケール**（`CalendarDispatcher` の月次/年次 Tick で進捗）。
- **財政ドレイン**：継続支出が `FiscalRules` の歳出に乗る＝**中断/再開の決断**（資金難で止まる聖堂）。完成効果は種別ごと（要塞=戦略ゲートC-7・シップヤード=`buildPower`増・記念碑=正統性/希望）。
- **事業の継承**：発起人（ネームド）の死で、制度化（`Organization.institutionalization`）が低ければ頓挫リスク・高ければ続行（#812 と同じ式を**事業に適用**）。
- 接続：`CalendarDispatcher`（TIME-6）×`FiscalRules`×`SuccessionRules`。**完成済みの要塞 #76-78/C-7 に「建てる物語」を前置する**。

#### PIL 工期と品質のトレードオフ（急造→崩落）
- 工期優先は進捗倍率↑だが**構造品質**が下がり、崩落/火災/襲撃イベント（`EventEngine` #116）で**進捗が後退**する（数年分を失う）。品質優先は遅いが堅い。
- 決断は「方針レバー1つ」、帰結はイベントで創発（事前に確率を見せない＝[no-outcome-preview] の方針に一貫）。
- 接続：PIL-1×`EventEngine`。

#### PIL 技術は人に宿る（棟梁・工法の伝来）
- ネームド技術者（`Person.engineering`/`CareerTrack.テクノクラート`）が**工法（technique）を保持**：保持者が事業に就くと進捗/品質ボーナス、**死亡で喪失**、移籍/亡命/引き抜きで**勢力間を伝播**。
- 新工法の獲得は「外で学ぶ」＝他勢力の完成事業・先進星系への**人の派遣**（ジャックのフランス遍歴の型）。
- 接続：`Person`/`CareerPipelineRules`×`ResearchRules`（抽象研究点に**人という乗り物**を与える）×`EspionageRules`/亡命`CultureRules.ExileLikelihood`（引き抜きの経路）。

### ★★ 高（権力闘争・内戦への展開）

#### PIL 継承戦争（The Anarchy＝国家規模の関ヶ原）
- 君主死亡×`SuccessionLawRules.SuccessionCrisisRisk` 高で**請求者並立→内戦状態**：諸侯（`Allegiance`）が請求者ごとに旗幟（戦う/静観/寝返り）を選び、`LoyaltyRules.ResolveCascade` を**国家分裂スケール**で回す。
- 内戦中は中央権威が消え、無法（安定度低下・利権の実力奪取＝PIL-4 と連動）。決着＝実効兵力の不動点（既存式の再利用）。
- 接続：`SuccessionLawRules`×`LoyaltyRules`（**新カスケードを作らず適用範囲を拡げる**）×`DiplomacyRules`（外部勢力の介入）。

#### PIL 利権と特許状（採石権・市場開設権の争奪）
- **特許状＝授与・取消・争奪できる利権資産**：市場開設権（市の立つ権利）・資源採掘権・通行税徴収権。元首/政府が組織・ネームドへ授与（`OfficeRules.CanPropose` の所掌で申請）。
- **管轄争い**：同じ利権を複数主体（宗教組織 vs 貴族 vs 政府部局）が請求＝`MinistryRules.SectionalismFriction` の利権版。取消は怨恨（opinion/忠誠低下）。
- 接続：OWN#1032（資産所有権＝徴税権は既に視野）×`MagnaCartaRules.Charter`（一般制約の**個別利権版**）×`GovernmentRegistry`。

### ★ 中（内政の物語・キャリア・lore）

#### PIL 大事業が都市を育てる
- 進行中のメガプロジェクトが立地 `Province` に**人口流入**（`DemographicsRules`）と**市場成立**（`Market` #179・市場開設権 PIL-4 と連動）をもたらし、村→町→都市の成長物語に。事業中断で衰退（出稼ぎ職人が去る）。
- 接続：PIL-1×`Province`/`GovernanceRules`×`DemographicsRules`×`MarketRules`。

#### PIL 聖職キャリアと理想 vs 野心
- `CareerTrack` に**聖職トラック**を追加（武/官/技に次ぐ第4系統）：宗教組織（`ReligionRules` #172）の役職ラダー（修道士→院長→司教級）を `Office`/`GovernmentRegistry` の枠で持つ。
- 理想駆動（共同体の希望↑）と出世主義（省益/権力闘争）が**同じラダーを別の動機で登る**＝人物の対比は数値でなく**行動の帰結**で描く。
- 接続：`CareerPipelineRules`×`OfficeRules`×`ReligionRules`。SAW-7（教会の経済力）とは直交（あちらは経済・こちらは人事）。

#### PIL（lore）世代スケールの秘史
- 「数十年前の不正（処刑・簒奪）が世代を超えて露見し、現王権の正統性を揺るがす」型の開示データ。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。`Regime.legitimacy`（王朝サイクル #867）への効果と連動。

### ❌ 不採用（重複・タイクン化・後退）

| 不採用 | 理由 |
|---|---|
| 建築マイクロ（資材・職人の個別配置・建物グリッド） | **タイクン回避の大原則に反する**。決断は高位レバーのみ・進捗はエンジン駆動 |
| 教会の経済力（十分の一税・交易支配） | **SAW-7（#1078）がカバー**。PIL-7 は人事ラダーのみ |
| 羊毛交易・市場の需給 | **`MarketRules` #179/SAW がカバー**。PIL-5 は「市場の成立」契機のみ |
| 個人の愛憎ドラマの直接実装 | 群像は既存（LIFE/列伝#784/イベント#116）の**創発**で出す。脚本化しない |
| 中世固有の風物（ギルド・巡礼・贖宥状） | 世界観が違う。**構造パターンのみ**輸入（贖宥は SAW-7 側） |

`PIL-1 → PIL-2`（メガプロジェクト＝最も固有で欠落の大きい signature・背骨）→ `PIL-3`（技術は人に宿る＝群像×研究の接続）→ `PIL-6`（継承戦争＝既存カスケードの最大活用）→ `PIL-4 → PIL-5`（利権→都市成長＝内政の物語）→ `PIL-7 → PIL-8`（キャリア・lore）。
