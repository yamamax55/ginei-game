# トゥキュディデス『戦史』参考設計（EPIC #228）

> 参照元：トゥキュディデス『戦史（ペロポネソス戦争史）』。紀元前5世紀、アテナイとスパルタを中心とする全ギリシャを巻き込んだ27年戦争の記録。
> 「永遠の資産（ktema es aei）」——人間の本性は変わらず、力学は繰り返す——という著者の宣言が本書の核。神話を排した**客観的因果の観察**が国際政治リアリズムの原点。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に大量の外交・政治・社会純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#228` として issue 化する提案。
> 著作権注意：固有名・文章・キャラクター・固有設定は流用せず、**国際政治メカニクス／世界観の構造パターンのみ**を参考にする。

## 0. なぜ「戦史」が本システムに役立つか

当プロジェクトは国際政治・外交・社会動態の**広範な純ロジック層を既に保有**している：

| 既存（外交・政治・社会） | カバー範囲 |
|---|---|
| `DiplomacyRules`/`DiplomacyState`（#189） | 外交状態遷移・条約・戦争宣言・講和 |
| `WarGoalRules`/`CasusBelli`（#192） | 厭戦・戦争目標正統性・講和受諾 |
| `LoyaltyRules`/`Allegiance`（#817） | 忠誠カスケード・関ヶ原型寝返り |
| `CivilianControlRules`（#145） | 文民統制・クーデターリスク |
| `FactionState`/`FactionStateRules` | 国家状態合成（王朝/統治体/組織/共同体） |
| `ConsentRules`/`Polity`（#836/837） | 同意・協力の撤収・統治不能 |
| `HopeRules`/`Community`（#852-856） | 希望・末人・社会の解体 |
| `DynastyRules`/`Regime`（#867） | 正統性・腐敗・天命喪失 |
| `PartyRules`/`Party`（#159） | 政党・議員・党首選出 |
| `PersonRules`/`Person`（#866） | 人物・適材適所・役割 |
| CLW-2/CLZ（#222/#1132） | 戦争激化・摩擦・三位一体 |
| CLW-7（#227） | 攻撃の限界点（過剰拡張） |

**しかし、これらは「戦争が起きた後の動態」か「個別メカニクス」**であり、戦史が固有に描く以下が**欠けている**：

| 戦史が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **構造的開戦圧力（覇権移行恐怖）** | `DiplomacyRules.DeclareWar` は事後判断。**新興勢力の台頭→既存覇権の恐怖→引き金無しに開戦へ傾く**構造的な前戦圧力が無い |
| **国家行動の3動機（恐怖・名誉・利益）** | 外交判断の重みは opinion と状態。**「なぜ戦うのか」の内部動機モデル**（恐怖→予防戦/名誉→報復/利益→拡張）が無く、AIが「合理的に損するとわかっていても戦う」回路が弱い |
| **強者の論理・服従の強要** | `DiplomacyRules` は対等な相互合意型。**圧倒的力の差を背景にした一方的要求**（服従か征服か）という非対称外交が無い |
| **長期戦による理性の腐食** | `HopeRules`/`WarGoalRules.WarWeariness` は個別に機能。**長期戦→言葉の変質→穏健選択肢が政治的自殺に変わる**フィードバックループが無い |
| **扇動政治家と民主政の過剰拡張** | `PartyRules` は政党構造。**扇動型政治家が世論を煽り、理性的制約（文民統制・限界点）を飛び越えて無謀な遠征を可決させる**デマゴーグ回路が無い |

**結論**：戦史は「**戦争はなぜ起きるのか（構造）**」と「**戦争が社会に何をするのか（腐食）**」という**2つの欠落軸**を与える。①**覇権移行恐怖（THU-1）**＋②**リアリズム3動機（THU-2）**が前戦ロジックの核で、③**強者の論理（THU-3）**＋④**理性の崩壊（THU-4）**＋⑤**衆愚政治（THU-5）**が戦争動態の腐食面を補う。**クラウゼヴィッツ（#1132）が「戦争はどう動くか」なら、戦史は「なぜ戦争は不可避になり、社会を食い尽くすのか」**という別の問いに答える。

## 1. 役に立つ視点（要約）

戦史の世界観を、**本システムに効く形**で1行ずつ：

1. **「構造が戦争を生む」**＝トゥキュディデスの罠。アテナイの台頭とスパルタの恐怖——些細な事件ではなく、**力の均衡が崩れる瞬間**が戦争を生む。台頭しすぎると包囲網・予防戦争が自動生成される。→ 当プロジェクトの銀河覇権争いに**構造的緊張の自動生成**を与える。
2. **「国家の本音は3動機（恐怖・名誉・利益）」**。道徳・法・正義はしばしば大義名分で、内部判断はこの3つ。→ AIの外交・開戦判断を**動機駆動の意思決定**に変え、「なぜ戦うのか」に説得力を持たせる。
3. **「強者はなせることを行い、弱者は耐える」**（メロス対話）。正義は対等な力の間でのみ機能する。→ 中立・弱小勢力への**服従強要・朝貢・保護国化**を外交の一形態として正式化する。
4. **「戦争は人間の理性を奪う乱暴な教師」**（コルキュラ内乱）。長期戦が「穏健＝臆病」「強硬＝勇気」へ言葉を変質させ、和平を政治的自殺にする。→ `HopeRules`×`WarGoalRules` に**社会腐食フィードバック**を加える。
5. **「民主政は扇動に脆い」**（シケリア遠征）。理性的指導者の死後、デマゴーグが世論を煽って無謀な大遠征を可決させ自滅。→ `PartyRules`×`PersonRules` に**扇動型政治家**を実装し、過剰拡張（CLW-7）の民主政パスを開く。
6. **「同盟の引きずり込み」**。義務的な同盟援助が、本来は当事者でない国を不望の戦争に引き込む。→ `DiplomacyRules.SignTreaty` の同盟義務に**引きずり込みコスト**（厭戦・損耗を同盟国が受け取る）を付ける。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**外交#189・戦争#192・政体#117・CLW系・HopeRules等を作り直さない**。THU はそれらに**欠落軸を足し、接続する**だけ（additive）。

### ★★★ 最優先（真の欠落・戦史の signature）

#### THU-1 トゥキュディデスの罠（覇権移行恐怖・構造的開戦圧力）
- **国力トレンド検出**：各勢力の総合力（戦力×経済×領土）の**伸び率**を時系列でスコア化。急成長する新興勢力を検出する。
- **恐怖蓄積モデル**：既存の強勢力に、新興勢力の成長速度×近接度×同盟拡張で**恐怖値（fear）**が蓄積。fear が高いほど `DiplomacyRules.DeclareWar` の判断が**先制・予防戦争**へ傾く（些細な事件が引き金になる）。
- **エスカレーション結合**：fear が高い構造下で小紛争が発生すると、CLW-2(#222) の激化度が**通常より急速に**上昇（構造的恐怖が個別の限定紛争を総力戦化させる）。
- 接続：新 `HegemonyRules`（純ロジック・test-first）＋`DiplomacyRules` の開戦判断に接続。fear は `FactionState` に保持。

#### THU-2 国家の3動機（恐怖・名誉・利益）— リアリズムのAI駆動原理
- **動機重みベクトル**：各勢力に「恐怖(fear)・名誉(honor)・利益(interest)」の重みを持たせる（政体/指導者の個性で配分が変わる：軍閥=interest寄り、王党=honor寄り、被包囲=fear寄り）。
- **外交判断への投影**：`DiplomacyRules` の開戦/講和/同盟の判断を、この3重みでスコア評価。fear=予防戦争/honor=侮辱報復/interest=領土拡大。道徳・正義は**大義名分**（イベントテキストに出る）だが内部判断は3動機。
- **接続**：新 `RealistMotiveRules`（純ロジック）。`FactionStateRules.Tick` に動機ドリフト（戦争継続でfear上昇 / 大勝でhonor上昇）を追加。THU-1 fear と直結。

### ★★ 高（戦史の核を構造化）

#### THU-3 メロス対話（強者の論理・服従の強要）
- **力の非対称判定**：強勢力が中立/弱小勢力に対し、軍事力比×地政学的価値で「服従要求」を発動可能。
- **要求の段階**：朝貢/中立放棄/従属（保護国化）/併合。受諾＝従属（自治縮小・資源供出・非敵対義務）、拒否＝征服（占領#131。住民処遇の重い外交代償）。
- **中立の脆さ**：`DiplomacyRules` の中立ステータスは「対等な力か後ろ盾がある時のみ維持可能」——後ろ盾が利害なし（THU-2 の interest＜コスト）なら見捨てられる。
- 接続：`DiplomacyRules` の拡張 or 新 `CoerciveDemandRules`（純ロジック）。`StrategyRules.ResolveOccupation` × #160 フェザーン（中立の商業国家が最もメロス的状況に置かれうる）。

#### THU-4 戦争による理性の崩壊（言葉の変質・社会の過激化）
- **過激化フィードバック**：戦争激化度 CLW-2(#222) × 損害 × 長期化で `FactionState.Community.Hope` への腐食を蓄積（hope への係数ペナルティ）。
- **言葉の変質**：過激化が高いと `WarGoalRules.PeaceAcceptance` が**強制的に低下**（和平が政治的に困難になる）。穏健・講和#192 の選択肢が `EventEngine` で支持ペナルティを付けられる（「臆病」イベント）。
- **内乱の火種**：過激化が閾値超で `CoupRules`/`HopeRules.UpdateDissent` がトリガーしやすくなる（コルキュラ型内乱→#194/#215）。
- 接続：新 `WarRadicalizationRules`（純ロジック。`FactionState` に `radicalization` 値を追加・`HopeRules`×`WarGoalRules`×`EventEngine` に接続）。CLZ三位一体 CLW-3(#223) の民衆の情熱と連動。

#### THU-5 衆愚政治とデマゴーグ（無謀な遠征と自滅）
- **扇動型政治家**：`PersonRules.Person` に `demagogueRating`（0..1）を追加。扇動型政治家は `PartyRules` で世論#193 への煽り能力を持ち、支持#113 を背景に**過大な軍事行動の可決確率**を上昇させる。
- **理性的指導者の喪失**：指導者が死亡/失脚(`LifecycleRules`)すると後任がデマゴーグになりやすい（`VacancyRules.SelectSuccessor` に政体差を反映）。
- **過剰拡張の自滅**：無謀な遠征が CLW-7(#227) 限界点を超えて大損害→厭戦#192×支持崩壊→政体動揺（革命#219）。民主政で起きやすい（政体#117 との係数差）。
- 接続：`PersonRules` 拡張＋新 `DemagogueRules`（純ロジック）。`CivilianControlRules.CoupRisk` の別バリアント（クーデターでなく**衆愚による過剰拡張**リスク）。

### ★ 中（既存を薄く補強）

#### THU（同盟の引きずり込み）
- **義務的援助コスト**：`DiplomacyRules.SignTreaty` の同盟条約に**引きずり込みコスト**（同盟国の戦争に巻き込まれると `WarGoalRules.WarWeariness`・`FleetPool` に損耗が乗る）を付与。
- 接続：`DiplomacyRules.TreatyRules`（#191 DIP-2）の拡張。純ロジック追加のみ。

### ❌ 不採用（重複・既存で十分）

| 不採用 | 理由 |
|---|---|
| 厭戦・戦争目標の疲弊そのもの | **`WarGoalRules.WarWeariness` が既存でカバー**。THU は接続で腐食を足すだけ |
| 疫病（アテナイの疫病） | **`LifecycleRules.AnnualMortality` が既存でカバー**（指導者の死亡は自然なテストケースで扱う） |
| 地形・陣形・戦術 | **CLZ/SUN 系がカバー**。個別戦闘の微操作はタイクン化回避で実装しない |
| 民主政vs専制の政体論そのもの | **`CivilianControlRules`/#117 政体が既存でカバー**。THU-5 は政体差を係数で使うだけ |
| 個別提督の人物列伝（歴史的固有名） | **著作権・固有名禁止**。構造（デマゴーグ型/理性型の軸）のみ参照 |
| 同盟の裏切り・離脱そのもの | **`LoyaltyRules`/`DiplomacyRules.BreakTreaty` が既存でカバー** |

### 推奨着手順
`THU-1 → THU-2`（覇権移行恐怖＋動機モデル＝前戦ロジックの核・相互依存が強い）→ `THU-3`（メロス対話＝力の非対称外交の正式化）→ `THU-4`（社会腐食フィードバック＝和平困難ループ）→ `THU-5`（衆愚政治＝民主政の過剰拡張パス）。

> いずれも既存の外交・政治・AIを**後退させず接続**する additive 設計。THU-1/2 は外交AI全体の動機を刷新するため先行実装価値が高い。
