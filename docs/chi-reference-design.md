# チ。-地球の運動について- 参考設計（EPIC #CHI）— 君は物語の主人公ではない

> 参照元：『チ。-地球の運動について-』（魚豊）。弾圧下の時代に、ある真理（地動説）を命がけで追求する人々が、**一人では完成せず、世代を超えてバトンを渡しながら**真理に近づいていくリレー群像劇。中心テーマ＝**「君は物語の主人公ではない」**——個人は真理の中継者にすぎず、完成を見ずに死に、記録にも残らない無名の積み重ねが歴史を前進させる。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の群像戦略劇）にとって**役に立つ視点**だけを抽出し、EPIC `#CHI` として issue 化する提案。
> 著作権注意：固有名・キャラクター・具体的な筋は流用せず、**「弾圧を超えて受け継がれる真理のリレー」「主人公不在の群像」の構造パターンのみ**を参考にする。

## 0. なぜ「チ。」が本システムに役立つか — アンチ主人公史観

当プロジェクトは群像・人物の純ロジックを大量に持つ（[CLAUDE.md] 参照）：

| 既存（群像・継承・知） | カバー範囲 |
|---|---|
| 列伝/殿堂（#784/#785） | 記録される人物の経歴 |
| `Organization`/`SuccessionRules`#812 | 英雄死後の組織存続（制度化 vs 個人カリスマ） |
| 大聖堂PIL（#1081） | 世代を超えるメガプロジェクト（建設の継承） |
| `ResearchRules`#123-127・VOY（#1202） | 研究・不確実な開示 |
| `DisclosureLedger`（FND-4） | 秘史の連鎖開示 |
| 主人公#735（`ProtagonistRules`）・社交場SLN/群像TALE | ネームド中心の群像 |

**しかし、これらは「記録される個人」「主人公的な英雄」を中心に据えている**。チ。が固有に体現する以下が**欠けている**：

| チ。が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **真理/知のリレー**＝一人では完成せず世代を超えてバトンが渡る | #812は組織存続、PILは建設の継承。**知識・思想・真理が個人の死を超えてリレーされる**回路が無い |
| **君は主人公ではない**＝個人は中継者・完成を見ずに死ぬ | 人物は主人公的（#735・列伝）。**「誰も主役でない・全員が中継点」というアンチ主人公史観**が無い |
| **名もなき貢献者**＝記録に残らない者が歴史を前進させる | 列伝#784は記録される者。**記録に残らない無名の積み重ね**がモデル化されていない |
| **異端弾圧と知の殉教**＝真理を守るため命を賭ける・禁書・拷問 | `ReligionRules`に異端はあるが、**弾圧が殉教を生み、かえって真理を伝播させる**逆説が無い |
| **禁じられた知の地下伝承**＝弾圧下で隠れて受け継ぐ（写本/暗号/秘密結社） | 諜報はあるが、**禁書の地下ネットワークで知が生き延びる**回路が無い |
| **パラダイムの転換**＝世界観そのものが覆る（天動説→地動説） | VOYは解読の不確実性。**世界の見方を覆す真理が、観測の積分で勝つ**動学が無い |

**結論**：チ。は当プロジェクトに**「主人公不在の、真理のリレー」**を与える。`Organization`#812（組織の存続）・PIL（建設の継承）の隣に、**知識・思想・真理という"無形物"の世代継承**を置く。そして最大の貢献は**アンチ主人公史観**——本作は群像劇だが、それでもネームド提督が「主人公」になりがちだ。チ。は「**歴史は無名の中継者の積分であり、誰も主役ではない**」という視点を足す。これはバックログのトルストイ『戦争と平和』（非人格的な歴史の力・英雄史観の解体）と共鳴し、本作の群像劇を一段深くする。

## 1. 役に立つ視点（要約）

1. **真理は一代では完成しない**。個人は中継者にすぎず、バトンを次へ渡して、完成を見ずに死ぬ。
2. **君は主人公ではない**。歴史に主役はいない——全員が中継点で、無名の積み重ねが前へ進める。
3. **記録に残らない者が世界を動かす**。列伝に載らない貢献こそ、真理の前進の本体。
4. **弾圧は殉教を生み、殉教は伝播を生む**。禁じれば禁じるほど、その知は地下で強く受け継がれる。
5. **禁書は地下を流れる**。写本・暗号・秘密結社が、抑圧された知を次代へ運ぶ。
6. **世界の見方は覆る**。観測の積分が、ある日パラダイムをひっくり返す。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**列伝#784/殿堂#785・#812・PIL・`ResearchRules`・`DisclosureLedger` を作り直さない**。CHI はそれらに**「無形物（知/真理）の世代リレー」と「アンチ主人公史観」**を足し、接続するだけ（additive）。タイクン化回避＝リレーは高位の決断（誰に託すか/弾圧するか）、真理の前進は創発。

### ★★★ 最優先（真の欠落・CHI の signature）

#### CHI 真理/知のリレー（無形物の世代継承）
- `LegacyRelayRules`（static・test-first）＝知識・思想・事業などの**無形の達成**を、保持者の死を超えて**次代へバトン**する。各中継者は前任の達成に自分の寄与を**積み増す**（誰も単独で完成させない）。バトンが途切れる（後継不在・弾圧）と達成が失われ、繋がれば前進。`Organization`#812 の**無形物版**。
- 接続：#812（制度化と同型の継承）・列伝#784（中継者の記録）・継承#152（世代）・`DisclosureLedger`（達成の段階開示）。

#### CHI 名もなき中継者とアンチ主人公史観
- `UnsungContributorRules`（static）＝**記録に残らない貢献**をモデル化。歴史の前進は、列伝#784に載るネームドだけでなく、**無名の中継者の積分**による。主人公#735 を「歴史の主役」とせず、**全員が中継点**とする視点（達成への寄与度は記録の有無と独立）。
- 接続：`ProtagonistRules`#735（アンチ主人公）・列伝#784/殿堂#785（記録される者との対比）・モブ→ネームド・トルストイ（バックログ・歴史の力）。

### ★★ 高（弾圧と伝播）

#### CHI 異端弾圧と知の殉教（弾圧が伝播を生む逆説）
- `MartyrdomRules`（static）＝真理/思想の保持者を弾圧（禁書・処刑・拷問）すると、短期は伝播を抑えるが、**殉教が同調者を生み**、かえって長期の伝播を強める非線形（殉教者は記憶のシンボルになる）。弾圧の強さと伝播の関係はU字（緩い弾圧＜放置＜苛烈な弾圧の反動）。
- 接続：`ReligionRules`#172異端・`HopeRules`#852（殉教が希望を灯す）・支持#113・`DisclosureLedger`・OTH 弾圧との対比。

#### CHI 禁じられた知の地下伝承（禁書のネットワーク）
- `UndergroundLoreRules`（static）＝弾圧下の知が**地下ネットワーク**（写本・暗号・秘密結社）で生き延び次代へ運ばれる。露見すれば断たれるが、分散・冗長化で頑健になる。発見は諜報、保護は秘匿。
- 接続：`EspionageRules`（捜索/秘匿）・SAW-3 風説（地下の情報流）・VOY 来歴（写本の伝来）・`DisclosureLedger`。

### ★ 中（パラダイム・lore）

#### CHI パラダイムの転換（観測の積分が世界観を覆す）
- 蓄積した観測/証拠が閾値を超えると、**世界の見方（パラダイム）が転換**する（天動説→地動説型）。旧世界観の権威（教権・体制）は抵抗するが、証拠の積分が勝つ。VOY-2 競合解釈の**「決着がつく」版**。
- 接続：`ResearchRules`#123・VOY-2 競合解釈（#1204）・クーン型・`DynastyRules`（旧体制の権威）・`DisclosureLedger`。

#### CHI（lore）主人公不在の歴史観
- 「歴史は無名の中継者の積分」「君は主人公ではない、だが君のバトンが未来を作る」＝群像劇の哲学の開示データ。英雄史観の解体（トルストイ）と、それでも個の寄与が意味を持つ逆説。
- 接続：`DisclosureLedger`（FND-4）・世界観codex・列伝#784。コード新設は最小。

### ❌ 不採用（重複・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 組織の存続そのもの | **#812 がカバー**。CHI は無形物（知/真理）のリレーを足す |
| 建設事業の世代継承 | **PIL #1081 がカバー**。CHI は知識・思想の継承（無形）に特化 |
| 研究ツリー・不確実な開示 | **`ResearchRules`/VOY がカバー**。CHI はパラダイム転換と弾圧伝播を接続 |
| 実在の科学者・具体的筋の再現 | 構造のみ。固有名・キャラは使わない |
| 異端判定そのもの | **`ReligionRules`#172 がカバー**。CHI は弾圧→殉教→伝播の逆説を足す |
