# E.H.カー『歴史とは何か』 参考設計（EPIC #CARR）

> 参照元：E.H.カー『歴史とは何か』（歴史哲学・史学方法論の講義）。**「歴史とは現在と過去との間の尽きることなき対話である」**を核に、**歴史家による事実の選別**（無数の出来事のうち歴史家が採用したものだけが"歴史的事実"になる）・**歴史家自身がその社会と時代の産物**である（解釈は解釈者の立場に染まる）・**因果の序列化と偶然の扱い**・**進歩史観（歴史に方向はあるか）**を論じる、歴史記述の認識論。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に巨大な歴史・記録・社会シミュ純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#CARR` として issue 化する提案。
> 著作権注意：固有名・文章・固有設定は流用せず、**史学方法論（選別/対話/因果/進歩史観）の構造パターンのみ**を参考にする。架空の勢力・出来事で実装する。

## 0. なぜ「カー『歴史とは何か』」が本システムに役立つか

当プロジェクトは「真実・記録・歴史記述」に関わる純ロジックを**すでに二本柱で保有**している（[CLAUDE.md]「テスト基盤」節・既存参考設計 参照）：

| 既存（記録・真実・歴史の二本柱） | カバー範囲 |
|---|---|
| `DisclosureRules`/`DisclosureLedger`/`DisclosureEntry`（FND-4 #495） | 条件→**確定的な連鎖開示**（秘史A→秘史B、不動点まで）＝真実を明るみに出す |
| NTEF `NarrativeControlRules`/`OfficialNarrative`（#1639） | 国家が**真実を能動的に製造・改竄**＝公式言説 vs 実体の乖離（真実乖離度） |
| HRDT `TestimonyRules`/`HistoriographyRules`（#1702-3） | **出来事がどう記録・伝聞・検証されるか**＝出典つき証言・異説併記・**勝者が正史を書く** |
| `DynastyRules`/`Regime`（#867） | 王朝サイクル・天命・腐敗・正統性 |
| `CultureRules`（#194）／`JusticeRules`（#918） | 文化の同化/分離／正義観構成→正統性 |
| `EspionageRules`／`EventEngine`（#116） | 情報収集・露見／データ駆動イベント |
| `CampaignState`/`FactionState`（統合層） | 正統性/合意/結束/希望の合成・崩壊判定 |

**しかし、この二本柱は「真実は1つで、それを①開示する（FND-4）②製造する（NTEF）③記録する（HRDT）」**営みであり、カーが固有に論じる**「歴史記述という解釈行為そのものの認識論」**——**過去のどの事実を"歴史的"と選ぶかは現在の関心が決める／解釈者は自分の時代の産物／因果は序列化され偶然は事後に消される／進歩には方向があるかという信念が国を動かす**——は**欠けている**：

| カー『歴史とは何か』が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **現在と過去の対話**＝同じ過去でも**現在の関心が変わると採用される事実が変わり、解釈が書き換わる**（捏造でなく"選び直し"） | NTEF は**能動的捏造**（嘘を作る）、HRDT は**勝者が一度確定**する正史。**現在の関心の変化で過去の解釈が静かに再選別される**（嘘なしの再解釈）回路が無い |
| **歴史家による事実の選別**＝無数の出来事のうち**歴史家が採用したものだけが歴史的事実になる**（採用されない事実は存在しないも同然） | HRDT は証言の**信頼度**で重みづけ。**「どの事実を歴史に算入するか」という選別フィルタ自体**（関心の外の事実は記録されても歴史にならない）が無い |
| **歴史家はその社会・時代の産物**＝解釈は**解釈者の立場（政体/思想/階級）に必ず染まる**（中立な歴史は無い） | 解釈の主体（語り手）の**立場バイアス**を構造化していない。HRDT 威信は"勝者"の力だが、**解釈者の時代背景が選別を歪める**軸が無い |
| **因果の序列化と偶然の扱い**＝歴史家は諸原因に**序列**をつけ、**偶然（クレオパトラの鼻）を事後の物語が消し去る** | 戦績・占領は起こるが、**「なぜそうなったか」の因果序列**と、**偶然/必然の比率を事後ナラティブが書き換える**回路が無い |
| **進歩史観＝歴史に方向はあるか**（進歩を信じる社会／衰退を信じる社会で決断が変わる） | `DynastyRules` は循環史観（盛衰の反復）。**「歴史は進歩する/方向がある」という直線史観の信念**が勢力の決断・正統性に効く軸が無い |
| **過去を現在の基準で裁く誤り**（時代錯誤の評価）と、その逆＝過去の文脈で理解する | `JusticeRules` は同時代の正義観。**過去の出来事を現在の価値で遡及評価して正統性が動く**（英雄が後世に断罪/復権される）軸が無い |

**結論**：カー『歴史とは何か』は当プロジェクトの「記録・真実」三本柱（開示=FND-4／製造=NTEF／記録=HRDT）に、**第四の軸＝「解釈という選別行為の認識論」**を与える。具体的欠落軸は **①現在との対話（関心の変化で過去の解釈が再選別される・嘘なしの書き換え）②事実の選別フィルタ（採用された事実だけが歴史になる）③解釈者の立場バイアス（中立な歴史は無い）④因果序列と偶然の事後抹消 ⑤進歩史観という直線史観の信念**。いずれも NTEF（捏造）・HRDT（記録）・`DisclosureRules`（開示）の**いずれとも直交**し、`CampaignState`/`FactionState` と HRDT 正史層に薄く接続して、**「同じ過去が世代/政体ごとに違う歴史として語り直され、英雄が後世に断罪/復権し、進歩を信じる国と循環を信じる国が違う賭けに出る」**という創発を生む。

## 1. 役に立つ視点（要約）

カー『歴史とは何か』を、**本システムに効く形**で1行ずつ（既存issue/モジュールへの共鳴を明記）：

1. **歴史は現在と過去の対話**＝現在の関心が変わると過去の解釈が書き換わる（捏造でなく選び直し）。→ NTEF（#1639 能動的捏造）・HRDT（#1702-3 勝者が一度確定する正史）の**隙間＝嘘なしの再解釈**を埋める。`DynastyRules`#867 の正統性が世代で揺れる回路に。
2. **歴史家が事実を選別**＝採用された出来事だけが"歴史的事実"になる。→ HRDT `TestimonyRules`#1702 の証言群に**選別フィルタ**を載せ、関心の外の事実を歴史から落とす（記録はされるが歴史にならない）。
3. **解釈者はその時代の産物**＝中立な歴史は無く、解釈は政体/思想/階級に染まる。→ `CultureRules`#194・`JusticeRules`#918・政体`GovernanceRules`#109 を**解釈バイアスの源泉**として読む。
4. **因果は序列化され、偶然は事後に消される**＝勝者の物語が「必然だった」と語り、敗者の物語が「運が悪かった」と語る。→ `BattleManager` 戦績・HRDT 正史#1703 に**因果序列と偶然比率**を付け、ナラティブが書き換える。
5. **進歩史観＝歴史に方向はあるか**。進歩を信じる勢力は拡大・改革に賭け、循環/衰退を信じる勢力は保守に回る。→ `DynastyRules`#867（循環史観）の**対＝直線史観の信念**を勢力の決断係数・正統性へ。
6. **過去を現在の基準で裁く**＝英雄が後世に断罪/復権される（時代錯誤の評価）。→ `JusticeRules`#918 の遡及版＝故人#152・列伝#785 の評価が世代で動く。
7. **（lore）歴史に客観的真実はあるか**＝開示エンジン（秘史/啓蒙/ニーチェ）の認識論側面。→ `DisclosureLedger`（FND-4）への lore データ入力。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`DisclosureLedger`#495・NTEF#1639・HRDT#1702-3・`DynastyRules`#867・`JusticeRules`#918 を作り直さない**。CARR はそれらに**「解釈という選別行為の認識論」（現在との対話・選別フィルタ・立場バイアス・因果と偶然・進歩史観）**を足し、接続するだけ（additive）。タイクン化回避＝歴史の語り直しはプレイヤーに台帳のミクロ編集をさせず、**高位の決断（関心の設定・史観の採択）→エンジン駆動→創発帰結（正統性の揺らぎ・賭けの傾き）**に徹する。

### ★★★ 最優先（真の欠落・CARR の signature）

#### CARR 現在との対話＝過去解釈の再選別（嘘なしの語り直し）
- **`HistoricalInterpretationRules`（static・test-first）**＝確定済みの正史（HRDT#1703）や過去の出来事群を、**現在の関心（政体/支配勢力/思想）に応じて再選別・再解釈**する。捏造（NTEF）でなく**採用する事実の取捨と意味づけの変更**＝同じ過去から異なる「現在にとっての歴史」が生まれる。現在の関心が変わると（政変・占領・世代交代）**解釈が静かに書き換わる**＝正統性・士気の根拠が揺れる。
- 接続：HRDT `HistoriographyRules`#1703（確定正史を入力）・NTEF#1639（捏造との対＝嘘なしの再解釈）・`DynastyRules`#867（正統性）。`CampaignState`/`FactionState` 接続＝**観測層追従**。NTEF `ExtractiveDecay` の認知版とは別系統（捏造は乖離を溜めるが、再解釈は乖離を作らず"意味"だけを動かす）。

#### CARR 事実の選別フィルタ（採用された事実だけが歴史になる）
- **`FactSelectionRules`（static・test-first）**＝HRDT `TestimonyRules`#1702 の証言/出来事群に**選別フィルタ**を載せる。現在の関心に合致する事実は**歴史に算入**され（正統性・教訓・列伝に効く）、関心の外の事実は**記録されても歴史にならない**（採用されない事実は存在しないも同然＝カーの核）。関心が変わると過去に落とした事実が**後から歴史に拾い直される**（再発見）。
- 接続：HRDT-1 `TestimonyRules`#1702（証言群＝選別の素材）・CARR-1（再選別の本体）・列伝#785/殿堂#784（歴史に残る人物の選別）。**有界リングバッファ＋集約で無制限増加を防ぐ**（PERF規律）。

### ★★ 高（解釈者バイアス・因果と偶然）

#### CARR 解釈者の立場バイアス（中立な歴史は無い）
- **`InterpreterBiasRules`（static）**＝歴史を語る主体（支配勢力/語り手＝政体・思想・階級）の立場が、CARR-1 の再選別・CARR-2 の選別フィルタを**系統的に歪める**。専制は秩序の物語を好み、民主は自由の物語を好む＝**同じ過去でも解釈者の時代背景で違う歴史になる**。中立な歴史記述は原理的に存在しない（バイアスは消せず、複数のバイアスの併存だけが客観性に近づく）。
- 接続：`GovernanceRules`#109（政体）・`CultureRules`#194・`JusticeRules`#918（解釈者の価値観）・CARR-1/2（歪める対象）・BEH#1240（認知バイアスの源泉）。

#### CARR 因果の序列化と偶然の事後抹消（勝者は必然、敗者は不運と語る）
- **`CausalNarrativeRules`（static・test-first）**＝出来事（会戦/占領/政変）の結果に対し、**諸原因の序列**（決定的原因/副次原因）と**偶然/必然の比率**を割り当て、事後ナラティブが書き換える。勝者の歴史は「必然だった」と偶然を消し、敗者の歴史は「運が悪かった」と必然を偶然に転嫁する＝同じ事件の因果が立場で逆転。偶然比率が高い勝利は**正統性が脆い**（"運だった"と覆されやすい）。
- 接続：`BattleManager` 戦績・HRDT-2 正史#1703（因果が正史の一部）・`DynastyRules`#867（必然の物語＝天命）・PRL#1232（戦術勝利→戦略破滅の因果誤読）。`CampaignState` 接続＝**観測層追従**。

### ★ 中（進歩史観・遡及評価・lore）

#### CARR 進歩史観 vs 循環史観（歴史に方向はあるかという信念が決断を変える）
- **`HistoricalParadigmRules`（static・test-first）＋enum `HistoryView{進歩,循環,衰退}`**＝勢力が抱く史観が**決断係数**を変える：進歩史観は改革・拡大・技術投資に賭け（未来は良くなる）、循環史観は盛衰の反復に備え保守的、衰退史観は守りに入る。史観は出来事（連勝/災厄/改革成功）で**ドリフト**し、正統性（`DynastyRules`#867）・希望（`HopeRules`#852）に係数で効く。実効値パターン（基準非破壊）。
- 接続：`DynastyRules`#867（循環史観＝既存）・`ResearchRules`#123（進歩史観→技術投資偏り）・`HopeRules`#852（未来への希望）・`CampaignRules` 決断。

#### CARR 過去の遡及評価（英雄が後世に断罪/復権される）
- **`RetrospectiveJudgmentRules`（static）**＝故人（#152）・列伝（#785）の評価を**現在の価値基準で遡及評価**する＝時代錯誤の評価。政体/思想が変わると過去の英雄が断罪（粛清の対象/負の遺産）または復権（再評価で正統性の源泉）される。`JusticeRules`#918 の遡及版。コード新設は最小（`JusticeRules` を過去対象に適用）。
- 接続：`JusticeRules`#918（同時代→遡及）・`LifecycleRules`#152（故人）・列伝#785/殿堂#784・CARR-1（再解釈の人物版）。

#### CARR（lore）歴史に客観的真実はあるか（解釈の認識論）
- 「歴史は現在と過去の対話」「中立な歴史は無いが、複数の偏った歴史の併存が客観性に近づく」「進歩を信じる者と循環を信じる者は違う未来を生きる」＝開示データ。**コード新設せず** `DisclosureLedger`（FND-4）への lore データ入力。CCX-6（世界観codex退避）方針に一貫。
- 接続：`DisclosureLedger`（FND-4）・NTEF lore（真実の終焉）・HRDT lore（勝者が書く正史）と三位で世界観codexの「歴史記述の認識論」群を成す。

### 推奨着手順
`CARR-1 → CARR-2`（現在との対話＋事実の選別＝CARR の認識論の核・最も固有で欠落が大きい signature。NTEF=捏造/HRDT=記録 の隙間を埋める）→ `CARR-3`（解釈者バイアス＝選別を歪める源泉）→ `CARR-4`（因果と偶然＝事後ナラティブが勝敗の意味を書き換える）→ `CARR-5`（進歩史観＝史観が決断を変える・最も創発が大きい）→ 残り（CARR-6 遡及評価／CARR-7 lore）。
