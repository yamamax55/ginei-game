# ヴェニスの商人 参考設計（EPIC #VEN）

> 参照元：シェイクスピア『ヴェニスの商人』（16世紀末）。商業都市国家ヴェニスを舞台に、**人肉抵当の証文・法廷の逆転劇・高利貸しと差別・友情の債務**を描く喜劇。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋経済・法・群像の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#VEN` として issue 化する提案。
> 著作権注意：原作はパブリックドメインだが、本作では固有名・文章・キャラクターは流用せず、**法・契約・金融の構造パターンのみ**を参考にする。**特定の現実民族・宗教集団の表象は採らない**（架空勢力の構造として一般化＝差別の「構造」だけをメカニクス化する）。

## 0. なぜ「ヴェニスの商人」が本システムに役立つか

当プロジェクトは経済・法・群像の純ロジックを大量に保有している（[CLAUDE.md] 参照）：

| 既存（経済・法・人物） | カバー範囲 |
|---|---|
| `BankRules`#186・SAW-6（#1077） | 信用創造・取付け・ネームド商人のレバレッジ/破産 |
| `TreatyRules`（DIP-2 #191） | **国家間**の条約・破約 |
| OWN #1032（資産所有権） | 私有財産・家産制→法治 |
| `Charter`/`MagnaCartaRules`#624 | 制約権力・課税同意・抵抗権 |
| `SeparationOfPowersRules`#171 | 三権分立・専制リスク |
| `ReligionRules`#172-175 | 改宗圧力・異端・社会効果 |
| `CultureRules`#194 | 同化・分離独立・亡命 |
| SLN #1145（関係グラフ） | 親愛/反目/恩義/宿怨 |

**しかし、これらに「私人間の契約とその司法的執行」が無い**。ヴェニスの商人が固有に描くのは：

| ヴェニスの商人が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **証文（契約書）の文言主義**＝契約は文言どおり執行される | 条約は国家間（DIP-2）のみ。**私人間契約**（条項・担保・期限・保証人）のエンジンが無い |
| **法廷の逆転劇**＝「血を一滴も流さず肉1ポンド」＝文言の隙を突く弁論 | 紛争を解決する**裁判**が無い。文才（`CivilAptitude`）が輝く舞台が無い |
| **高利貸しと利子の倫理**＝宗教の利子禁忌→金融業が特定共同体に集中→差別と報復 | `ReligionRules` に**金融倫理**の歯が無い（SAW-7 教会経済とは別軸） |
| **異邦人条項**＝外国人は法に等しく守られない（財産没収リスク） | 市民権・法的地位の差が無い＝**収奪vs包摂#843 の法制度版**が無い |
| **「ヴェニスの信用」＝契約執行の信頼性が通商都市の競争力** | フェザーン#160（商社国家）の**繁栄の制度的源泉**が説明されていない |
| **保証人（友情の債務）**＝他人の借金に体を張る | SLN 恩義はあるが、**人間関係が金融リスクになる**回路が無い |

**結論**：ヴェニスの商人は当プロジェクトに**「契約と裁判」という法のミドルレイヤー**を与える。上は憲法・権力分立（#170/#171/#624）、下は資産・銀行（OWN/#186）が既にあり、**その間＝私人間の約束を国家が執行する仕組み**が抜けている。これが入ると「法治が通商を呼ぶ」（フェザーン#160）・「法が等しく守らない者は逃げる/牙を剥く」（#843/#194）が制度として動く。

## 1. 役に立つ視点（要約）

1. **契約は世界を動かす最小の法**。国家間条約（DIP-2）の私人版＝条項・担保・期限・保証人。破れば担保執行＝**約束がメカニクスになる**。
2. **裁判は「戦わない決闘」**。弁論（文才・情報）で勝敗が決まり、判決が強制執行される。軍才と対になる**文才の見せ場**。
3. **文言主義は両刃**。文言どおりの執行が信用を作るが、文言の隙が逆転を生む（血の一滴）。**厳格さと衡平のトレードオフ**。
4. **利子の倫理は経済地図を描く**。利子禁忌→金融業が周縁共同体へ集中→儲かるが憎まれる→差別と報復の火種。
5. **法の適用範囲が国の器を決める**。異邦人も等しく守る法＝通商ハブの競争力（ヴェニス/フェザーン）。守らない法＝資本逃避と怨恨。
6. **保証人＝友情の値段**。恩義（SLN）が金融リスクに変換される＝群像と経済の接点。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**銀行#186/SAW-6/条約DIP-2/OWN#1032 を作り直さない**。VEN は「私人間契約＋裁判」のミドルレイヤーを**足し、接続する**だけ（additive）。タイクン化回避＝個々の契約をプレイヤーに管理させない（ネームド人物・国家級の契約のみ＝PERF原則1）。

### ★★★ 最優先（真の欠落・VEN の signature）

#### VEN 契約エンジン（証文＝条項・担保・保証人・期限）
- `Contract`（純データ）＝当事者2＋保証人/条項（金額・期限・担保＝資産OWN#1032 参照・違約条項）。`ContractRules`（static・test-first）＝`IsDue`/`Fulfill`/`Default`（不履行→担保執行）/`GuarantorLiability`（保証人へ波及）。
- 接続：OWN#1032（担保＝資産）・`BankRules`/SAW-6（融資契約はこの上に乗る）・`TreatyRules`（国家間はそちら＝住み分け）。

#### VEN 裁判・法廷（訴訟→弁論→判決→執行）
- `TrialRules`（static）＝提訴（契約不履行/財産紛争）→**弁論**（原告/被告の代理人の**文才 `CivilAptitude`×情報**で優劣・決定論 roll）→判決（履行命令/担保執行/棄却）→執行。
- 接続：`PersonRules`（文才の見せ場＝**軍才と対になる人材価値**）・`SeparationOfPowersRules`#171（司法の独立度が判決の公正さを決める）・#627 デュープロセス。

#### VEN 文言主義と衡平（「血の一滴」の逆転）
- 判決は既定で**文言主義**（条項どおり）。ただし代理人の文才が高ければ**文言の隙**（衡平の抗弁）で逆転しうる＝厳格執行と衡平のトレードオフ。国の**法文化**（文言寄り↔衡平寄り）が係数。
- 接続：VEN-1/2 の上。`Constitution`#170（法の支配の質）・逆転劇は `EventEngine`/列伝#784 の名場面素材。

### ★★ 高（経済・社会への波及）

#### VEN 利子禁忌と金融共同体（宗教×金融）
- 勢力/共同体の宗教に**利子禁忌**属性→禁忌下では金融業が**周縁共同体へ集中**（儲かるが憎まれる）→好況時は共存・不況時に**差別と報復**の火種（`ClassTension` の宗教金融版）。
- 接続：`ReligionRules`（SAW-7 教会経済と相補＝こちらは**禁忌が生む金融の社会構造**）・`BankRules`・#110 階級。**現実民族の表象は採らない＝架空共同体の構造のみ**。

#### VEN 異邦人の法的地位（市民権と没収リスク）
- 勢力ごとに**法の人的適用範囲**（市民のみ↔居留民も↔万民）。範囲外の者は**財産没収・追放のリスク**＝資本逃避・亡命（`ExileLikelihood`#194）・怨恨（報復動機）。
- 接続：収奪vs包摂#843 の**法制度版**・OWN#1032（保護されない資産）・`CultureRules`#194。

#### VEN 契約執行の信頼性＝通商ハブ競争力
- 勢力の**契約執行信頼度**（司法独立#171×法文化×人的適用範囲）が**交易・商社の立地選好**に効く＝「ヴェニスの信用」。フェザーン#160 の繁栄の**制度的源泉**を与える。
- 接続：#160（商社国家）・`MarketRules`/FRM#1022（取引コスト係数）・`LogisticsRules`。

### ★ 中（群像・lore）

#### VEN 保証人と友情の債務・（lore）箱選び
- 保証人＝**恩義（SLN-1 関係グラフ）が金融リスクへ変換**される回路（友の借金に体を張る→不履行で関係/資産が壊れる）。縁談の**選別試験**（箱選び＝外見と中身）は SLN-7 縁談のフレーバーイベント・`DisclosureLedger` の lore。
- 接続：SLN-1/SLN-7（#1146/#1152）・`EventEngine`・FND-4。コード新設は最小。

### ❌ 不採用（重複・倫理・タイクン化）

| 不採用 | 理由 |
|---|---|
| 銀行・信用創造そのもの | **#186/SAW-6 がカバー**。VEN は契約＋裁判の枠だけ足す |
| 国家間条約 | **DIP-2 `TreatyRules` がカバー**（住み分け＝私人間のみ） |
| 現実の民族・宗教集団の表象 | 倫理・著作権方針。**差別の「構造」のみ**架空共同体で一般化 |
| 個々の少額契約のマイクロ管理 | タイクン化/PERF原則1違反。**ネームド・国家級の契約のみ** |
| 恋愛喜劇パートの脚本化 | 別ジャンル。縁談・箱選びは SLN-7/イベントのフレーバーで足りる |

`VEN-1 → VEN-2`（契約＋裁判＝法のミドルレイヤーの核）→ `VEN-3`（文言と衡平＝裁判に遊びを入れる）→ `VEN-6`（執行信頼性→通商＝フェザーンに効く）→ `VEN-5 → VEN-4` → `VEN-7`。

> いずれも既存（銀行/条約/資産/宗教/文化）を**後退させず接続**する additive 設計。**「私人間の約束を国家が執行する」ミドルレイヤー**が入ることで、法治→通商→繁栄（フェザーン#160）と、法の不平等→資本逃避・報復（#843/#194）の両面が制度として動き出す。文才（`CivilAptitude`）に**軍才と対になる見せ場（法廷）**を与えるのも収穫。
