# ヘロドトス『歴史』 参考設計（EPIC #HRDT）

> 参照元：ヘロドトス『歴史』（古代ギリシアの史書）。**大帝国（専制）と自由都市連合の衝突**（ペルシア戦争）を、各地を旅した著者が**「私はこう聞いた／自分はこう判断する」と出典と検証を明示しながら**語る——伝聞・複数の異説の併記・異民族の風習（ノモス）の相対主義的観察・**僭主の興亡サイクル**・分不相応な拡大が招く破滅（ヒュブリス）を描く、世界最初期の「方法を見せる歴史記述」。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋思想・歴史・群像の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#HRDT` として issue 化する提案。
> 著作権注意：固有名・文章・登場人物・固有設定は流用せず、**史書の語り口・検証手続き・異文化観察・僭主サイクルの構造パターンのみ**を参考にする。架空の勢力・風習で実装する。

## 0. なぜ「ヘロドトス『歴史』」が本システムに役立つか

当プロジェクトは「真実・記録・正統性・異民族」に関わる純ロジックを既に保有している（[CLAUDE.md] 参照）：

| 既存（記録・真実・文化・専制） | カバー範囲 |
|---|---|
| `DisclosureRules`/`DisclosureLedger`（FND-4 #495） | **確定的な連鎖開示**（秘史A→秘史B、不動点まで） |
| VOY-1〜6 解読（#1202） | **確率的・競合的・解なしもありうる解読**（過剰解釈の罠/学派の威信） |
| `EspionageRules`（諜報） | 情報収集・諜報の成功/露見・サボタージュ |
| `EventEngine`/`GameEventDef`（#116） | データ駆動の条件発火→通知/選択肢→効果 |
| `CultureRules`/`Culture`（#194） | 同化圧力・分離独立・ナショナリズム・亡命 |
| `DynastyRules`/`Regime`（#867） | 王朝サイクル・天命・腐敗・正統性 |
| `JusticeRules`（#918 サンデル） | 複数の正義観で行動の是認が変わる |
| CHI 真理のリレー（#1254） | 無形の達成の世代継承・記録に残らぬ貢献 |
| NOTIF `NotificationCenter` | 通知の単一窓口・有界リングバッファ |

**しかし、ヘロドトスが固有に描く「歴史記述という営み自体」**——出来事ではなく**それがどう記録・伝聞・検証され、視点ごとに食い違うか**——は無い：

| ヘロドトス『歴史』が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **伝聞と検証の語り**＝「私はこう聞いた／信じないが記す」＝**出典と信頼度を付けて記録**する | `DisclosureLedger` は「条件を満たせば確定開示」。**証言が出典・信頼度つきで貯まり、複数の異説が併記される**記録層が無い |
| **同じ事件に複数の食い違う異説**を並べ読者に委ねる | VOY は遺物1点の競合解釈。**個々の歴史的事件ごとに勢力/語り手が異なる説を持ち、勝者が"正史"を書く**回路が無い |
| **異民族の風習（ノモス）を相対主義的に観察**＝「習慣こそ万物の王」 | `CultureRules` は同化/分離の力学。**他文化を観察して情報・誤解・畏怖・蔑視が生まれる異文化"認識"**が無い |
| **僭主の興亡サイクル**（簒奪→専横→打倒の反復・僭主譚） | `DynastyRules` は王朝の腐敗サイクル（正統な王権前提）。**非正統に権力を奪った僭主が恐怖支配し打倒される**短周期サイクルが無い |
| **自由（イソノミア）vs 専制の戦闘士気差** | 士気#810・正統性はあるが、**「自分のために戦う自由民」と「鞭で駆られる臣民」の動機差**が戦力に出ない |
| **ヒュブリス＝分不相応な拡大が破滅を招く**（神罰・運命の振り子） | 拡大の創発帰結（補給#94・反乱#109）はあるが、**過剰な成功そのものが反転リスクを生む**抑制軸が無い |
| **神託の多義性＝曖昧な予言が決断を誤らせる** | `EventEngine` は明示効果。**両義的に解釈できる神託**（自国に都合よく読んで破滅）が無い |

**結論**：ヘロドトス『歴史』は当プロジェクトに**「歴史は記録される営みであり、視点ごとに食い違い、勝者が正史を書く」という"歴史記述の認識論"**を与える。`DisclosureLedger`（確定開示）と VOY（遺物解読）の隣に、**証言・出典・信頼度つきで貯まり異説が併記される記録層**を置き、さらに**異文化認識・僭主サイクル・自由vs専制の士気・ヒュブリス・両義的神託**という欠落軸を加える。CHI（無形の達成の継承）と響き、**群像劇と世界観EPIC（秘史/開示）を一段深く**する。

## 1. 役に立つ視点（要約）

ヘロドトス『歴史』の世界観を、**本システムに効く形**で1行ずつ：

1. **歴史は"記録される営み"であり、出典と検証つきで語られる**。「こう聞いた／信じないが記す」＝記録に**信頼度**が付く。→ `DisclosureLedger`（FND-4）の確定開示に**不確かさと出典**を足す＝VOY-2（#1202）の認識論を**事件記録**へ拡張。
2. **同じ事件に複数の食い違う異説が併存し、勝者が正史を書く**。→ 開示#450・正統性`DynastyRules`#867 と共鳴＝「真実は調べれば出る」を「真実は視点で割れ、権力が確定させる」へ。
3. **異文化は観察対象であり、観察は情報にも誤解にもなる**。習慣こそ王＝相対主義。→ `CultureRules`#194 に**他者認識**（情報優位/偏見/畏怖）を足す。
4. **権力の非正統な奪取は短い専横サイクルを生む**（僭主譚）。→ `DynastyRules`#867（王朝サイクル）の**非正統・短周期版**＝下剋上ZUO-4 とも響く。
5. **自由民は自分のために戦い、臣民は駆られて戦う**＝政体が士気を決める。→ 士気#810・`GovernanceRules`#109 へ政体由来の動機係数。
6. **過剰な成功は反転を招く（ヒュブリス）**。→ 拡大に**自滅リスクの抑制軸**＝PRL（真珠湾・戦術勝利→戦略破滅 #1232）と同系の「分不相応の罰」。
7. **曖昧な神託は自分に都合よく読まれ破滅を導く**。→ `EventEngine`#116 に**両義的予言**＝OTH-3（確証バイアス）/BEH（フレーミング #1240）と響く。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`DisclosureLedger`#495・VOY#1202・`CultureRules`#194・`DynastyRules`#867・`EventEngine`#116 を作り直さない**。HRDT はそれらに**「歴史記述の認識論」（出典つき証言・異説併記・異文化認識・僭主サイクル・自由の士気・ヒュブリス・両義神託）**を足し、接続するだけ（additive）。タイクン化回避＝記録・神託・拡大はいずれも高位の決断と創発的帰結で、プレイヤーに台帳のミクロ編集をさせない。

### ★★★ 最優先（真の欠落・HRDT の signature）

#### HRDT 出典つき証言と検証の記録層（伝聞・信頼度・「信じないが記す」）
- `TestimonyRules`（static・test-first）＝出来事が起きると**証言エントリ**（出典＝目撃/伝聞/敵情報/神託、信頼度0..1、勢力視点）が記録層に貯まる。複数の独立証言が一致すると信頼度が上がり、矛盾すると**異説として併記**される。プレイヤー/勢力は信頼度に応じて行動の確度を変える＝**確実な真実は事後に判明**（`no-outcome-preview` と一貫）。
- 接続：`DisclosureLedger`（FND-4＝確定開示の隣に不確かな記録）・VOY-2（#1202 競合解釈の認識論を事件へ）・`NotificationCenter`（証言＝通知の出典つき版）。**有界リングバッファ＋集約で無制限増加を防ぐ**（PERF規律）。

#### HRDT 異説併存と"勝者が書く正史"（複数の真相が権力で確定される）
- `HistoriographyRules`（static）＝1つの歴史的事件に**複数の異説**（勢力ごと/語り手ごと）が結びつく。`TestimonyRules` の信頼度・勢力の威信・支配の継続で**どの説が"正史"に確定するか**が決まる（勝者が書く）。確定後も新証拠（諜報/開示）で**覆りうる**＝歴史修正。
- 接続：HRDT-1（証言）・`DynastyRules`#867（正統性が正史を決める）・開示#450・`DisclosureLedger`。**勝敗の事後ナラティブ**を `BattleManager` 戦績へ薄く接続（正史としての勝者）。

### ★★ 高（異文化認識・僭主サイクル）

#### HRDT 異文化認識（ノモス観察＝他者の風習を観て情報/偏見/畏怖が生まれる）
- `EthnographyRules`（static）＝勢力が他文化を**観察**すると、相手の風習・強み・弱みについて**認識**（精度は接触量と諜報で上がる）が貯まる。認識は**情報優位**（弱点を突く）にも**偏見**（過小評価で奇襲を食う/過大評価で萎縮）にもなる＝相対主義（習慣こそ王＝自文化基準の誤読）。
- 接続：`CultureRules`#194（文化データ＝観察対象）・`EspionageRules`（観察＝情報収集の文化版）・HRDT-1（観察も証言として記録）・BEH（#1240 認知バイアスの源泉）。

#### HRDT 僭主サイクル（非正統な権力奪取→専横→打倒の短周期）
- `TyrannyRules`（static）＝非正統に権力を奪った支配者（僭主）は**正統性が低く恐怖支配に依存**し、抑圧で短期は安定するが**正統性が回復せず打倒リスクが時間で上昇**する短周期サイクル。`DynastyRules`#867（正統な王朝の長周期腐敗）の**対＝非正統の短命**。打倒後は新僭主か正統回復へ分岐。
- 接続：`DynastyRules`#867（王朝サイクルの非正統版）・`PowerRules`#164（傀儡/実権）・ZUO-4 下剋上（簒奪過程）・`CoupRules`#215（宮廷/革命）。

### ★ 中（自由の士気・ヒュブリス・両義神託・lore）

#### HRDT 自由vs専制の戦闘動機（政体が士気の天井を決める）
- `CombatModifiers`#106 に**政体由来の動機係数**を足す＝自由民（民主/共和で**自分のために戦う**）は士気の下限が高く敗走しにくい、専制下の徴募兵は数で勝るが動機が脆い（中央が崩れると瓦解）。`GovernanceRules`#109 の政体×`FleetMorale`#810。実効値パターン（基準非破壊）。
- 接続：`CombatModifiers`#106（係数の一元化窓口へ追加）・`GovernanceRules`#109（政体）・士気#810・FOCUS#872。

#### HRDT ヒュブリス（過剰な拡大が反転を招く抑制軸）
- `HubrisRules`（static）＝勢力の急拡大・連勝・分不相応な遠征が**ヒュブリス値**を蓄積し、補給線の伸びきり・占領地の不安定・士気の慢心と相まって**反転リスク**（離反/天災イベント/決定的敗北の確率）を高める。**過剰な成功そのものが罰を呼ぶ**＝拡大の自己抑制。事後に判明（preview しない）。
- 接続：PRL（#1232 戦術勝利→戦略破滅）と同系・`LogisticsRules`#844（版図一体化＝伸びきり）・`GovernanceRules`#109（占領不安定）・`EventEngine`#116（反転イベントの発火源）。

#### HRDT 両義的神託（曖昧な予言を都合よく読んで誤る）
- `EventEngine`#116 の拡張データ＝**両義的に解釈できる神託イベント**（複数解釈があり、勢力は自分に都合よく読む＝確証バイアス）。誤読のまま決断すると後で逆の意味が判明＝**フレーミングの罠**。コード新設は最小（`GameEventDef` のデータ＋解釈選択）。
- 接続：`EventEngine`#116・OTH-3（確証バイアス）・BEH（#1240 フレーミング）・HRDT-1（神託も出典つき証言）。

#### HRDT（lore）世界観の開示データ（歴史は記録され視点で割れる）
- 「歴史は勝者が書くが、敗者の証言も残る」「習慣こそ万物の王（文化相対主義）」「驕る者は罰される」＝開示データ。**コード新設せず** `DisclosureLedger`（FND-4）への lore データ入力。CCX-6（世界観codex退避）方針に一貫。
- 接続：`DisclosureLedger`（FND-4）・HRDT-2（正史の確定と lore が響く）。

### ❌ 不採用（重複・既存で十分・タイクン化）

| 不採用 | 理由 |
|---|---|
| 確定的な連鎖開示そのもの | **`DisclosureLedger`（FND-4 #495）がカバー**。HRDT は「不確かな証言・異説」を隣に足すだけ |
| 遺物1点の確率的解読・学派の威信賭け | **VOY-1〜6（#1202）がカバー**。HRDT は事件の異説併記へ拡張（住み分け） |
| 文化の同化/分離独立/ナショナリズムそのもの | **`CultureRules`#194 がカバー**。HRDT は「他者の観察＝認識」だけ足す |
| 王朝の長周期腐敗サイクルそのもの | **`DynastyRules`#867 がカバー**。僭主は非正統・短周期の対として足す |
| 諜報の成功/露見の仕組み | **`EspionageRules` がカバー**。異文化観察は文化版の入口として接続のみ |
| 史書編纂UI・年代記の自動文章生成 | タイクン化＝記録は信頼度つきデータと通知で背景的に。文章生成はしない |
| 旅行記・地理探索ミニゲーム | マイクロ操作増。観察は `EthnographyRules` の認識値として抽象化 |

### 推奨着手順
`HRDT-1 → HRDT-2`（出典つき証言＋勝者が書く正史＝HRDT の認識論の核・最も固有で欠落が大きい signature）→ `HRDT-3`（異文化認識＝観察が情報にも偏見にも）→ `HRDT-4`（僭主サイクル＝`DynastyRules` の対）→ `HRDT-5 → HRDT-6`（自由の士気→ヒュブリス＝戦闘と拡大に政体・運命の歯）→ 残り（HRDT-7/8＝両義神託・lore）。

> いずれも既存（開示/文化/王朝/イベント/戦闘係数）を**後退させず接続**する additive 設計。`DisclosureLedger`（確定開示）と VOY（遺物解読）の隣に**出典つきで割れる事件記録**が入り、**「真実は調べれば出る」から「真実は視点で食い違い、勝者が正史を確定し、敗者の証言も残る」**という歴史記述の認識論で世界観EPICと群像劇が一段深くなる。僭主サイクル・自由の士気・ヒュブリス・両義神託が政体と戦争に運命の振り子を与えるのが最大の収穫。
