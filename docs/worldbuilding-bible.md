## 0. トーン / 基調
- PS/PC-98版『銀河英雄伝説IV EX』の手触り（艦隊会戦＋提督の器量＋政治）を**オリジナル設定**で再構築する。
- 既存IPの固有名詞は使わず、**構図のオマージュ**（専制 vs 民主、回廊の攻防、名将たちの群像）に留める。
- 数百年スケールの興亡。英雄の器量が戦局を動かすが、思想・補給・地形が枠を定める。

## 1. 時代設定（数百年後）
- 人類は恒星間に拡散し、かつての統一文明「**旧 銀河連邦**（仮）」は**大崩壊（ザ・コラプス）**で瓦解した。【要・作者判断：崩壊原因＝大戦／資源枯渇／AI災禍 等】
- 崩壊から約 **300 年**。航行可能な宙域は**回廊（コリドー）**で結ばれ、その間は航行困難な**深宇（オフグラフ）**。星系＝拠点、回廊＝戦線。（Phase C のグラフ構造と一致）
- ワープは**時間がかかる**ため、戦略は「どの回廊にいつ戦力を置くか」の読み合いになる。
- 諸勢力が回廊を巡って角逐する**多極時代**。二大勢力＋共産圏＋軍閥群という非対称な四つ巴。

## 2. 勢力（Factions）
> 実装：各勢力は `FactionData`（`factionName`/`color`/`ideology`/`nonHostileFactions`/`legacyFaction`/`ranks`）。
> 既定は「異勢力＝敵」。味方/中立は `nonHostileFactions` で設定。色は下記を `FactionData.color` の出所にする。

### 2-1. 王党派（おうとうは） — 専制・帝政系
- **思想**：強力な君主のもとでの秩序・効率。能力主義の貴族制。
- **色**：深紅 `#D23232`（`legacyFaction=帝国`）。
- **特徴**：規律と練度。整然とした陣形、突破力。旗艦は荘厳・大型。
- **内部対立**【要・作者判断】：門閥（旧貴族）vs 実力派（新貴族）。
- **階級ラダー**（`ranks`、tier 昇順）：准将5／少将6／中将7／大将8／**上級大将9**／元帥10。
  - ※ `RankSystem` 既定の「上級大将9」を持つのは王党派のみ（他勢力は欠番＝大将8の次は元帥10）。

### 2-2. 民主派（みんしゅは） — 共和・同盟系
- **思想**：選挙・合議による主権在民。自由と引き換えに意思決定は遅い。
- **色**：群青 `#3380E6`（`legacyFaction=同盟`）。
- **特徴**：柔軟・粘り強い。防御戦・遅滞戦・世論（士気）に強い。智将型。
- **内部対立**【要・作者判断】：主戦派 vs 和平派、軍と議会の緊張。
- **階級ラダー**：准将5／少将6／中将7／大将8／元帥10（上級大将は無し）。

### 2-3. 共産主義圏 — **非対称勢力**（Issue #17 で深掘り）
- **思想**：党による計画と集団主義。個より全体。
- **色**：橙赤 `#E6661A`（`legacyFaction=帝国` を暫定割当）【要・作者判断】。
- **非対称性**（#17）：独自の能力体系・艦種・編制（例：政治将校による士気底上げ、量で押す人海／物量ドクトリン）。
  穏健派／強硬派の内部対立を勢力内データで表現する想定。
- **階級ラダー**【要・作者判断】：軍事称号を党称号に置換（例：上級司令官＝大将相当8、総司令官＝元帥相当10）。
  `RankSystem.ResolveTier` が欠番を直近 tier に丸めるため、tier さえ合わせれば他勢力と同列比較できる。

### 2-4. 軍閥（ぐんばつ）群 — 中立・地方勢力
- **思想**：イデオロギーより自衛・利権・領袖への忠誠。複数の独立勢力。
- **色**：青緑 `#4DCC66` 系（軍閥ごとに色相をずらす）（`legacyFaction` は隣接大勢力に暫定割当）。
- **特徴**：小規模・玉石混交。傭兵的に転びやすく、`nonHostileFactions` で一時的同盟を表現。
- 既存の検証用「自治領（Autonomous）」アセットはこの系統の雛形として流用可。

> **勢力関係マトリクス（default）**：王党派・民主派・共産は相互に敵対。軍閥は基本中立だが状況で一方に与する。
> 実装は各 `FactionData.nonHostileFactions` で表現（空＝全敵対）。【要・作者判断：恒久同盟の有無】

## 3. 命名規則（Naming Conventions）
> 提督＝`AdmiralData.admiralName`、旗艦名＝部隊表示やシナリオで使用。固有IPと被らないオリジナルで。

- **王党派（独・北欧語感）**：人名は重厚な子音＋称号（例：ヴァルダー、グレーヴェン、ロートシュタイン）。
  旗艦＝古語/神話・猛禽・刃（例：《グングニル》《シュヴァルツアドラー》《ヴェルダンディ》）。
- **民主派（英・ラテン＋理念）**：人名は実務的（例：カーター、ヴェガ、ロウェル）。
  旗艦＝理念・自由・歴史的地名（例：《リバティ》《コンスティチューション》《エヴァーグリーン》）。
- **共産圏（露・象徴語感）**：人名は同志称＋姓（例：同志ヴォルコフ、同志ライサ）。
  旗艦＝理念・労農・赤系（例：《赤い暁》《プロレタリ》《十月》）。
- **軍閥（雑多・地名由来）**：領袖名＋出身星系（例：「ガニメアのドゥーガル」）。旗艦＝通称・獣・俗名。
- **共通ルール**：旗艦名は二重山括弧《》で囲む。提督表示は「姓（＋階級）」。重複回避のため勢力ごとに頭文字レンジを緩く分ける【任意】。

## 4. 未発見宙域（Undiscovered / オフグラフ）
- 回廊網の外側に広がる**深宇**。航行困難で、まだ繋がっていない**未発見回廊**が眠る。
- ゲーム的役割：将来の**マップ拡張フック**（新回廊の“発見”で戦線が増える）、ハザード源（不安定宙域＝ブラックホール等＝`BlackHole`/A-5）、伝説・遺構の舞台。
- Phase C ではまず既知グラフのみ実装。未発見宙域は**フレーバー＋拡張余地**として保持。

## 5. 年表（Timeline スケルトン）
> 西暦は宇宙暦（U.C.）と仮称。数値は仮置き【要・作者判断】。

| 時期 | 出来事 |
|---|---|
| U.C. 0 | 旧 銀河連邦 成立（統一文明の絶頂） |
| U.C. 約 +700 | **大崩壊（ザ・コラプス）**。連邦瓦解、暗黒の数十年 |
| 崩壊後 〜+80 | 回廊網の再発見、星系国家の叢生 |
| 〜+150 | **王党派**台頭（強権による秩序回復）／**民主派**連合の結成 |
| 〜+220 | **共産革命**、第三極の確立 |
| 〜+280 | 軍閥割拠期。回廊を巡る恒常的国境戦 |
| **+300（現在＝ゲーム開始）** | 四つ巴の角逐。プレイヤーはここから群像の一角を担う |

> 既存サンプル会戦（ヴァンフリート星域会戦／アムリッツァ星域会戦／回廊の戦い）は、この年表上の**代表的会戦**として位置づけ可。

## 6. 政治レイヤーとの接続（Issue #14 / RankSystem）
- 階級は `FactionData.ranks` ＋ `RankSystem`（勢力をまたいだ同列判定・昇進の唯一の窓口）で実装済み。
- 提案（#14）は**自階級の tier で範囲制限**（可決可能=青／不相応=赤）。戦果で merit↑→昇進で範囲拡大、という IV EX 再現を本バイブルの階級ラダーに載せる。
- 共産圏は称号が非対称（§2-3）でも、tier を合わせれば同じ政治ロジックで扱える。
