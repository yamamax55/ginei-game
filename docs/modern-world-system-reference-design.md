> 資本主義世界経済を**単一の分業体制**として把握し、地域を**中核/半周辺/周辺**の構造的地位で位置づける——

## 0. なぜ「近代世界システム」が本システムに役立つか

当プロジェクトは経済・内政のマクロ純ロジックと星系間の地理構造を大量に保有している（CLAUDE.md 参照）：

| 既存（マクロ・内政） | カバー範囲 |
|---|---|
| `GovernanceRules`/`Province`（#109） | 単一星系の安定度・統合度・反乱圧力・産出係数 |
| `MarketRules`/`Good`/`Market`（#179-182） | 単一市場の需給均衡価格・裁定→収束 |
| `LogisticsRules`（#844） | 所有星系の連結成分→版図一体化度（自勢力内の連結性） |
| `FiscalRules`/`FiscalState`（#161/162） | 財政・国債・為替・税負担・福祉 |
| `CampaignState`/`CampaignRules` | 全勢力の国家状態統合。`EffectiveStability`（安定度×一体化度）・`LeadingFaction`（現時点の安定度トップ） |
| `DynastyRules`/`Regime`（#867） | 王朝の腐敗→正統性喪失→革命サイクル（政体版の長期循環） |
| `NonviolenceRules`/`Movement`（#831/#832） | 非暴力抵抗・支持転換（反システム運動の形式は既存） |
| `ColonizationRules`（#129） | 未入植系→占領→Province 生成（編入の物理的側面） |
| `ResourceProductionRules`（#93） | 系統別（工業/農業/鉱業/居住）産出・安定度比例 |
| `CapitalRules`/`RedistributionRules`（#917/#163） | 資本集積・格差・再分配（勢力内の垂直的格差） |

**しかし、これらは「単一星系の安定度」または「勢力内の垂直格差」であり、世界システム論が固有に描く以下が欠けている：**

| 世界システム論が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **構造的地位（中核/半周辺/周辺）＝経済ポジションの水平ヒエラルキー** | 星系の「絶対的安定度」はある。しかし**どの星系が高付加価値産業を担い、どれが一次産品を担うか**という構造的ヒエラルキーが無い。`ResourceProductionRules` は系統類型だが地位差（利潤水準）が無い |
| **不等価交換＝交易で周辺の剰余が構造的に中核へ移転** | `MarketRules` は対等な均衡。**地位差に基づく搾取的剰余移転**が無い。均衡の「上に構造を被せる」欠落 |
| **覇権サイクル＝覇権国が生産→商業→金融優位→衰退の多段階を辿る** | `LeadingFaction` は現時点の安定度最大値。**段階的覇権衰退の長期サイクル**が無い。`DynastyRules` は政体サイクルだが経済覇権の段階性が欠ける |
| **半周辺の緩衝機能＝二極化を防ぐ中間層の地位流動** | 星系に構造的地位が無い。半周辺が上昇/転落する**地位流動の動学**が無い |
| **依存的発展＝周辺は中核に従属する形でしか発展しない** | `GovernanceRules.RebelPressure` は占領不安定由来。しかし**構造的搾取が反乱を蓄積する**創発的帰結の論理が薄い |

**結論**：世界システム論は当プロジェクトの「単一星系内の安定度」と「勢力間の軍事均衡」に、**「星系間の構造的分業ヒエラルキー」**という第三の軸を与える。`CampaignState`（最上層統合）に**一段上から構造を被せる**形で、既存のマクロ経済を後退させない。

## 1. 役に立つ視点（要約）

世界システム論の構造パターンを**本システムに効く形**で1行ずつ：

1. **「銀河は単一の分業体制」＝星系間の経済ポジションを構造化する視点**。個別星系の安定度でなく、銀河規模の**どこが中核でどこが周辺か**が見える。→`CampaignState.EffectiveStability`（現状＝勢力内平均）を「システム内地位」で補完。既存issue #844/#109 との接続。
2. **「中核は高付加価値・周辺は低付加価値の一次産品」＝利潤水準の地位差が産業高度化の戦略目標を生む**。→既存 `ResourceProductionRules`（系統別産出）に地位という一段上の構造を被せる（系統は不変・地位係数が乗る）。
3. **「不等価交換で搾取し、搾取が反乱を育てる」＝搾取の創発的帰結**。制覇した銀河が腐る理由を与える。→`GovernanceRules.RebelPressure` に搾取由来の不満を上乗せ。強い帝国が長期的に腐敗する必然性。
4. **「覇権は交代する：生産→商業→金融→衰退」＝覇権国の段階的衰退サイクル**。`DynastyRules`（王朝サイクル）の経済覇権版。→「このまま拡大すれば覇権が揺らぐ」という長期警告機能。
5. **「半周辺は緩衝帯＝二極化を防ぐ」＝中間的な星系が全員に取り込みインセンティブを与える**。単純な武力制圧から「構造的地位の操作」という知的戦略軸へ（タイクン化回避＝地位変動は暦 Tick で自動）。
6. **「反システム運動＝搾取が生む抵抗」**。`NonviolenceRules`/`JusticeRules` との接続で、支配の創発的崩壊を表現。→世界システムの終焉とエンディング（`DisclosureLedger` FND-4）へ。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`GovernanceRules`/`MarketRules`/`LogisticsRules`/`CampaignState` を作り直さない**。WSYS はそれらに**欠落軸を足し、一段上から被せる**だけ（additive）。

### ★★★ 最優先（真の欠落・世界システム論の signature）

#### WSYS-1 構造的地位（中核/半周辺/周辺）と利潤水準の地位差

- **WorldRole**：各星系（または各勢力）に `WorldRole{中核,半周辺,周辺}` を付与。地位＝産業の**利潤水準**の代理変数（中核＝高付加価値、周辺＝低付加価値一次産品の抽象）。
- **利潤係数**：地位に応じた `ProductionMultiplier`（中核>半周辺>周辺）。`ResourceProductionRules` の産出に地位係数を乗算（基準値は非破壊）。
- 接続：新 `WorldSystemRules`/`WorldPosition`（純ロジック・test-first）。`Province` に `WorldRole` を付与 →`ResourceProductionRules` の係数へ合流。タイクン化回避＝地位は1変数（細分化しない）。

#### WSYS-2 不等価交換＝周辺の剰余が中核へ移転（構造的搾取）

- **剰余移転**：交易/版図内産出の配分で、**周辺の産出が中核へ構造的に移転**する（地位差×連結性の係数）。`MarketRules` の均衡は不変＝その上に移転項を乗せる。
- 接続：`WorldSystemRules`（WSYS-1）×`LogisticsRules`（連結性）×`GovernanceRules.OutputFactor`。EditMode テストで移転量・地位差比例を固定。

### ★★ 高（長期サイクル・地位流動・反乱の帰結）

#### WSYS-3 ヘゲモニー循環（生産→商業→金融優位→衰退のサイクル）

- **覇権段階**：覇権国（`CampaignState.LeadingFaction`）が**生産優位→商業優位→金融優位→衰退**と段階を移り、金融化＝衰退の兆候として次の覇権へ交代。
- 接続：新 `HegemonyRules`/`HegemonyCycle`（純ロジック）。`DynastyRules` と同型（王朝サイクルの経済覇権版）。`CampaignState.LeadingFaction` の**段階拡張**（既存 API は不変・上に被せる）。

#### WSYS-4 半周辺の上昇/転落（地位流動・産業政策）

- 地位は固定でなく、**産業政策・蓄積・覇権の波及**で `周辺→半周辺→中核`／逆方向に動く。半周辺は上昇も転落もする緩衝帯。地位上昇を**戦略目標**化（高位の決断＝産業投資の方向）。
- 接続：WSYS-1。`GovernanceRules` 収束流儀（目標地位へ収束）＋`CampaignRules.Tick`（暦境界 Tick）相乗り。

#### WSYS-5 依存的発展と反システム運動（搾取→周辺不安定化→反乱圧力）

- 周辺は中核への**従属の形でしか発展しない**（依存的発展）。搾取は短期に中核を富ませるが、周辺の**不安定化・反システム運動**（反乱圧力）を蓄積する——搾取の創発的帰結が面白さの核。
- 接続：WSYS-2→`GovernanceRules.RebelPressure`／`CampaignState.IsCollapsing`／`JusticeRules`（不満な正義観）／`NonviolenceRules`（反システム運動の形式を再利用）。

### ★ 中（配線・観測・lore）

#### WSYS-6 世界システムを CampaignState へ配線＋観測層

- 地位/覇権段階/剰余移転を `CampaignState` に組み込み、観測層（`CoreStateInspector` 用語集または `GalaxyView`）でライブ表示。ここで初めて盤面へ接続。
- 接続：`CampaignState`/`CampaignRules`。`CoreStateInspector` に新フィールド（独立新ルートなら `Register` 1行）。

#### WSYS-7 （lore）世界システムの開示データ

- 「銀河は単一の分業体制」「覇権は交代する」「従属するほど反乱が育つ」「反システム運動が世界を変える」型の lore データ。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**のみ。

### ❌ 不採用（重複・既存で十分）

| 不採用 | 理由 |
|---|---|
| 単一市場の需給均衡・価格・裁定 | **`MarketRules`#179-182＋SAW 裁定**がカバー。WSYS は均衡の上に構造移転を被せるだけ |
| 通貨・為替・通貨改鋳 | **`FiscalRules.ExchangeRateFactor`＋SAW `CoinageRules`**がカバー |
| 版図の物流・連結・補給線 | **`LogisticsRules`#844/`SupplyRules`#94**がカバー。WSYS は一体化度の上に分業を被せる接続のみ |
| 階級別税・再分配・資本集積 | **`RedistributionRules`/`CapitalRules`#917**がカバー。WSYS は星系間の空間的搾取に限定 |
| 覇権国の軍事興亡そのもの | 戦略レイヤー既存。WSYS は経済覇権の段階循環のみ |
| 多段加工・商品ごとの生産ツリー | タイクン化回避＝地位＝利潤水準の1変数抽象（細分化しない） |
| 個々の企業/商人スケール取引 | **FRM#1022・SAW ネームド商人**がカバー。WSYS は構造マクロ専任 |
| コンドラチェフ波の独立実装 | 覇権サイクル（WSYS-3）と `DynastyRules` の合成で代替。独立 enum 不要 |

### 推奨着手順

`WSYS-1`（構造的地位＝最も固有で欠落の大きい signature）→ `WSYS-2`（不等価交換＝搾取の核）→ `WSYS-3`（ヘゲモニー循環＝長期警告）→ `WSYS-4`（地位流動＝戦略目標）→ `WSYS-5`（反システム運動＝搾取の帰結）→ `WSYS-6`（配線）→ `WSYS-7`（lore）。

> いずれも既存マクロ経済・内政・統合層を**後退させず接続**する additive 設計。`CampaignState` 最上層に最も効く。
