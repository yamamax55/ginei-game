# ヴォイニッチ手稿 参考設計（EPIC #VOY）

> 参照元：ヴォイニッチ手稿（15世紀頃の写本）。**未知の文字体系**で記され、植物画・天文図・人体図・薬草図を伴うが、**100年以上の研究を経ても解読されていない**実在の遺物。暗号説・失われた言語説・無意味説（ホークス的でっち上げ）が競合し続ける。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋開示エンジン/秘史/研究の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#VOY` として issue 化する提案。
> 著作権注意：実在の遺物だが固有の図版・文字は流用せず、**「解読不能の遺物」をめぐる認識・研究・真贋の構造パターンのみ**を参考にする。

## 0. なぜ「ヴォイニッチ手稿」が本システムに役立つか

当プロジェクトは開示・秘史・研究の純ロジックを保有している（[CLAUDE.md] 参照）：

| 既存（開示・秘史・研究） | カバー範囲 |
|---|---|
| `DisclosureLedger`/`DisclosureRules`（FND-4 #495） | 秘史の**連鎖開示**（前提→条件→開示・不動点まで） |
| 世界観EPIC（秘史#450/天井CAP/エンディング） | 開示が世界観・エンディングを解放 |
| `ResearchRules`（#123-127・設計のみ） | 技術ツリー・研究投資 |
| `EspionageRules` | 情報入手・サボタージュ |
| ホーガン『星を継ぐもの』（バックログ） | 科学的推理で秘史を解く（開示の理想形） |
| `EventEngine`#116 | 条件発火イベント |

**しかし、`DisclosureLedger` は「条件を満たせば真実が確定的に開示される」決定論モデル**であり、ヴォイニッチ手稿が固有に体現する以下が**欠けている**：

| ヴォイニッチ手稿が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **解読の不確実性**＝研究を投じても部分的にしか解けない・完全解読に至らない | 開示は0/1（隠れている/開いた）。**確率的・部分的な解読の漸進過程**が無い |
| **競合する解釈仮説**（暗号説/言語説/無意味説が並立） | 開示される真実は一意。**複数の競合解釈が威信を賭けて並立し、後で覆る**回路が無い |
| **真贋・過剰解釈の罠**＝意味を見たい欲望が偽の解読を生む | 開示は常に真。**誤読／偽の開示（後で否定される）／確証バイアス**が無い |
| **来歴（プロヴェナンス）が手がかり**＝誰の手を経たかが解読/真贋を左右 | 秘史は所在のみ。**遺物の伝来経路が研究価値・贋作リスクを決める**回路が無い |
| **研究者・学派の威信賭け**＝解読競争が学界の名声を動かす | 研究は技術獲得。**解読仮説に学者が名声を賭け、当たれば殿堂・外せば失墜**が無い |
| **無意味の可能性**＝そもそも解くべき意味が無いかもしれない | 秘史は必ず意味を持つ。**「解なし」もありうる**不確実性が無い |

**結論**：ヴォイニッチ手稿は当プロジェクトの開示エンジンに**「不確実な開示」という第二モード**を与える。`DisclosureLedger`（確定的な連鎖開示）の隣に、**確率的・部分的・競合的・誤読しうる解読**を置く＝「真実は調べれば確定的に出てくる」naive な秘史を、**「真実は不確かで、解釈は競合し、過剰解釈は罠で、解なしもありうる」**という認識論的に豊かな秘史へ拡張する。これは世界観EPIC（秘史#450・ID vs 進化論・啓蒙）の**メタ層**＝「我々は世界をどこまで・どう知りうるか」を遊びにする。

## 1. 役に立つ視点（要約）

1. **遺物は解けるとは限らない**。研究投資は部分解読を進めるが、完全解読も「解なし」もありうる＝確定的でない秘史。
2. **解釈は競合する**。同じ遺物に暗号説/言語説/無意味説が並立し、証拠で勝敗が揺れ、新発見で覆る。
3. **見たいものが見える罠**。パターンを見出したい欲望が偽の解読（過剰解釈）を生み、確証バイアスで強化される。
4. **来歴が価値を決める**。誰の手を経たか＝研究の手がかりであり、贋作の温床でもある。
5. **学者は仮説に名を賭ける**。当たれば殿堂、外せば失墜＝研究は威信のギャンブル。
6. **解読は世界観を開く（が、確実ではない）**。解読の進展が秘史#450・エンディングを解放するが、誤読は偽のエンディングへ誘いうる。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`DisclosureLedger`（FND-4）・`ResearchRules`・`EventEngine` を作り直さない**。VOY は開示に**「不確実モード」**を足し、接続するだけ（additive）。`DisclosureLedger` の確定的開示は**そのまま温存**＝VOY は遺物（artifact）に対する別経路。タイクン化回避＝解読は研究投資の高位決断、解釈の競合・誤読は創発。

### ★★★ 最優先（真の欠落・VOY の signature）

#### VOY 遺物と漸進解読（確率的・部分的なディサイファ）
- `Artifact`（純データ＝未解読の遺物・断片群・真の意味は隠れ変数）＋`DeciphermentRules`（static・test-first）＝研究投資（`ResearchRules` 相乗り）で**解読度（0..1）が確率的に進む**。部分解読で**部分的な開示**が出る。完全解読・頭打ち（解読不能）・**解なし**（無意味だった）の三終端。
- 接続：`ResearchRules`#123-127（投資）・`DisclosureLedger`（部分開示の出力先＝確定開示と並ぶ第二経路）・FND-2 セーブ（解読度の永続化）。

#### VOY 競合解釈と仮説の並立（暗号説/言語説/無意味説）
- 1遺物に複数の**解釈仮説**（`Interpretation`）が並立。各仮説は証拠（解読断片）で**信頼度**を持ち、新断片で上下し、優勢仮説が**暫定の真実**として開示に使われる。後の発見で**覆る**（パラダイム転換 `ResearchRules`/クーン型）。
- 接続：`DeciphermentRules`・`DisclosureRules`（暫定開示＝後で撤回されうる）・`EventEngine`（覆りイベント）。

### ★★ 高（誤読・来歴）

#### VOY 過剰解釈と偽の開示（確証バイアスの罠）
- 解読度が低いのに**確信度が先行**すると**偽の開示**（後で否定される誤読）が出る＝見たいものを見る罠。誤った解釈に基づく行動はペナルティ（偽のエンディングへ誘導・資源の浪費）。解毒は新証拠・対立学派の反証・解読度の上昇。
- 接続：`DisclosureLedger`（撤回フラグつき開示）・OTH-3 疑心の自走（確証バイアスと同型ロジック流用可）・`JusticeRules`/学派対立。

#### VOY 来歴（プロヴェナンス）と真贋
- 遺物の**伝来経路**（所有者の鎖）が研究の手がかり（経路が信頼の証）かつ**贋作リスク**（経路が怪しいと偽物の可能性）を決める。来歴調査は諜報・外交で進み、贋作と判明すれば解読は徒労（が、誰が・なぜ作ったかという別の秘史へ）。
- 接続：`EspionageRules`（来歴調査）・OWN#1032（遺物の所有権/移転）・VEN（贋作＝詐欺契約）・`DisclosureLedger`。

### ★ 中（威信・lore）

#### VOY 学派と威信のギャンブル
- 学者（`Person`/`ICharacter`）が解釈仮説に**名声を賭ける**＝担いだ仮説が勝てば殿堂#785・負ければ失墜（人事・支持#113）。解読競争が学界の派閥#113を動かす。
- 接続：`Person`/列伝#784/殿堂#785・内部勢力#113・`CareerPipelineRules`（学閥）。

#### VOY（lore）認識の限界・解なしの哲学
- 「解けない遺物は世界に知の限界を突きつける」「無意味かもしれないものに意味を投影する人間」＝開示の**メタ哲学**データ。世界観EPIC（啓蒙/ID vs 進化論/天井CAP＝知の天井）と直結。
- 接続：`DisclosureLedger`（FND-4）・世界観codex。コード新設は最小。

### ❌ 不採用（重複・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 開示エンジンの再実装 | **`DisclosureLedger`（FND-4）がカバー**。VOY は不確実モードを足すだけ |
| 研究ツリーそのもの | **`ResearchRules`#123-127 がカバー**。解読は投資先として相乗り |
| 実際の暗号解読パズル・ミニゲーム | タイクン化/別ジャンル。解読は確率過程として抽象化 |
| 実在の図版・文字体系の再現 | 著作権/別作業。構造（未解読遺物の認識論）のみ |
| 確証バイアスの新規実装 | **OTH-3 #1187 のロジックを流用**（疑心の自走と同型） |

### 推奨着手順
`VOY-1 → VOY-2`（漸進解読＋競合仮説＝不確実開示の核）→ `VOY-3`（誤読の罠＝signature）→ `VOY-4`（来歴・真贋）→ 残り（VOY-5/6）。

> いずれも既存（開示/研究/諜報）を**後退させず接続**する additive 設計。`DisclosureLedger` の確定開示の隣に**「不確実・競合・誤読しうる解読」**を置くことで、秘史#450・世界観EPICが**「真実は調べれば出る」から「真実は不確かで解釈は争い解なしもありうる」**という認識論的に豊かなものになる。**ホーガン『星を継ぐもの』（科学的推理で秘史を解く）の影**＝謎→仮説→検証→（時に誤読）→開示の連鎖が、世界観の背骨として動き出すのが最大の収穫。
