# 『アラブが見た十字軍』参考設計（EPIC #CRSD）

> 参照元：アミン・マアルーフ『アラブが見た十字軍』（同一の戦争を、勝者＝侵攻側でなく**敗者・被侵攻側（アラブ／イスラーム世界）の年代記から再構成した史書**）。
> 「西欧から見れば聖地解放の聖戦、東方から見れば野蛮な異教徒の侵略」——**同じ事件が、立場によって正反対の意味づけで記録される**こと。被侵攻側の**政治的分裂**が敗北を招いたこと。憎悪と蔑視の前線が、やがて**休戦・交易・文化的賛嘆の接触帯**へ変わること。そして戦争が終わっても**何世紀も残る遺恨（長い記憶）**を植えること——を描く「視点反転の史書」。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋歴史・開示・文化・外交の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#CRSD` として issue 化する提案。
> 著作権注意：固有名・実在人物・王朝名・地名・宗教名・文章は一切流用せず、**「同一戦争の視点反転記述／被侵攻側の分裂／前線の接触帯／遺恨の長期記憶」という構造パターンのみ**を参考にする。架空の勢力・思想で実装する。

## 0. なぜ『アラブが見た十字軍』が本システムに役立つか

当プロジェクトは「記録・真実・文化・外交・遺恨」に関わる純ロジックを既に保有している（[CLAUDE.md]「テスト基盤」節 参照）：

| 既存（記録・真実・文化・外交・正統性） | カバー範囲 |
|---|---|
| `DisclosureRules`/`DisclosureLedger`（FND-4 #495） | 条件→確定的な連鎖開示（秘史A→秘史B、不動点まで） |
| HRDT 証言・異説併存（`TestimonyRules`/`HistoriographyRules`） | 出典つき証言・**複数の異説が併存**し勝者が"正史"を確定する |
| NTEF 記録改竄（`SecurityRules`系・公式言説） | **国家が真実を能動的に捏造・上書き**する（記録省の逆向き） |
| `ReligionRules`（#172-175）・QUR/PRPH | 改宗圧力・異端・**聖戦圧力**・宗派分裂 |
| `CultureRules`/`Culture`（#194） | 同化圧力・分離独立・ナショナリズム・亡命 |
| `DiplomacyRules`/`TreatyRules`（DIP-1/2 #190/#191） | 二国間外交状態・条約・同盟・不可侵 |
| `WarGoalRules`（DIP-3 #192） | casus belli・厭戦・講和受諾 |
| SGZ-1 多極均衡（#1103）／ZUO-2 会盟（#1195） | 最強台頭→弱小連衡・多国間盟約・違背の討伐 |
| `LoyaltyRules`/関ヶ原（#817） | 諸侯の分裂・調略・寝返りカスケード |

**しかし、これらが扱わない『アラブが見た十字軍』固有の「視点反転の史書」**——勝者でなく**敗者・被侵攻側の目で同一戦争を見直す**こと——がある：

| 『アラブが見た十字軍』が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **同一事件が立場で正反対の道徳的意味を帯びる**（解放の聖戦 ⇔ 野蛮な侵略） | HRDT は「異説が併存し勝者が確定」、NTEF は「国家が捏造」。だが**1つの確定事実に、各勢力の記録が自動で正反対の価値を付与（自己正当化フレーム）**し、中立の真実が原理的に無い回路が無い |
| **被侵攻側の政治的分裂が敗北の主因**（各個撃破され、対抗統一は手遅れに来る） | `LoyaltyRules`/SGZ-1 は会戦/外交の分裂。だが**外圧（侵略）に対して被侵攻側が分裂で各個撃破され、危機が遅れて対抗統一を生む**という防衛側分裂の力学が無い |
| **前線が憎悪→接触帯へ変わる**（敵を野蛮人と蔑視しつつ、休戦・交易・技術/文化の摂取・相互賛嘆が進む） | `CultureRules` は領内の同化/分離。**交戦中の宗教/文明の境界線上で、蔑視と実務的協力・文化摂取が同時に進む接触帯**が無い |
| **戦争が終わっても遺恨が何世紀も残る**（侵略の記憶が後世の casus belli・敵対の鋳型になる） | `WarGoalRules` の厭戦は戦中で消える。**講和後も世代を超えて持続し、後の開戦動機・敵対係数に化ける長期記憶（歴史的遺恨）**が無い |
| **聖戦の論理が両側で動員に使われる**（防衛側も対抗聖戦で初めて統一する／聖戦は内部統制の道具） | `ReligionRules.HolyWarPressure` は改宗/異端の宗教均衡。**外部の聖戦が被侵攻側の対抗聖戦を誘発し、それが分裂した防衛側を遅れて束ねる**という動員の鏡像が無い |

**結論**：『アラブが見た十字軍』は当プロジェクトに**「歴史は勝者の一視点でなく、敗者側から見ると正反対に見える」という視点反転の認識軸**を与える。HRDT（異説併存・勝者が正史）と NTEF（国家が真実を捏造）の隣に、**①同一事実の二面的価値づけ（各勢力が自動で自己正当化フレームを生む）②被侵攻側の分裂→遅すぎる対抗統一 ③憎悪と接触が同居する前線 ④講和後も残る歴史的遺恨**という4本の欠落軸を置く。これは開示エンジン FND-4 の「両面記述」（同一秘史を視点別に語る）を実装で支え、`LoyaltyRules`/`ReligionRules`/`CultureRules`/`WarGoalRules` を**後退させず接続**する additive 設計。プレイヤーに**「勝った戦争が、相手側の年代記では侵略として千年残る」**という長期の業（カルマ）を体験させる。

## 1. 役に立つ視点（要約）

『アラブが見た十字軍』の世界観を、**本システムに効く形**で1行ずつ：

1. **同じ戦争に中立の真実は無く、各勢力が正反対の意味で記録する**。一方の解放は他方の侵略。→ HRDT 異説併存・FND-4 両面記述（#450）に**「1事実→視点別の道徳的価値づけ自動生成」**を足す。
2. **被侵攻側は分裂しているから負ける**。各個撃破され、危機が深まって初めて対抗統一が（多くは手遅れに）生まれる。→ `LoyaltyRules`#817・SGZ-1#1103・ZUO-2#1195（連衡/会盟）の**外圧版＝防衛側分裂**。
3. **聖戦は侵略にも防衛にも使える両刃の動員装置**。外部の聖戦が対抗聖戦を呼び、分裂した側を遅れて束ねる。→ `ReligionRules.HolyWarPressure`#172 に**鏡像（防衛側の対抗聖戦）**を足す。
4. **前線は憎しみと接触が同居する**。敵を蔑視しながら休戦・交易・技術と文化を摂取し、ときに相互賛嘆へ。→ `CultureRules`#194・`DiplomacyRules`#190 に**国境接触帯（蔑視と文化摂取の同時進行）**を足す。
5. **戦争の傷は講和では消えない**。侵略の記憶は世代を超え、後世の開戦動機・敵対の鋳型になる。→ `WarGoalRules`#192 の厭戦の**長期版＝歴史的遺恨**＝後の casus belli。
6. **（lore）勝者の歴史と敗者の歴史は別物**。同じ事件を勝者は栄光、敗者は災厄として伝える。→ コード新設せず `DisclosureLedger`（FND-4）への**両面 lore データ入力**。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`DisclosureLedger`#495・HRDT（証言/異説）・NTEF（記録改竄）・`ReligionRules`#172・`CultureRules`#194・`DiplomacyRules`#190・`WarGoalRules`#192・`LoyaltyRules`#817 を作り直さない**。CRSD はそれらに**「視点反転の史書」（同一事実の二面価値づけ・防衛側分裂・接触帯・歴史的遺恨）**を足し、接続するだけ（additive）。タイクン化回避＝記述・統一・遺恨はいずれも高位の決断と創発帰結で、プレイヤーに台帳のミクロ編集はさせない。

### ★★★ 最優先（真の欠落・CRSD の signature）

#### CRSD 同一事件の視点別価値づけ（両面記述＝中立の真実が無い）
- `PerspectiveRules`（static・test-first）＝1つの確定した出来事（占領/会戦/虐殺/和約）に対し、各勢力の記録が**立場依存の道徳的価値**（解放/正義 ⇔ 侵略/暴虐）を**自動付与**する。価値は「自勢力が加害か被害か×思想一致度×信仰差」で決まり、**同一事実から正反対のフレーム**が生まれる（中立点が原理的に無い）。HRDT の「異説併存（事実が割れる）」とは直交＝**事実は1つ、意味が割れる**。
- 接続：HRDT `HistoriographyRules`（異説の隣＝事実共有でも価値が割れる）・`DisclosureLedger`#450（両面記述＝同一秘史の視点別開示）・`JusticeRules`#918（正義観で是認が変わる）。**有界化＝事件は集約しリングバッファで保持**（PERF規律）。

#### CRSD 被侵攻側の分裂と遅すぎる対抗統一（各個撃破→危機統一）
- `FragmentedDefenseRules`（static）＝外部勢力の侵攻に対し、被侵攻側が**分裂している間は各個撃破される**（個別の領主/勢力が連携せず逐次投入され敗北）。侵攻の進行（喪失領域）が閾値を超えると**対抗統一圧力**が高まり、遅れて連合が成立する＝**多くは手遅れ**だが反攻の核になる。`LoyaltyRules` の調略カスケードの**外圧版**。
- 接続：`LoyaltyRules`#817（分裂/連合の不動点反復を流用）・SGZ-1#1103（弱小連衡の外圧トリガー）・ZUO-2#1195（会盟）・`DiplomacyRules`#190（対抗同盟の成立）。

### ★★ 高（聖戦の鏡像・前線の接触帯）

#### CRSD 対抗聖戦＝聖戦の鏡像動員（侵略の聖戦が防衛の聖戦を呼ぶ）
- `CounterCrusadeRules`（static）＝外部勢力が**聖戦**を掲げて侵攻すると、被侵攻側に**対抗聖戦圧力**が蓄積し、一定で**防衛側を信仰の旗の下に統一**する（CRSD-2 の対抗統一に宗教的動機を与える）。聖戦は士気を上げるが、**内部の異端/宗派対立を一時棚上げにするだけで根治しない**（収まれば再分裂）。
- 接続：`ReligionRules.HolyWarPressure`#172（鏡像＝対抗聖戦）・CRSD-2（対抗統一の宗教版トリガー）・士気#810・PRPH 宗派分裂（棚上げの再燃）。

#### CRSD 国境接触帯（蔑視と文化摂取の同時進行）
- `FrontierContactRules`（static）＝敵対する文明/信仰の境界線（前線星系/回廊）で、交戦と並行して**蔑視（偏見係数）**と**実務的接触（休戦・交易・技術/文化の摂取・相互賛嘆）**が同時に進む。接触が長いほど偏見が薄れ文化摂取が進むが、聖戦圧力（CRSD-3）が高いと接触が止まり蔑視へ揺り戻す＝**憎悪と理解の綱引き**。
- 接続：`CultureRules`#194（同化/分離の境界版）・`DiplomacyRules`#190（局地休戦）・通商#94・`ResearchRules`#123（敵文化からの技術摂取）。

### ★ 中（長期遺恨・両面 lore）

#### CRSD 歴史的遺恨（講和後も残る長期記憶＝後の casus belli）
- `GrievanceRules`（static）＝侵略/虐殺/聖地喪失などの出来事が、講和後も**遺恨**として被害側に蓄積し、**世代を超えて減衰しつつ持続**する。遺恨は後の casus belli の正当性・敵対係数・対抗聖戦の着火しやすさに化ける＝**戦争は終わっても終わらない**。`WarGoalRules` の厭戦（戦中で消える）の長期対偶。
- 接続：`WarGoalRules`#192（厭戦の長期版＝遺恨→後の戦争目的）・`DiplomacyRules`#190（opinion の歴史的下方バイアス）・CRSD-1（価値づけの記録が遺恨の源）・LIFE 世代#151。

#### CRSD（lore）勝者と敗者の二重史（世界観の両面開示データ）
- 「同じ事件を勝者は栄光、敗者は災厄と伝える」「侵略者もやがて土地に根づき混じる」「聖戦は内を束ねる外向きの装置」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**両面 lore データ入力**（同一秘史を勝者視点／敗者視点で対に登録＝CRSD-1 の価値づけを物語化）。世界観codex退避方針に一貫。

### ❌ 不採用（重複・既存で十分）

| 不採用 | 理由 |
|---|---|
| 出典つき証言・異説併存・"勝者が正史を書く" | **HRDT（`TestimonyRules`/`HistoriographyRules`）が既にカバー**。CRSD は「事実が割れる」でなく「価値が割れる」を足すだけ（CRSD-1 は直交） |
| 国家による記録の能動的捏造・改竄 | **NTEF（記録改竄・公式言説）がカバー**。CRSD は捏造でなく**立場依存の自動価値づけ**＝別軸 |
| 改宗圧力・異端・宗派分裂そのもの | **`ReligionRules`#172・PRPH/QUR がカバー**。CRSD は**対抗聖戦という鏡像動員**のみ足す |
| 領内の同化・分離独立・ナショナリズム | **`CultureRules`#194 がカバー**。CRSD は**前線の接触帯**のみ足す |
| 二国間の外交状態・条約・同盟・casus belli の基盤 | **`DiplomacyRules`/`TreatyRules`/`WarGoalRules`（DIP）がカバー**。CRSD は遺恨を**接続**するだけ |
| 諸侯の調略・寝返りカスケード（会戦/内政） | **`LoyaltyRules`#817 がカバー**。CRSD-2 はそのカスケードを**外圧版に流用**（重複新設しない） |
| 危機の偶発点火・同盟連鎖参戦・動員の先制圧力 | **SARA（サラエボ）がカバー**。CRSD は侵略への防衛側分裂が主眼＝別軸 |

### 推奨着手順
`CRSD-1`（同一事件の視点別価値づけ＝最も固有で欠落の大きい signature・他子の土台）→ `CRSD-2`（防衛側分裂＝戦争の構造）→ `CRSD-3`（対抗聖戦＝CRSD-2 に宗教動機）→ `CRSD-4`（接触帯＝前線のテクスチャ）→ `CRSD-5`（遺恨＝CRSD-1 の長期帰結）→ `CRSD-6`（lore＝コード新設なし・最後に物語化）。

> いずれも既存の記録（HRDT/NTEF）・宗教・文化・外交を**後退させず接続**する additive 設計。HRDT（異説併存）とは「事実が割れる ⇔ 価値が割れる」で直交し、開示 FND-4 の「両面記述」を実装で支える。
