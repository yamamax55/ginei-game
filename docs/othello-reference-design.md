# オセロー 参考設計（EPIC #OTH）

> 参照元：シェイクスピア『オセロー』（17世紀初頭）。実力で登り詰めた**外様の将軍**が、昇進人事に恨みを持つ腹心の**讒言と偽証拠**（ハンカチ）で疑心を植え付けられ、**自軍の内側から崩壊**していく悲劇。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋諜報・群像・人事の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#OTH` として issue 化する提案。
> 著作権注意：原作はパブリックドメインだが、固有名・文章・キャラクターは流用せず、**信頼破壊・人事・諜報の構造パターンのみ**を参考にする。**現実の人種・民族表象は採らない**＝「外様（実力登用の異邦人）」という構造のみ架空勢力で一般化する。

## 0. なぜ「オセロー」が本システムに役立つか

当プロジェクトは諜報・人事・関係の純ロジックを保有している（[CLAUDE.md] 参照）：

| 既存（諜報・人事・関係） | カバー範囲 |
|---|---|
| `EspionageRules` | 諜報任務の成功/発覚・情報入手・サボタージュ |
| SGZ-4 離間の計（#1106） | **勢力ペア**の opinion 工作＝同盟崩し |
| MKV #1138・LEAR-3（#1178） | 佞臣の進言品質・忠誠テストの逆選択 |
| SLN-1 関係グラフ（#1146） | 人物ペアの親愛/反目/恩義/宿怨 |
| `SeniorityRules`/`MeritRankRules`#900 | 席次・軍功授爵・昇進 |
| `CommandStaffRules`#885 | 指揮班（提督/副提督/参謀）の能力反映 |
| #169 門地開放・VEN-5（#1171） | 実力主義登用・異邦人の法的地位 |

**しかし、オセローが固有に描く「内側からの破壊」**は無い：

| オセローが固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **讒言・偽証拠の植え付け**（ハンカチ）＝標的を失脚させる工作 | 諜報は情報入手/サボタージュのみ。**人物の信用を破壊する工作（失脚工作）**が無い |
| **個人間の離間**＝上官と腹心の信頼を壊し指揮系統を内部崩壊させる | SGZ-4 は**勢力間**。**人物ペアの信頼破壊**→指揮班#885の性能劣化が無い |
| **疑心の自己強化**＝植えられた疑念は調べるほど黒く見える（確証バイアス） | 関係値は外部イベントで動くのみ。**疑念が自走する**動学が無い |
| **外様将軍の脆弱性**＝実力登用の異邦人は功績で買われるが帰属が不安定 | #169/VEN-5 は登用と法的地位。**「外様であること」が工作の付け入り口になる**係数が無い |
| **昇進の恨み＝裏切りの製造**（イアーゴは昇進を飛ばされた） | 席次/軍功はある。**人事の不公平→怨恨→内部工作の動機**という因果が無い |
| **工作の露見**＝実行者の身内（エミリア）が最後に暴く | 発覚は確率のみ。**協力者・身内からの暴露**という露見経路が無い |

**結論**：オセローは当プロジェクトに**「人物を狙う諜報＝失脚工作」と「組織が内側から壊れる回路」**を与える。外への諜報（SGZ-4 離間・SAW-3 風説）は揃った——残るは**内への刃**：人事の不公平が怨恨を生み（動機）、外様の脆弱性が入口になり（標的選定）、偽証拠が疑心を植え（手段）、疑心が自走して（増幅）、指揮班が壊れる（帰結）。**銀英伝の「査問会」「内部の足の引っ張り合い」**そのもの。

## 1. 役に立つ視点（要約）

1. **最強の攻撃は内側から**。艦隊を撃つより、名将と腹心の信頼を壊す方が安い＝失脚工作は武力の代替。
2. **疑心は植えれば自分で育つ**。偽証拠は一度でいい——確証バイアスが残りをやってくれる（調べるほど黒く見える）。
3. **人事の不公平は敵を製造する**。昇進を飛ばされた者の怨恨は、外の敵より危険な内の敵になる。
4. **外様は強くて脆い**。実力登用の異邦人は功績で買われるが、帰属の不安が工作の付け入り口になる。
5. **工作は身内から露見する**。完璧な計略の綻びは実行者の周辺（協力者・家族）から生まれる。
6. **平時は武人を持て余す**。戦争が消えた将軍の手持ち無沙汰は、それ自体が不安定要因。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`EspionageRules`・SGZ-4・SLN-1・席次/軍功・指揮班#885 を作り直さない**。OTH は「人物標的の工作」と「内部崩壊の回路」を**足し、接続する**だけ（additive）。タイクン化回避＝工作は高位の決断（仕掛ける/防ぐ）、疑心の増殖は創発。**現実の人種表象は採らない**。

### ★★★ 最優先（真の欠落・OTH の signature）

#### OTH 失脚工作（讒言・偽証拠の植え付け）
- `DiscreditRules`（static・test-first）＝**人物を標的とする諜報任務の新類型**：偽証拠の植え付け（ハンカチ型）・讒言の吹き込み。成功で標的の**信用**（対上官・対同僚の関係値）を毀損。コストと発覚リスク（`EspionageRules` の枠に乗せる）。
- 接続：`EspionageRules`（任務類型の追加＝再実装しない）・SLN-1 関係グラフ（毀損の書き込み先）・SGZ-4（勢力間離間の**人物版**＝対をなす）。

#### OTH 個人間の離間→指揮系統の内部崩壊
- 人物ペア（上官↔腹心・提督↔副提督/参謀）の信頼が毀損されると**指揮班#885 の実効能力が劣化**（`viceAssistRatio`/`chiefAssistRatio` の信頼係数＝実効値パターン・基準非破壊）→最悪は罷免・査問・粛清の連鎖。**艦隊を撃たずに艦隊を弱体化させる**。
- 接続：`CommandStaffRules`#885・SLN-1・#817（疑心が旗幟へ波及＝寝返りやすくなる）。

#### OTH 疑心の自己強化（確証バイアス）
- 植えられた疑念は**自走する**：疑心値が閾値を超えると、中立な出来事も負に解釈され関係値が自動悪化（調べるほど黒く見える）。**解毒は困難**＝直接対話（DLG）・第三者の証言・工作の露見のみが止められる。
- 接続：SLN-1（減衰の逆＝負の自走）・DLG #1156（対話が解毒の数少ない手段＝ADVの見せ場）・MKV（情報品質）。

### ★★ 高（動機と標的＝工作の生態系）

#### OTH 昇進の恨み（怨恨の製造）
- 昇進人事で**飛ばされた者**（席次上位なのに下位が抜擢・軍功が報われない）に**怨恨**（SLN-1 宿怨）が発生→不満が閾値を超えた人物は**内部工作の潜在実行者**になる（買収されやすい・自発的讒言）。イリアス型（戦利品→ストライキ）の**人事版**。
- 接続：`SeniorityRules`/`MeritRankRules`#900・HCM#992（人事の帰結）・`FactionLoyaltyRules`（腐敗国ほど怨恨が動機化）。

#### OTH 外様将軍の脆弱性
- **実力登用の異邦人**（門地開放#169・テクノクラート・亡命者#194）は能力ボーナスの代わりに**帰属不安**係数＝失脚工作・疑心の**付け入られやすさ**が高い。登用のリターンとリスクのトレードオフ（外様を重用する国は強いが、内部工作に脆い）。
- 接続：#169・VEN-5（法的地位）・#194（亡命者）・`CaptivityRules.Recruit`（投靠者も外様）。**架空勢力の構造のみ＝現実集団の表象は不採用**。

#### OTH 工作の露見（身内からの暴露）
- 失脚工作は**実行者の周辺**（協力者・配偶者・部下＝関係グラフの隣接者）から確率露見する＝完全犯罪を難しくするカウンター。露見すると工作者の信用が逆に崩壊（ブーメラン）＋被害者の名誉回復。
- 接続：`EspionageRules.DetectionRisk` の拡張・SLN-1（隣接者の忠誠/関係が露見率を決める）。

### ★ 中（lore）

#### OTH（lore）緑の目の怪物・平時の武人
- 「嫉妬は緑の目の怪物」「戦争が消えた将軍の空虚」＝疑心と平時の武人の不安定を開示データへ。平時の軍人不満はクーデターリスク（#145 `CoupRisk`）の入力に薄く接続。
- 接続：`DisclosureLedger`（FND-4）・#145。コード新設は最小。

### ❌ 不採用（重複・倫理・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 諜報エンジンの再実装 | **`EspionageRules` がカバー**。OTH は人物標的の任務類型を足すだけ |
| 勢力間の離間 | **SGZ-4 #1106 がカバー**（OTH は人物ペア版＝対をなす） |
| 現実の人種・民族表象 | 倫理方針。「外様」の**構造のみ**架空勢力で一般化 |
| 恋愛悲劇の脚本化・精神状態シミュ | 別ジャンル。疑心は関係値の自走モデルで表現・物語は挿話TALE-2/DLGが担う |
| 査問会・軍法会議の本格実装 | 裁判は **VEN-2 `TrialRules` がカバー**（失脚工作の帰結として接続のみ） |
