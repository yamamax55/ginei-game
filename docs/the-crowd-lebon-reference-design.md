# 群衆心理 参考設計（EPIC #CRWD）

> 参照元：ギュスターヴ・ル・ボン『群衆心理』（*Psychologie des foules*, 1895）。**個人が群衆に溶けると別個の「集団精神」が立ち上がり、知性は下がり感情と被暗示性が跳ね上がる**——非合理が伝播し、指導者の威光が群れを束ね、些細なきっかけで歓喜から恐慌へ反転する、という大衆行動の構造を描く社会心理学の古典。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に巨大な社会・政治シミュ純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#CRWD` として issue 化する提案。
> **著作権注意**：本書および派生 issue は、固有名・原文・キャラクター・固有設定を一切流用しない。参考にするのは**大衆行動の構造パターン／メカニクスのみ**。歴史実在人物・実在運動の固有名も用いない。

## 0. なぜ「群衆心理」が本システムに役立つか

当プロジェクトは「個人の意思決定」と「マクロ社会の均衡」を**大量に保有**している（[CLAUDE.md] テスト基盤節 参照）：

| 既存（カバー範囲） | 担う層 |
|---|---|
| `FleetMorale`（士気・敗走 `IsRouted`・回復） | **個艦/部隊単位**の士気と敗走（per-fleet・空間伝播なし） |
| `LoyaltyRules`/`BattleAllegianceRules`（#817 関ヶ原型） | 寝返り**カスケード**（不動点反復）＝戦う前に決まる戦い |
| `NonviolenceRules`/`Movement`（#831） | 可視化された弾圧で沈黙の多数を支持へ転換・運動の勝利 |
| `HopeRules`/`Community`（#852） | 希望の枯渇→末人発火・信仰/秩序ルート |
| `GovernanceRules`/`Province`（#109） | 安定度・反乱圧力（住民マクロ） |
| `BEH-6` ハーディング（#1246・行動経済学） | 模倣の連鎖＝バブル/取付け/敗走/世論の雪崩（**一回的同調**） |
| `BEH-7` フレーミング/ナッジ（#1247） | 見せ方が決断を動かす＝プロパガンダ/名分の**増幅率** |
| `DynastyRules`/`Organization`（#867/#812） | 天命喪失・カリスマの継承と組織存続 |
| `EventEngine`/`DisclosureLedger`（#116/FND-4） | データ駆動イベント・lore 開示 |

**しかしこれらは「個人の決断バイアス」か「マクロ均衡の収束」**であり、ル・ボンが固有に描く以下が**欠けている**：

| 群衆心理が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **群衆＝別個の集団精神（level-shift）**。個人が集まると知性低下・感情/被暗示性が跳ね上がる「単一の心」が立つ | 既存は個人(`Person`)かマクロ(`FactionState`)の二層のみ。**「個の集合が群れに転じる」相転移**と、群衆化したときの能力変質モデルが無い |
| **断言/反復/感染の三段（指導者が観念を植える時間動学）** | `BEH-7` は単一選択の枠の増幅率（一発）。**「証明なき断言→反復で真実化→接触で伝播」という時間をかけた信念形成の三段プロセス**が無い |
| **指導者の威光（prestige）と、一度の失敗での非連続崩壊** | `Organization`(#812)はカリスマの継承、`DynastyRules`は天命。**個別指導者の「威光」が群衆を磁化し、一度の露呈/敗北で不可逆に瓦解する**動学が無い |
| **群衆の易変性＝歓喜⇔恐慌の急反転（同じ群れが英雄にも暴徒にもなる）** | `FleetMorale` の敗走は閾値で起こるが per-fleet・回復は時間。**「些細なきっかけで群れ全体が一斉反転する」可逆スイッチ**が無い |
| **恐慌/敗走の空間伝播（隣の崩れが伝染る）** | `FleetMorale.IsRouted` は各部隊が自分の被弾/交戦で独立に敗走。**隣接する友軍の敗走が伝染する空間カスケード**（戦線が将棋倒しに崩れる）が無い |

**結論**：群衆心理は、既存の「個人バイアス（行動経済学）」と「マクロ均衡（社会シミュ）」の**間に欠けていた中間層＝群衆という創発主体**を与える。具体的欠落軸は**①個↔群の相転移と能力変質 ②断言/反復/感染の三段で観念を植える時間動学 ③指導者の威光と非連続崩壊 ④恐慌の空間伝播（敗走カスケード）**の4本。会戦の士気（戦線将棋倒し）と戦略の世論（プロパガンダの効きが反復で蓄積）の両方に効く。
> **BEH-6/7 との切り分け**：行動経済学は「個人の認知バイアスの統一レイヤー」、ル・ボンは「群衆という別個主体の創発・指導者操作の時間動学」。重なる「模倣の雪崩」「枠の増幅」は**CRWD では新設せず BEH に委ねて接続のみ**（不採用表）。CRWD は BEH が触れない相転移・三段動学・威光崩壊・空間伝播に絞る。

## 1. 役に立つ視点（要約）

群衆心理の構造を、**本システムに効く形**で1行ずつ：

1. **個が群れに溶けると別の心が立つ**＝知性は下がり感情・被暗示性は跳ね上がる。→ `Person` でも `FactionState` でもない**創発主体「群衆」**を一時的に立てる新軸。`HopeRules`(#852)末人発火・`GovernanceRules` 反乱圧力の上流。
2. **群衆の能力は変質する**＝個別の理性的判断より、単純で極端な観念に動く。→ 群衆化した集団は**説得（理屈）でなく断言（暗示）に従う**。`NonviolenceRules`(#831)の支持転換に「群衆化係数」を挟む。
3. **指導者は断言・反復・感染で観念を植える**＝証明なき断言を反復し、接触で伝染させる。→ `BEH-7` の一発の枠でなく**時間をかけた信念の蓄積**。プロパガンダ・名分(ZUO-3)・対話(DLG)の効果を反復で累積させる動学。
4. **威光が群れを磁化し、失敗で一瞬に瓦解する**＝指導者の権威は実績でなく威光、しかも非連続に崩れる。→ `Organization`(#812)継承・`DynastyRules`(#867)天命とは別の**個別指導者の威光ゲージ**＝寝返り(#817)・支持(#113)の上流。
5. **群衆は易変＝歓喜と恐慌は紙一重**＝同じ群れが英雄称賛から暴徒/敗走へ一斉に反転する。→ `FleetMorale` 敗走に**群れ単位の一斉スイッチ**と、戦略の世論（熱狂バブル→幻滅）に可逆反転を足す。
6. **恐慌は伝染する**＝隣の崩れを見て自分も崩れる。→ `FleetMorale.IsRouted` を**空間カスケード化**＝戦線の将棋倒し。`LoyaltyRules.ResolveCascade`(#817)の不動点反復を士気の空間版へ流用。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`FleetMorale`/`LoyaltyRules`/`NonviolenceRules`/`HopeRules`/`BEH-6`/`BEH-7` を作り直さない**。CRWD はそれらに**欠落軸（相転移・三段動学・威光・空間伝播）を足し、接続する**だけ（additive・タイクン化回避＝高位の決断→エンジン→創発帰結）。

### ★★★ 最優先（真の欠落・群衆心理の signature）

#### CRWD 群衆化の相転移と集団精神（個↔群の level-shift）
- **群衆化モデル**：人口/集団が密度・興奮・共通刺激で閾値を越えると**群衆状態**へ相転移し、`crowdIntensity`(0..1) を持つ。群衆化するほど**被暗示性↑・理性↓・極端化↑**（個々の `Person` の判断より単純で極端な観念に従う）。
- 接続：新 `CrowdRules`/`Crowd`（純ロジック・test-first）。`GovernanceRules` 反乱圧力・`HopeRules`(#852)末人発火の**上流の創発主体**＝住民が群衆化して初めて暴動/熱狂が点く。マクロ(`FactionState`)と個(`Person`)の中間層。

#### CRWD 断言・反復・感染（指導者が観念を植える時間動学）
- **三段モデル**：①**断言**＝証明なき主張を投入（初期の被信度は低い）②**反復**＝同じ断言を時間をかけて繰り返すと被信度が累積（飽和あり）③**感染**＝群衆密度×`crowdIntensity` で被信した観念が周囲へ伝播。`BEH-7` の一発の増幅率と違い**時間軸で効果が蓄積する**のが核。
- 接続：`CrowdRules` に三段ステップを実装。`BEH-7`(フレーミング/ナッジ・#1247)＝**1tickの増幅率**を入力に取り、**反復で累積**させる時間動学を CRWD が足す（BEH-7 を作り直さない）。プロパガンダ・名分(ZUO-3)・対話(DLG)・選挙世論(#159)の効きを「反復回数」で蓄積。`EventEngine`(#116)の風説/演説イベントの効果源。

### ★★ 高（指導者の威光・恐慌の空間伝播）

#### CRWD 指導者の威光（prestige）と非連続崩壊
- **威光ゲージ**：指導者が群衆を磁化する力 `prestige`(0..1)。勝利/成功で漸増、しかし**一度の露呈・敗北・約束の不履行で不可逆に崩落**（連続でなく崖）。威光が高い間は断言の被信度・群衆の結束に強い乗数、崩落すると群衆が一斉に離反（指導者を見限る）。
- 接続：新軸として `CrowdRules.Prestige*`。`Organization`(#812・カリスマの継承)＝死後存続、`DynastyRules`(#867・天命)＝王朝サイクルとは**別レイヤー**＝個別指導者の即時の磁力。`LoyaltyRules`(#817)の寝返り susceptibility・支持(#113)・`BattleAllegianceManager` 会戦寝返りの上流入力（威光崩落→寝返り誘発）。**重複しない＝威光は読み取り入力として既存カスケードへ供給**。

#### CRWD 恐慌の空間カスケード（敗走の伝染・戦線将棋倒し）
- **空間伝播**：ある部隊が敗走(`FleetMorale.IsRouted`)すると、**隣接する友軍部隊の士気を引き下げ連鎖崩壊させる**（距離減衰・威光の高い指揮官が近いと抑制）。`LoyaltyRules.ResolveCascade`(#817)の不動点反復を**空間×士気版**へ流用（ロジック委譲・重複新設しない）。
- 接続：`FleetMorale`(既存・per-fleet)を読み、新 `RoutContagionRules`(純ロジック・test-first)が**隣接友軍への伝播量**を返す（`FleetMorale` 本体は非破壊・実効値パターン）。会戦の `FleetRegistry.AllFlagships`＋距離で近傍を取る。戦線が一気に崩れる創発＝銀英伝の「雪崩を打つ敗走」。

### ★ 中（群衆の易変・世界観lore）

#### CRWD 群衆の易変と一斉反転（歓喜⇔恐慌スイッチ）
- **可逆スイッチ**：群衆化した集団は些細なきっかけ（象徴的勝敗・指導者の一言・噂）で**熱狂↔恐慌を一斉反転**する。`crowdIntensity` が高いほど反転が起きやすく、振れ幅も大きい（易変性＝群衆の本性）。
- 接続：`CrowdRules` に反転判定。会戦では `FleetMorale` の群れ単位スイッチ（敗走/士気高揚の一斉切替）、戦略では世論（熱狂バブル→幻滅）として `GovernanceRules` 安定度・支持(#113)へ。`EventEngine`(#116)の象徴イベントがトリガ。

#### CRWD（lore）群衆と指導者の世界観開示データ
- 「群衆は理性でなく感情で動く」「威光は実績でなく演出」「英雄も暴徒も同じ群れ」——大衆と指導の関係を**コード新設せず** `DisclosureLedger`(FND-4)への**lore データ入力**として供給。秘史/啓蒙EPICに接続。
- 接続：`DisclosureLedger`(FND-4)。CCX-6（世界観codex退避）方針に一貫。

### ❌ 不採用（重複・既存で十分・接続のみ）

| 不採用 | 理由 |
|---|---|
| 模倣の連鎖／世論の雪崩そのもの | **`BEH-6`(#1246 ハーディング)が既にカバー**。CRWD は作り直さず、群衆化の相転移を**上流に接続**するだけ |
| フレーミング/枠の増幅率（一発の見せ方） | **`BEH-7`(#1247)がカバー**。CRWD は BEH-7 の増幅率を入力に取り**反復で累積**させる時間動学のみ足す |
| 寝返りカスケード（不動点反復の本体） | **`LoyaltyRules.ResolveCascade`(#817)がカバー**。CRWD は威光を入力供給＋空間士気版へロジック委譲（重複新設しない） |
| 弾圧の可視化→沈黙の多数の転換 | **`NonviolenceRules`(#831)がカバー**。CRWD は群衆化係数を挟む接続のみ |
| 希望枯渇→末人発火 | **`HopeRules`(#852)がカバー**。CRWD 群衆化は末人発火の上流刺激として接続 |
| カリスマの継承・組織存続／天命サイクル | **`Organization`(#812)/`DynastyRules`(#867)がカバー**。CRWD の威光は**個別指導者の即時磁力**＝別レイヤーで非破壊接続 |
| 個人の決断バイアス（プロスペクト/サンクコスト等） | **行動経済学#1240がカバー**。CRWD は個でなく群れの創発に限定 |
| 検閲/言論統制・秘密警察 | **`SecurityRules`(#166)・1984#1638がカバー** |

### 推奨着手順
`CRWD-1`（群衆化の相転移＝全ての土台・他が乗る創発主体）→ `CRWD-2`（断言/反復/感染＝群衆を動かす時間動学）→ `CRWD-3`（威光＝指導者の磁力。CRWD-1/2 の効果に乗る）→ `CRWD-4`（恐慌の空間カスケード＝会戦への即効・最も体感が強い）→ `CRWD-5`（易変・一斉反転＝CRWD-1〜4 の上に乗る挙動）→ `CRWD-6`（lore・最後・コード不要）。

> いずれも既存の士気/社会シミュを**後退させず接続**する additive 設計。CRWD-1〜3/5 は純ロジックを TestHarness/EditMode で先に固定し盤面配線は後段（純ロジックのまま後方互換）、CRWD-4 は会戦の `FleetMorale` へ実効値パターンで反映（基準非破壊）。新 state を盤面へ接続する子は完了条件に**観測層追従**（`CoreStateInspector` 用語集＋必要なら `Register`）を含める。
