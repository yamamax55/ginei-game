# 『1984年』参考設計（EPIC #NTEF）

> 参照元：ジョージ・オーウェル『1984年』（ディストピア政治小説）。
> 党による**全体監視・歴史の改竄（記録省）・二重思考・恒久戦争**で「真実」そのものを国家が製造する世界。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋巨大な社会・政治シミュ純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#NTEF` として issue 化する提案。
> **著作権注意**：固有名（党名/標語/登場人物/省庁名/地名/造語）・文章・キャラクター・固有設定は一切流用しない。**統治メカニクス／全体主義の構造パターンのみ**を参考にする。設計・実装でも固有名は使わず、汎用語（記録改竄・公式言説・監視・恒久敵）で表現する。

## 0. なぜ『1984年』が本システムに役立つか

当プロジェクトは**社会・政治シミュのマクロ純ロジックを大量に保有**している（[CLAUDE.md]「テスト基盤」節 参照）：

| 既存（カバー範囲） | 何をしているか |
|---|---|
| `SecurityRules`/`SecurityApparatus`（#166） | 反対派の弾圧 `DissentSuppression`・クーデター探知・**弾圧の支持ペナルティ** |
| `DisclosureRules`/`DisclosureLedger`/`DisclosureEntry`（FND-4 #495） | 秘史/真相の**開示**（条件→開示、連鎖開示の不動点）＝**隠された真実を明るみに出す** |
| `HopeRules`/`Community`（#852） | 希望の枯渇→末人（無気力な被支配者）の発火・専制の暴走 `IsTyranny` |
| `GovernanceRules`/`Province`（#109） | 安定度・占領統合・反乱圧力（思想一致が安定に効く） |
| `ResearchRules`/`ResearchProject`（#123-127） | 研究産出・政体による研究偏り `IdeologyBias` |
| `ConsentRules`/`Polity`（#836） | 「権力は借り物」＝協力・正統性・抑圧、非協力で統治不能 |
| `JusticeRules`（#918-923） | 住民の正義観構成→正統性・最大の不満（イベント火種） |
| `CultureRules`/`Culture`（#194） | 同化圧力・分離独立・ナショナリズム |
| `EventEngine`/`NotificationCenter`（#116/NOTIF-1） | データ駆動イベント・通知の単一窓口 |
| `CampaignState`/`FactionState`（統合層） | 正統性/合意/結束/希望の合成・崩壊判定 |

**しかし、これらはすべて「真実は1つで、それを弾圧／開示／合意で扱う」前提**であり、『1984年』が固有に描く**「国家が真実そのものを製造・改竄する」**回路が**欠けている**：

| 『1984年』が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **歴史の改竄＝記録省（過去の記録を遡及的に書き換え、不都合な事実を消す）** | `DisclosureRules` は**真実を開示**する向き。**公式の歴史を能動的に捏造・上書きし、過去の記録を破棄する逆向き**が無い。`CampaignState` に「公式言説（official narrative）」と「実際の履歴」の乖離を持たない |
| **二重思考＝住民が公式言説と現実の矛盾を同時に信じる（認知の分裂）** | `JusticeRules`/`GovernanceRules` は住民の思想を1つの値で持つ。**公式に信じる像と実体の乖離（認知ギャップ）**という第二の状態軸が無い |
| **全体監視＝テレスクリーンの遍在が自己検閲を生む（萎縮効果）** | `SecurityRules.DissentSuppression` は**事後弾圧**。**監視の遍在そのものが反対の表面化を抑える事前の萎縮（chilling effect）**と、それが支持でなく**沈黙**を生む回路が無い |
| **恒久戦争＝勝てない外敵を永続させ国内を統制する（戦争の目的が勝利でなく統制）** | `WarGoalRules`/`DiplomacyRules` は**勝つ/講和する**実戦争。**勝利を意図しない「国内統制のための演出された永続敵」**が無い |
| **言語の貧困化＝語彙を削り思考可能な範囲を狭める（思想統制の言語版）** | 思想統制は弾圧（事後）か同化（`CultureRules`）。**反対概念そのものを表現不能にして異論の発生率を下げる**前段が無い |
| **内部の党＝外部の党＝大衆の三層構造（統制の階層的非対称）** | 階級は `FiscalClass`/`Party` で経済・政治はあるが、**統制への露出度が層で違う（被監視・洗脳の濃度差）**という統制階層が無い |

**結論**：『1984年』は当プロジェクトの社会シミュに**「真実が1つでない世界＝国家が現実を製造する」という第二の認識軸**を与える。具体的な欠落軸は **①歴史改竄（記録の製造）②二重思考（認知ギャップ）③監視の萎縮効果 ④恒久敵による統制**の4本。これらは弾圧（`SecurityRules`）・開示（`DisclosureRules`）・希望（`HopeRules`）の**いずれとも直交**し、`CampaignState`/`FactionState` に薄く接続して**短期は極めて安定だが真実乖離が蓄積し崩壊が突発する**という創発を生む（収奪モデル `ExtractiveDecay` の認知版）。

## 1. 役に立つ視点（要約）

『1984年』の世界構造を、**本システムに効く形**で1行ずつ：

1. **真実は権力が製造する**＝過去の記録を遡及改竄し「公式の歴史」を上書きする。→ 既存 `DisclosureRules`（真実の開示）の**鏡像**として「公式言説」を `CampaignState` に新設し、実履歴との乖離を蓄積させる。
2. **二重思考**＝住民は公式言説と現実の矛盾を同時に受容する。乖離は無痛ではなく**認知ギャップ**として静かに溜まる。→ `GovernanceRules`/`JusticeRules` の「住民が信じる像」と「実体」を分離。
3. **監視は弾圧の前段**＝遍在する監視が反対を**表面化する前に萎縮**させる（事後弾圧の `SecurityRules` の上流）。支持を生むのではなく**沈黙**を生む。→ `SecurityRules` に萎縮効果のレイヤーを接続。
4. **戦争は統制の手段**＝勝てない外敵を永続させて結束と統制を維持する（勝利を意図しない演出戦争）。→ `WarGoalRules`/`DiplomacyRules` と区別される「恒久敵」レバー。`HopeRules`/`FactionState` の結束へ。
5. **言語の貧困化**＝表現可能な概念を削れば異論は発生しにくい。→ 二重思考の前段＝認知ギャップ蓄積率を下げる修正子（独立EPIC化せず①②に内包）。
6. **統制は階層で濃度が違う**＝中枢ほど真実を知り、末端ほど洗脳される非対称。→ `FiscalClass`/`Party` の階層に「被統制露出度」を足す（独立化せず②へ内包）。
7. **（lore）真実の終焉＝啓蒙の暗い鏡**。開示エンジン（秘史/啓蒙/ニーチェ）の対極に「真実が国家に独占される世界」を置く。→ `DisclosureLedger` への lore データ入力（コード新設なし）。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`SecurityRules`#166／`DisclosureRules`FND-4／`HopeRules`#852／`GovernanceRules`#109／`WarGoalRules`#192 を作り直さない**。NTEF はそれらに**欠落軸（真実の製造）を足し、接続する**だけ（additive）。すべて**高位の決断→エンジン駆動→創発帰結**で、マイクロ操作を増やさない（タイクン化回避）。

### ★★★ 最優先（真の欠落・『1984年』の signature）

#### NTEF 歴史改竄エンジン（公式言説の製造と記録の遡及上書き）
- **公式言説モデル**：`CampaignState`/`FactionState` に「公式に流布する歴史/状態像」を持たせ、実体（実際の戦績・経済・占領史）との**乖離（真実乖離度 0..1）**を保持。
- **改竄レバー**：高位の決断で「不都合な記録を上書き／消去」＝**短期は安定度・結束を回復**（敗北を勝利に書き換える）するが、**真実乖離が蓄積**し、開示（`DisclosureRules`）や敗北の累積で**突発崩壊**する。
- 接続：新 `NarrativeControlRules`/`OfficialNarrative`（純ロジック・test-first）。`DisclosureRules` の鏡像（開示↔隠蔽）として対をなし、`CampaignState` に接続＝**観測層追従**。`ExtractiveDecay`（#900 法家の罠）の認知版＝短期最強・長期崩壊。

#### NTEF 二重思考と認知ギャップ（住民が公式像と実体を同時に受容）
- **認知ギャップモデル**：住民の「公式に信じる像」と「実体」を分離し、改竄が続くほど両者が乖離。乖離は無痛でなく**潜在不満**として静かに蓄積（弾圧で表に出ないだけ）。
- **破綻**：乖離が閾値を超える／開示イベントで現実を突きつけられると、二重思考が崩れ**反乱圧力が一気に顕在化**（`GovernanceRules.RebelPressure` へ合流）。
- 接続：新 `DoublethinkRules`（純ロジック・test-first）。`GovernanceRules`/`JusticeRules`（住民認知）× NTEF 歴史改竄 ×`HopeRules`（末人＝考えない方が楽な均衡）。**言語貧困化・統制階層は本ルールの修正子として内包**（独立化しない＝取捨選択）。

### ★★ 高（監視と恒久敵）

#### NTEF 監視の萎縮効果（弾圧の前段＝反対が表面化する前に沈黙する）
- **萎縮レイヤー**：監視密度が高いほど反対が**表面化する前に抑制**される（事前）。`SecurityRules.DissentSuppression`（事後弾圧）の上流に置く。支持を生むのではなく**沈黙**を生み、不満は消えず潜在化（NTEF 認知ギャップへ流れ込む）。
- **副作用**：高監視は協力 `ConsentRules` を蝕み、長期で正統性を下げる（恐怖統治の代償）。
- 接続：`SecurityRules`/`SecurityApparatus`（#166）に**萎縮レイヤーを追加**＝重複新設せず既存ルールへメソッド/修正子を足す。`ConsentRules`（協力低下）×`DoublethinkRules`（潜在化）。

#### NTEF 恒久敵による統制（勝てない外敵を演出し結束を維持）
- **恒久敵レバー**：高位の決断で「永続的な外的脅威」を維持＝**結束 `FactionState`・支持を底上げ**するが、実際の勝利を意図しない（演出戦争）。歳出・厭戦が溜まり、敵が消える/虚構が露見すると反動。
- 接続：`WarGoalRules`（#192 厭戦/戦争目標）×`DiplomacyRules.IsHostile`×`HopeRules`（結束）。**実戦争（勝つ）と区別された統制目的の敵対**＝既存 `WarGoalRules` に「統制動機」フラグを足して接続（並行戦争システムを作らない）。

### ★ 中（世界観lore）

#### NTEF（lore）真実の終焉＝啓蒙の暗い鏡
- 「真実が国家に独占される世界」＝開示エンジン（秘史/啓蒙/ニーチェ「神は死んだ」）の対極。歴史改竄が極まると**開示が原理的に不能になる**ディストピア・エンディング系譜。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。CCX-6（世界観codex退避）方針に一貫。NTEF 歴史改竄度を `EventContext` で読み、開示の可否に反映。

### ❌ 不採用（重複・既存で十分・取捨選択）

> 「全部入り」は失敗。以下は既存でカバー済みか、当プロジェクトの粒度を下げる（タイクン化）ため**不採用**。

| 不採用 | 理由 |
|---|---|
| 反対派の弾圧・逮捕・処刑そのもの | **`SecurityRules`#166・`CaptivityRules`#154 が既にカバー**。NTEF は萎縮（前段）のみ接続 |
| 秘密警察組織・クーデター探知 | **`SecurityRules`/`SecurityApparatus`#166 がカバー**。作り直さない |
| 末人・無気力な被支配者 | **`HopeRules`/`Community`#852 がカバー**。二重思考は接続するだけ |
| 思想の同化・ナショナリズム | **`CultureRules`#194 がカバー** |
| プロパガンダの個別演出（ポスター/放送のミクロ操作） | マイクロ操作＝タイクン化。NTEF は**公式言説の乖離度というマクロ1軸**に集約 |
| 言語改造を独立システム化 | 単独では薄い。**`DoublethinkRules` の乖離蓄積率の修正子として内包** |
| 内部党/外部党/大衆の階層を新レジストリ化 | `FiscalClass`/`Party` の階層を流用＝**被統制露出度の修正子として内包**、別レジストリ新設しない |
| 拷問・洗脳の個人ドラマ | 人間スケールの演出は範囲外（純ロジックの社会シミュに集約） |

### 推奨着手順
`NTEF-1`（歴史改竄＝最も固有で欠落の大きい signature・`DisclosureRules` の鏡像）→ `NTEF-2`（二重思考＝改竄の帰結を住民認知へ落とし `CampaignState` へ接続・観測層追従）→ `NTEF-3`（監視の萎縮＝既存 `SecurityRules` 拡張で軽量）→ `NTEF-4`（恒久敵＝`WarGoalRules` 拡張で結束へ）→ `NTEF-5`（lore＝最後にデータ入力）。

> いずれも既存の弾圧（`SecurityRules`）・開示（`DisclosureRules`）・希望（`HopeRules`）を**後退させず接続**する additive 設計。核は**「真実乖離度」という単一のマクロ軸**＝短期は極めて安定だが乖離が蓄積し崩壊が突発する（収奪モデル `ExtractiveDecay` の認知版）。マイクロ操作を増やさず、高位の決断→エンジン駆動→創発帰結に徹する。
