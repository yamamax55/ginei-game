# リア王 参考設計（EPIC #LEAR）

> 参照元：シェイクスピア『リア王』（17世紀初頭）。老王が**生前退位**して国を娘たちに分割し、**忠誠の言葉テスト**で誠実な末娘を追放、退位後の**権威の漸進剥奪**（従者100人→50→25→0）の果てに荒野で狂う悲劇。副筋に**庶子の簒奪**（エドマンド）と**勝者連合の内紛**（ゴネリル/リーガンの共倒れ）。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋継承・権力・群像の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#LEAR` として issue 化する提案。
> 著作権注意：原作はパブリックドメインだが、固有名・文章・キャラクターは流用せず、**権力移譲・忠誠・継承の構造パターンのみ**を参考にする。

## 0. なぜ「リア王」が本システムに役立つか

当プロジェクトは継承・権力の純ロジックを保有している（[CLAUDE.md] 参照）：

| 既存（継承・権力・忠誠） | カバー範囲 |
|---|---|
| `SuccessionLawRules`（PDX-1 #646） | 継承法（長子/分割/指名/選挙）・継承危機リスク——**ただし君主の死を起点とする** |
| `SuccessionRules`/`Organization`#812 | 英雄死後の組織存続（制度化 vs 個人カリスマ） |
| `PowerRules`#164 | 実権/傀儡/影の支配者（`IsPuppet`/`IsShadowRuler`） |
| MKV #1138（君主論） | 直言参謀と佞臣（政治情報の品質） |
| SGZ-1 #1103（三国志） | 多極均衡（最強台頭→弱小連衡） |
| `LoyaltyRules`#817・`CaptivityRules`#154 | 旗幟・寝返り・捕虜/亡命 |

**しかし、これらが扱わないリア王固有の局面**がある：

| リア王が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **生前退位**＝死なずに権力を渡す（称号と実権の分離） | 継承#646/#152 は**死亡起点のみ**。退位者が生きて残る**二重権力（院政）**が無い |
| **権威の漸進剥奪**（従者100→50→25→0） | 退位者の残存権威が**段階的に侵食される過程**のモデルが無い |
| **忠誠の言葉テスト＝逆選択**（美辞の姉たちが取り立てられ、無骨な末娘が追放） | MKV は佞臣の進言品質。**「忠誠を試す行為自体が嘘つきを選抜する」**逆選択の罠が無い |
| **庶子の怨恨と簒奪**（エドマンド＝嫡庶差別が野心を生む） | #646 に**嫡出/庶出の正統性差**と、差別が生む簒奪動機が無い |
| **勝者連合の戦後分裂**（共通敵が消えた途端ゴネリル/リーガンが共倒れ） | SGZ-1 は台頭への連衡。**勝利後の連合内紛**（戦後秩序の不安定）が無い |
| **道化の特権**＝身分外の者だけが王に真実を言える | 諫言は MKV の参謀経路のみ。**制度外の真実チャネル（無礼講）**が無い |
| **追放忠臣の変装奉仕**（ケント＝偽名で仕え続ける） | 追放/亡命#194 は退場のみ。**忠誠が役職を超えて潜伏帰参する**回路が無い |

**結論**：リア王は当プロジェクトに**「権力の渡し方を間違えると、生きているうちに国が壊れる」**という移譲失敗のメカニクスを与える。死亡継承（#646/#152）の前段に**生前退位・院政・名実分離**を、佞臣（MKV）の前段に**忠誠テストの逆選択**を、同盟（SGZ-1）の後段に**勝者連合の分裂**を足す——既存3系統の**時間軸の空白**を埋める補完パッケージ。

## 1. 役に立つ視点（要約）

1. **権力は渡し方で壊れる**。死より難しいのが生前の移譲＝称号を残して実権を渡すと二重権力（院政）、実権を残すと後継が育たない。
2. **忠誠を試すと嘘が勝つ**。言葉の忠誠テストは美辞の達人（佞臣）を選抜し、無骨な誠実（コーディリア/ケント）を弾く＝**逆選択**。
3. **権威は儀礼から剥がれる**。従者数・席次・呼称——実権を失った者の権威は**儀礼の段階的剥奪**として可視化される。
4. **差別は野心を製造する**。嫡庶の正統性差が庶子の簒奪動機を生む＝継承制度の歪みが内乱の種。
5. **共通の敵が消えると連合は割れる**。勝者たちの内紛は戦争の続き＝戦後秩序こそ不安定。
6. **真実は身分の外から来る**。道化＝制度外ゆえに罰せられず真実を言える。諫言制度の迂回路。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**継承#646/#152・権力#164・MKV#1138・SGZ#1102 を作り直さない**。LEAR はそれらの**時間軸の空白（生前/試験/戦後）**を足し、接続するだけ（additive）。タイクン化回避＝退位・テスト・分裂はいずれも**高位の決断と創発的帰結**。

### ★★★ 最優先（真の欠落・LEAR の signature）

#### LEAR 生前退位と院政（名実の二重権力）
- `AbdicationRules`（static・test-first）＝**生前退位**：退位者は称号/残存権威（カリスマ#812・人脈）を保持し、実権は後継へ。残存権威は**儀礼の段階的剥奪**（従者数モデル＝予算/護衛/席次が新権力者の決断で削られる）で時間侵食。二重権力の間は**命令系統の混乱**（忠誠の分裂＝誰に従うか）。
- 接続：`SuccessionLawRules`#646（死亡継承の前段）・`PowerRules`#164（`IsShadowRuler`＝院政の実態）・`Organization`#812（個人カリスマの残存）・`LoyaltyRules`#817。

#### LEAR 忠誠テストの逆選択（試すほど嘘が勝つ）
- 君主が**忠誠の表明を求める**と、応答品質は**雄弁さに比例し誠実さに反比例**しうる＝美辞の佞臣が取り立てられ、無骨な忠臣が遠ざけられる確率モデル。テストを繰り返すほど宮廷の**情報品質が劣化**（MKV-3 直言/佞臣の前段＝佞臣が「製造」される過程）。
- 接続：MKV #1138（直言参謀と佞臣）・`CounselRules`（SGZ-2 #1104 献策の品質）・`AdmiralSkillRules`/voice（TALE-6 雄弁）。

### ★★ 高（継承・外交への波及）

#### LEAR 庶子の継承権と怨恨
- `SuccessionLawRules` に**嫡出/庶出の正統性差**を追加（庶子＝継承順位の劣後 or 排除）。排除された庶子は**怨恨→野心**（簒奪/出奔/敵勢力への投靠の動機係数）。門地開放#169（実力主義）とのトレードオフ＝庶子を登用する国は人材を得るが嫡流の正統性が揺れる。
- 接続：#646・#152・`CaptivityRules.Recruit`（投靠）・#169。

#### LEAR 勝者連合の戦後分裂
- 共通敵の消滅（戦争終結/共通脅威の崩壊）で**勝者連合の結束が減衰**→戦利品配分（イリアス型）・主導権・縁戚（エドマンド争奪型の人物要因）で**内紛→連合崩壊**の確率過程。SGZ-1（台頭への連衡）の**戦後フェーズ**。
- 接続：SGZ-1 #1103・`DiplomacyRules`（同盟の `DriftOpinion`）・`WarGoalRules`（講和後）・#817。

### ★ 中（群像・チャネル・lore）

#### LEAR 道化の特権（無礼講の真実チャネル）
- **身分外の者（道化/芸人/隠者）だけが罰されずに真実を言える**枠＝諫言の制度的迂回路。佞臣に囲まれた君主への**最後の正直な情報源**（逆選択 LEAR-2 の解毒剤・ただし聞くかは君主次第）。
- 接続：MKV-3・DLG（対話ADVで道化が真実を言うNPC）・SLN 社交場（宴席の道化）。

#### LEAR 追放忠臣の変装奉仕
- 追放された忠臣が**変名で潜伏し仕え続け**、危機に正体を明かして帰参する回路（忠誠は役職を超える）。追放＝退場ではなく**潜伏資産**になる。
- 接続：`Person`（#523 変名/呼称）・亡命#194・`CaptivityRules`・列伝#784（帰参の名場面）。

#### LEAR（lore）権力者の認識の皮肉
- 「嵐の荒野で初めて民の苦しみを知る」「視力を失って初めて見える」＝**権力の中枢ほど現実が見えない**皮肉の開示データ。失脚イベント→民情認識→（復権すれば）統治補正の小ループ。
- 接続：`DisclosureLedger`（FND-4）・`EventEngine`。コード新設は最小。

### ❌ 不採用（重複・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 死亡起点の継承・継承危機そのもの | **#646/#152 がカバー**。LEAR は生前退位だけ足す |
| 狂気の精神状態シミュレーション | 別機構を作らない。失脚・零落は既存（能力実効値/イベント）の係数で表現 |
| 家族ドラマの脚本化 | 構造（移譲/試験/分裂）のみ。人間関係の物語は挿話生成 TALE-2/DLG が担う |
| 実権/傀儡モデルの再実装 | **`PowerRules`#164 がカバー**。院政は `IsShadowRuler` の読み替えで接続 |
