# モンゴル帝国の興亡 参考設計（EPIC #MNGL）

> 参照元：杉山正明『モンゴル帝国の興亡』ほか同著者のユーラシア遊牧国家論。
> 13〜14世紀に**ユーラシアを一つに繋いだ遊牧国家**——機動する軍政組織、ジャムチ（駅伝）が支えた史上最速の通信網、出自を問わぬ実力登用、そして**分封（ウルス分与）と相続が生んだ「緩やかな連合→分裂」**を描く歴史叙述。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略）にとって**役に立つ視点**だけを抽出し、EPIC `#MNGL` として issue 化する提案。
> 著作権注意：固有名（人名・部族名・地名・国号）・文章・固有設定は流用せず、**統治/通信/相続の構造パターンとメカニクスのみ**を参考にする。

## 0. なぜ「モンゴル帝国の興亡」が本システムに役立つか

当プロジェクトは**定住型国家の統治・継承・人事の純ロジックを大量に保有**している（[CLAUDE.md] 参照）：

| 既存（カバー範囲） | 担当モジュール |
|---|---|
| 継承法（長子/分割/指名/選挙）の相続シェア・継承危機 | `SuccessionLawRules`（#646 PDX-1） |
| 封建制＝徴募・臣従反乱・門地開放 | `FeudalRules`/`Fief`（#168/#169） |
| 内政＝安定度収束・統合度・占領不満（per-惑星） | `GovernanceRules`/`Province`（#109） |
| 版図一体化度（所有星系の連結成分→国力割引） | `LogisticsRules`（#844 GEO-3） |
| 補給線（補給源から所有回廊で到達・敵ZOC遮断） | `SupplyRules`（#94 L-2） |
| 諜報（潜入成功率・情報獲得・発覚リスク） | `EspionageRules`/`SpyNetwork` |
| 人物・適材適所（出自を問わぬ役職配属・正名） | `PersonRules`/`Person`・`CareerPipelineRules`（#866/#155-157） |
| 文化・民族（同化・分離独立・ナショナリズム） | `CultureRules`/`Culture`（#194） |
| 経路探索（Dijkstra・FTL不可回廊回避） | `GalaxyPathfinder` |
| 統一時間・暦境界Tick・通知 | `GameClock`/`CalendarDispatcher`/`NotificationCenter` |

**しかし、これらは「定住・領域国家」を暗黙の前提**にしており、モンゴル帝国が固有に描く以下が**欠けている**：

| モンゴル帝国が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **ジャムチ＝駅伝網による情報伝達の物理（速度・遅延・遮断）** | 命令/情報は**瞬時に全盤面へ届く前提**。中継網の整備度で**指令が届くまでの遅延・距離減衰・遮断で統治力が落ちる**回路が無い |
| **分封（ウルス分与）＝版図を半自律の継承者領へ刻む** | `SuccessionLawRules.HeirShare` は**抽象的なシェア比率**のみ。**実際に星系群を継承者へ分与し、半自律のサブ勢力（appanage）として盤面に立てる**手続きが無い |
| **緩やかな連合→世代を経て分裂（各ウルスが独立khanate化）** | 継承危機の**リスク値**はあるが、**分与された領が時間とともに宗主から離反し独立勢力へ割れる**創発が無い |
| **遊牧国家＝移動する中核・季節移動・機動動員** | 内政は**固定領域**前提（`Province.systemId` 固着）。**中核（首都/オルド）が移動でき、機動性で動員/打撃が変わる**遊牧ロジックが無い |
| **クリルタイ＝合議による正統性付与（推戴）** | 継承は法/指名で決まる。**有力者の合議で君主を推戴し、欠席や不参加が正統性を割る**回路が無い |
| **パクス・モンゴリカ＝交易路の安全保障が国庫を太らす** | 交易/補給はあるが、**広域の通行安全（治安）が交易収入と版図一体化を生む**正のループが薄い |
| **千戸制＝出自を解体し機能本位で再編成する軍制** | 編制ツリー（`OrderOfBattle`）はあるが**部族/血統を崩して十進で再編し忠誠を中央へ吸い上げる**動学が無い |

**結論**：モンゴル帝国は当プロジェクトの定住国家ロジックに、**①情報伝達の物理（ジャムチ）②分封→分裂の相続動学（ウルス）③移動する遊牧国家④クリルタイ合議正統性**という4つの欠落軸を与える。とりわけ **「広大さゆえに分裂する」帝国** の創発は、本作の星間スケールの統治に直結する。

## 1. 役に立つ視点（要約）

モンゴル帝国の構造を、**本システムに効く形**で1行ずつ：

1. **情報こそ帝国の神経で、駅伝網がそれを運ぶ**。命令が届く速さ＝統治の速さ。中継網が薄い辺境は反応が鈍り離反しやすい。→ 瞬時前提の指令系に**遅延・距離・遮断**を入れ、`EspionageRules`/`NotificationCenter`/`GalaxyPathfinder` に物理を与える。
2. **広大な帝国は征服でなく相続で割れる**。継承のたびに版図を分与し、分与領は世代を経て独立する。→ `SuccessionLawRules` の抽象シェアを**実際の盤面分割（appanage 勢力生成）**へ昇格させ、`CampaignState`/`StarSystem.owner` を動かす。
3. **緩やかな連合は遠心力に勝てない**。距離・通信・忠誠が薄れた分与領は宗主から離れる。→ 版図一体化 `LogisticsRules`＋通信遅延＋忠誠の**分裂圧**＝`FactionState`/`FactionLoyaltyRules` の遠心バージョン。
4. **遊牧国家は固定領土でなく機動で戦う**。中核は移動し、機動性が動員・打撃に乗る。→ 定住前提の `Province`/`GovernanceRules` に**移動する中核**のオプション層を足す（既存を後退させない）。
5. **正統性は合議が与える**。クリルタイで推戴された君主は強く、不参加は分裂の前触れ。→ `SuccessionLawRules`/`LoyaltyRules` に**合議推戴**の正統性回路。
6. **平和は交易路の安全で買う（パクス・モンゴリカ）**。治安が交易収入と一体化を生み、それが更なる治安を支える。→ `SupplyRules`/`LogisticsRules`/`FiscalRules` を結ぶ**通行安全の正ループ**。
7. **出自を解体し機能で再編する**。血統でなく十進の機能単位へ組み替え忠誠を中央へ向ける。→ `OrderOfBattle`/`CareerPipelineRules` に**脱・血統の再編**動学（既存の適材適所と共鳴）。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`SuccessionLawRules`#646・`LogisticsRules`#844・`SupplyRules`#94・`EspionageRules`・`GovernanceRules`#109 を作り直さない**。MNGL はそれらに**欠落軸を足し、接続する**だけ（additive・タイクン化回避＝高位の決断→エンジン→創発帰結）。

### ★★★ 最優先（真の欠落・モンゴル帝国の signature）

#### MNGL ジャムチ＝駅伝（通信網）と情報伝達の物理
- **中継網モデル**：星系に**駅站（relay station）**の整備度を持たせ、命令/情報が**発信元から所有回廊を伝播する遅延（hop×距離）**を算出。網が薄い・遮断された辺境は**指令到達が遅れ統治力が落ちる**。
- 接続：`GalaxyPathfinder`（伝播経路）×`SupplyRules`（遮断条件を流用）×`NotificationCenter`（遅延付き通知）。新 `RelayNetworkRules`/`RelayState`（純ロジック・test-first）。瞬時前提を**情報の物理**で置換する核。

#### MNGL 分封（ウルス分与）＝版図を半自律継承者領へ刻む
- **分与手続き**：継承時に版図（星系群）を継承者へ**実際に分割**し、各ぶんを**半自律のサブ勢力（appanage）**として盤面へ生成。宗主への忠誠と自律度を持つ。
- 接続：`SuccessionLawRules.HeirShare`（シェア比率を入力に）×`CampaignState`/`StarSystem.owner`（盤面分割）×`FactionLoyaltyRules`（宗主忠誠）。新 `AppanageRules`/`Appanage`（純ロジック・test-first）。抽象シェアを**盤面操作**へ昇格。

#### MNGL 遠心力＝緩やかな連合の分裂（独立 khanate 化）
- **分裂圧**：分与領の**距離・通信遅延（MNGL ジャムチ）・忠誠低下・世代交代**が重なると宗主から離反し**独立勢力へ割れる**。版図が広いほど割れやすい（帝国の宿命）。
- 接続：`AppanageRules`（分与領）×`LogisticsRules`（一体化度）×`RelayNetworkRules`（通信遅延）×`FactionState`（世代/忠誠）。新 `FragmentationRules`（純ロジック・test-first）。征服でなく相続で版図が割れる創発。

### ★★ 高（遊牧と合議＝定住前提への対抗軸）

#### MNGL 遊牧国家＝移動する中核・機動動員（定住前提のオプション層）
- **移動する中核**：勢力が**固定首都を持たず中核（オルド）を移動**でき、機動性が**動員速度・打撃**へ乗る／定住化すると機動を失い安定を得る（遊牧↔定住のトレードオフ）。
- 接続：`GovernanceRules`/`Province`（定住側は不変）に**遊牧モードのフラグ＋係数**を足す `NomadicRules`/`NomadicState`（純ロジック・test-first）。既存の固定領域を後退させない additive 層。

#### MNGL クリルタイ＝合議推戴による正統性
- **合議推戴**：継承/即位を**有力者の合議**で承認＝出席者の同意で正統性が上がり、有力者の不参加/離反は**分裂の前触れ**として正統性を割る。
- 接続：`SuccessionLawRules`（指名/選挙の隣）×`LoyaltyRules`/`LeadershipElectionRules`（票/推戴の解決）。新 `KurultaiRules`（純ロジック・test-first）。合議の正統性を分裂圧（MNGL 遠心力）の抑止弁に。

### ★ 中（交易安全・脱血統の再編）

#### MNGL パクス・モンゴリカ＝交易路の安全保障の正ループ
- **通行安全**：広域の治安（敵ZOC/未統合の少なさ）が**交易収入と版図一体化を増やし**、それが治安を支える正のフィードバック。逆に分裂は交易路を断ち国庫を細らせる。
- 接続：`SupplyRules`/`LogisticsRules`（経路安全）×`FiscalRules`（交易収入）×`GovernanceRules`（治安）。**コード新設は最小**＝既存ルールを結ぶ係数ルール `TradeSecurityRules`（純ロジック・test-first）。

#### MNGL（lore）世界観の開示データ
- 「広さゆえに分裂する帝国」「情報網が国家の神経」「出自でなく機能で人を測る」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。世界観codex方針に一貫。

### ❌ 不採用（重複・既存で十分）

| 不採用 | 理由 |
|---|---|
| 継承法そのもの（長子/分割/指名/選挙のシェア・危機リスク） | **`SuccessionLawRules`#646 がカバー**。MNGL は分与の**盤面操作**だけ足す（接続のみ） |
| 千戸制＝編制ツリーの十進再編成（新規実装） | **`OrderOfBattle`#147・`CommandStaffRules`#885 がカバー**。脱血統は `CareerPipelineRules` の適材適所で表現＝新EPIC化しない |
| 出自を問わぬ実力登用 | **`PersonRules`/`CareerPipelineRules`#866/#155-157 が既にカバー**。接続のみ |
| 補給線/通商破壊の物理を新規実装 | **`SupplyRules`#94・`CommerceRaidingRules`#95 がカバー**。MNGL は通行安全の係数を薄く足すだけ |
| 文化同化・分離独立そのもの | **`CultureRules`#194 がカバー**。MNGL の遠心力は**通信・分与・忠誠**軸で別物として足す |
| 騎兵戦術/個別兵科の戦闘実装 | 会戦は既存の艦種#80・陣形#496 で表現。歴史固有兵科は流用しない |

### 推奨着手順
`MNGL-1`（ジャムチ＝情報伝達の物理。他の分裂/合議の前提になる神経系）→ `MNGL-2`（分封＝盤面分割の手続き）→ `MNGL-3`（遠心力＝MNGL-1/2 を合成した分裂の創発＝signature）→ `MNGL-5`（クリルタイ＝分裂の抑止弁）→ `MNGL-4`（遊牧の移動中核）→ `MNGL-6`（交易安全の正ループ）→ `MNGL-7`（lore）。

> 三本柱＝**①ジャムチ（情報伝達の物理 MNGL-1）②分封→遠心分裂（相続動学 MNGL-2/3）③クリルタイ合議正統性（MNGL-5）**。いずれも既存の継承/物流/諜報/内政を**後退させず接続**する additive 設計。星間スケールゆえの「広さが帝国を割る」創発を狙う。
