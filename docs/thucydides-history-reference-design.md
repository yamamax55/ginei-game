# トゥキュディデス『戦史』参考設計（EPIC #THUC）

> 参照元：トゥキュディデス『戦史（ペロポネソス戦争史）』。覇権国スパルタと新興アテナイの全面戦争を、神話でなく**人間の本性と力の論理**から記述した史書。
> 「戦争の真因はアテナイの台頭と、それがスパルタに引き起こした**恐怖**だった」——構造的力学が個々の出来事に先立つ、という史観。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に巨大な外交・社会シミュ純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#THUC` として issue 化する提案。
> 著作権注意：固有名（アテナイ/スパルタ/メロス/クレオン等）・文章・人物・固有設定は流用しない。**力学／交渉構造／政治力学の構造パターンのみ**参考にし、架空の勢力・状況で実装する。

## 0. なぜ「トゥキュディデス『戦史』」が本システムに役立つか

当プロジェクトは外交・戦争・政体の**マクロ純ロジックを大量に保有**している（[CLAUDE.md] 参照）：

| 既存（カバー範囲） | 何を扱うか |
|---|---|
| `DiplomacyRules`/`DiplomacyState`（#189 DIP-1） | opinion 修正子・外交状態遷移（平時/同盟/不可侵/属国/交戦）→`FactionRelations.IsHostile` |
| `TreatyRules`/`Treaty`（DIP-2 #191） | 条約効果・履行/違約・レバレッジ |
| `WarGoalRules`（DIP-3 #192） | casus belli・厭戦・講和受諾 |
| `AllianceChainRules`（SARA-2） | **条約義務による**連鎖参戦（局地戦→世界大戦化） |
| `FlashpointRules`/`UltimatumRules`（SARA-1/3） | 偶発事件の点火・最後通牒 |
| `DesperationWarRules`（PRL-2） | 封鎖で追い詰められた**弱者**の博打的開戦 |
| `LoyaltyRules`/`ConsentRules`/`PowerRules`（#817/#836/#164） | 関ヶ原型旗幟・合意統治・実権/傀儡 |
| `PartyRules`/`LeadershipElectionRules`（GOV-6/7） | 政党・支持・党首選出 |
| `Community`/`HopeRules`（#852） | 希望/末人（衆愚の素地の一部） |

**しかし、これらは「opinion・条約・casus belli・政体構造」のマクロ均衡**であり、トゥキュディデスが固有に描く以下が**欠けている**：

| 『戦史』が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **覇権交代の恐怖（トゥキュディデスの罠）** | 既存は opinion 低下/条約違約/偶発事件が開戦を駆動。**「相手の台頭速度そのものが恐怖を生み、追い越される前に叩く＝予防戦争」**という構造的力学が無い。SARA は点火、PRL は弱者の窮鼠＝**いずれも「強者が相対的衰退を恐れて先に動く」を扱わない** |
| **力の非対称下の交渉（メロス対話＝強者の論理）** | `UltimatumRules` は呑めない要求の提示。だが**「強者は為しうることを為し、弱者は受けるべきを受ける」式の、力差に応じた中立要求・服従強要・拒否すれば殲滅**という非対称交渉の解像度が無い |
| **嘆願者による引きずり込み（中立国の介入要請）** | `AllianceChainRules` は**既存の同盟義務**の連鎖。だが**「同盟関係に無い第三国が後ろ盾を求めて大国を係争に引き込む」**（介入の誘惑＝放置すれば敵陣営に渡る/介入すれば泥沼）という"非義務の引きずり込み"が無い |
| **衆愚政・デマゴーグ・群衆の激情と翻意** | `PartyRules`/支持#113 は選挙・議席。だが**民会が扇動者に煽られ過激決議→後悔して翻意（ミュティレネ論争）、虚栄の遠征に酔う（シケリア遠征）**という"群衆心理が国策を動かし揺り戻す"回路が無い |
| **疫病・災厄下の社会規範崩壊** | `Community`/希望は扱うが**極限ストレス下のアノミー（規律崩壊・短期享楽化）**は薄い |
| **将軍の追放・スケープゴート化** | 失策/敗戦時に司令を罰する政治圧（衆愚と将官人事の交点）。`RetirementRules`/捕虜#154 とは別軸 |

**結論**：トゥキュディデスは当プロジェクトの外交マクロに、**①覇権交代の恐怖（予防戦争の力学） ②力の非対称交渉（強者の論理） ③嘆願者による引きずり込み（非義務の介入） ④衆愚政（群衆の激情と翻意）** という4つの欠落軸を与える。いずれも**既存 `DiplomacyRules`/`WarGoalRules`/`AllianceChainRules`/支持#113 を後退させず、その上に「力学が出来事に先立つ」層を足す** additive 設計。

## 1. 役に立つ視点（要約）

『戦史』の史観を、**本システムに効く形**で1行ずつ：

1. **戦争の真因は出来事でなく構造（台頭への恐怖）**。aの急成長が b に「今叩かねば追い越される」恐怖を生む。→ SARA（点火）/PRL（窮鼠）が扱わない**強者の予防戦争**を `DiplomacyRules` opinion とは別系統で駆動。
2. **力が非対称なら交渉は対等でない（強者の論理）**。中立は許されず、服従か殲滅かを迫られる。→ `UltimatumRules` の非対称版＝力差が交渉の選択肢そのものを決める。
3. **同盟は安全保障であると同時に罠**。後ろ盾を求める弱小国が大国を係争へ引き込む。放置＝敵に渡る、介入＝泥沼。→ `AllianceChainRules`（義務連鎖）と対をなす**非義務の介入ジレンマ**。
4. **民主政は強さでも弱さでもある**。自由民は粘るが、群衆は扇動に流れ過激決議→翻意し、虚栄の大遠征で自滅する。→ 支持#113・`PartyRules` に**群衆心理の動学**（デマゴーグ係数・激情と揺り戻し）を足す。HRDT-5（自由vs専制の士気）と相補。
5. **力の論理は道徳に勝つが、驕りは罰を招く**。強者の非道（殲滅）は短期に通るが正統性・評判を蝕む。→ HRDT-6（ヒュブリス）/`DisclosureLedger` lore と接続。事後判明（no-outcome-preview）。
6. **歴史は構造の反復**。人間の本性が変わらぬ限り、覇権交代の危機は形を変えて再来する。→ 開示エンジン lore（秘史＝力学の自己認識）。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`DiplomacyRules`#189／`WarGoalRules`DIP-3／`AllianceChainRules`SARA-2／支持#113 を作り直さない**。THUC はそれらに**欠落軸を足し、接続する**だけ（additive）。タイクン化回避＝高位の決断（介入する/服従させる/扇動に乗る）→エンジン→創発帰結（予防戦争/泥沼/翻意の後悔）。

### ★★★ 最優先（真の欠落・トゥキュディデスの signature）

#### THUC 覇権交代の恐怖と予防戦争（トゥキュディデスの罠）
- **力学**：勢力ペアの**国力差の縮小速度**（台頭する側の成長率）を観測し、覇権国に「追い越される前に叩く」**恐怖係数**を蓄積。閾値で予防戦争の意思が立つ（AIの開戦動機／プレイヤーへの警告）。
- 接続：新 `HegemonicTransitionRules`（純ロジック・test-first）。入力＝`CampaignState.EffectiveStability`/国力・`LogisticsRules`版図。出力＝`DiplomacyRules` opinion とは別系統の**preventiveWarPressure**を `WarGoalRules`（casus belli＝予防）と `EventEngine`（危機イベント）へ供給。SARA-1（点火）の"なぜ火がつきやすかったか"の下地を与える。

#### THUC 力の非対称交渉（強者の論理＝中立は許されぬ）
- **力学**：交渉を**力差**でパラメタ化＝対等opinionの取引でなく、強者は「服従（属国化）か殲滅か」を弱者へ迫る。弱者の選択（服従/抵抗/第三者へ嘆願）が分岐。**抵抗して敗れれば最大被害、服従すれば正統性は保つが屈辱で離反種**。
- 接続：新 `CoercionRules`（純ロジック・test-first）。`UltimatumRules`(SARA-3) の**非対称版**＝力比が要求の強度と相手の選択肢を決める。`DiplomacyRules`（属国状態）・`WarGoalRules`（服従要求）・`LoyaltyRules`（屈服した属国の離反種）へ接続。

### ★★ 高（同盟の罠と衆愚）

#### THUC 嘆願者による引きずり込み（非義務の介入ジレンマ）
- **力学**：同盟関係に**無い**弱小・中立国が後ろ盾を求めて大国へ接近。大国は**介入ジレンマ**＝放置すれば相手が敵陣営の戦力になる／介入すれば自分の casus belli でない戦争へ巻き込まれる。介入は新たな前線と厭戦を生む。
- 接続：新 `EntanglementRules`（純ロジック・test-first）。`AllianceChainRules`(SARA-2＝義務連鎖) の**対**＝義務でない引きずり込み。`DiplomacyRules`（属国/同盟へ昇格）・`WarGoalRules`（厭戦増）・`EventEngine`（嘆願イベント）へ接続。

#### THUC 衆愚政・デマゴーグ・群衆の翻意（民会の激情と揺り戻し）
- **力学**：民主政体の**群衆気分**を状態化＝デマゴーグの扇動で過激決議（殲滅/大遠征）へ振れ、危機が去ると後悔して翻意（決議撤回）。**虚栄の大遠征**は群衆の高揚で承認されるが過拡張リスク（HRDT-6ヒュブリスへ）。専制政体は群衆気分を持たないが扇動耐性も低い別系統。
- 接続：新 `DemagogueRules`（純ロジック・test-first）。支持#113・`PartyRules`/`LeadershipElectionRules`(GOV-6/7)・`HopeRules`#852（希望/末人）に**群衆心理の動学**を足す。`EventEngine`（扇動/翻意イベント）・`CivilianControlRules`（敗戦時の将軍追放圧）へ。HRDT-5（自由vs専制の士気）と相補。

### ★ 中（社会崩壊・将官人事・lore）

#### THUC 災厄下のアノミー（疫病＝極限ストレスの規範崩壊）
- **力学**：疫病/飢饉/連敗など**極限ストレス**下で社会規範が崩れ、短期享楽化・規律低下・統治不能化（アノミー）。希望の枯渇とは別の"秩序の溶解"。
- 接続：**コード新設を避け** `Community`/`HopeRules`#852・`GovernanceRules`#109（安定度急落）・`EventEngine`#116（災厄イベント）への**接続のみ**＝既存に「ストレス→規範崩壊」修正子を足す薄い拡張。

#### THUC（lore）世界観の開示データ
- 「戦争の真因は台頭への恐怖（構造が出来事に先立つ）」「強者の論理は短期に通り長期に罰される」「群衆は煽られ、酔い、後悔する」「歴史は人間の本性ゆえ反復する」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。HRDT-8（歴史は勝者が書く）・PRL-6（だまし討ちの代償）と連鎖。

### ❌ 不採用（重複・既存で十分）

| 不採用 | 理由 |
|---|---|
| 条約義務による連鎖参戦 | **`AllianceChainRules`（SARA-2）が既にカバー**。THUC は"非義務の引きずり込み"のみ足す（`EntanglementRules`） |
| 偶発事件の点火・最後通牒 | **`FlashpointRules`/`UltimatumRules`（SARA-1/3）がカバー**。THUC は予防戦争の力学（恐怖の蓄積）と非対称交渉のみ |
| 弱者の窮鼠的開戦 | **`DesperationWarRules`（PRL-2）がカバー**。THUC は**強者**の予防戦争＝逆方向 |
| 僭主サイクル/ヒュブリス/歴史叙述 | **`TyrannyRules`/`HubrisRules`/`HistoriographyRules`（HRDT-2/4/6）がカバー**。THUC は接続のみ（予防戦争→ヒュブリス、衆愚→翻意） |
| 自由vs専制の戦闘士気係数 | **HRDT-5 がカバー**。THUC の衆愚（`DemagogueRules`）は政治決議の動学＝戦闘係数とは別層で相補 |
| 選挙・政党・党首選出そのもの | **`PartyRules`/`LeadershipElectionRules`（GOV-6/7）がカバー**。THUC は群衆気分の動学のみ足す |
| 厭戦・casus belli・講和受諾そのもの | **`WarGoalRules`（DIP-3）がカバー**。THUC は予防戦争/服従要求を casus belli として供給するだけ |

### 推奨着手順
`THUC-1`（覇権交代の恐怖＝最も固有で欠落の大きい signature・予防戦争の力学）→ `THUC-2`（力の非対称交渉＝強者の論理）→ `THUC-3`（引きずり込み＝同盟の罠）→ `THUC-4 → THUC-5`（衆愚政→将軍追放＝群衆心理の連鎖）→ 残り（THUC-6 接続のみ／THUC-7 lore）。

> いずれも既存の外交・戦争・支持マクロを**後退させず接続**する additive 設計。三本柱＝**①予防戦争の力学（THUC-1） ②力の非対称交渉（THUC-2） ③衆愚政の動学（THUC-4）**。SARA（同盟連鎖）/PRL（奇襲・窮鼠）/HRDT（叙述・ヒュブリス）と棲み分け、その隙間（強者の恐怖・強者の論理・非義務の引きずり込み・群衆の翻意）を埋める。
