# ホッブズ『リヴァイアサン』参考設計（EPIC #LEVI）

> 参照元：トマス・ホッブズ『リヴァイアサン』（1651年）。近代政治哲学の礎石。「自然状態＝万人の万人に対する闘争」「社会契約による主権の委託」「コモンウェルス＝死すべき神」の三本柱。
> 戦争でなく**なぜ秩序が存在するのか（自然状態の恐怖から逃れるための理性的選択）**を問う——「どんな暴君も無政府よりはマシ」という反直観的洞察が核。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に広大な政治・社会純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#LEVI` として issue 化する提案。
> 著作権注意：固有名・文章・キャラクター・固有設定は流用せず、**政治哲学メカニクス／世界観の構造パターンのみ**を参考にする。

## 0. なぜ「リヴァイアサン」が本システムに役立つか

当プロジェクトは政治・統治の**純ロジック層を幅広く保有**している：

| 既存（政治・統治・外交） | カバー範囲 |
|---|---|
| `ConsentRules`/`Polity`（#836/#837） | 協力の撤収・統治不能（ガンジー：抑圧→不同意） |
| `DynastyRules`/`Regime`（#867） | 正統性・腐敗・天命喪失（孔子：徳と天命） |
| `ConstitutionRules`/`Constitution`（#170） | 権力の憲法的制約・権利保護→正統性 |
| `CoupRules`（#215-219） | クーデター発生確率・帰結・事後正統性 |
| `CivilianControlRules`（#145） | 文民統制・軍部クーデターリスク |
| `LoyaltyRules`/`Allegiance`（#817） | 忠誠カスケード・条件付き寝返り |
| `DiplomacyRules`/`DiplomacyState`（#189） | 外交状態遷移・宣戦・講和 |
| `WarGoalRules`（#192） | 厭戦・戦争目標・講和受諾 |
| `HegemonyRules`（THU-1/#229） | 覇権移行恐怖・台頭勢力の検出→先制戦 |
| `FactionState`/`FactionStateRules` | 国家状態合成（天命喪失/統治不能/組織崩壊/末人） |
| `LogisticsRules`（#844） | 版図の地理的一体化度（分断→国力低下） |

**しかし、これらは「国家が存在することを前提とした内部動態」か「国家間の相互作用」**であり、ホッブズが固有に描く以下が**欠けている**：

| ホッブズが固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **自然状態のコストモデル（無政府＝万人の闘争）** | `FactionState.IsCollapsing` は崩壊「条件」を判定するが、崩壊**後の状態**（無政府宙域）とその動態コストが無い。崩壊後の「どんな統治でも無いよりマシ」という理性的受諾が計算されない |
| **安全保障ジレンマ（対称的軍拡スパイラル）** | `HegemonyRules`(THU-1) は「新興勢力の台頭→既存覇権の恐怖」という**非対称**な恐怖。ホッブズの安全保障ジレンマは**対称的**: 誰も攻める意図がなくても、全員が防衛のため武装→誰も安全でない螺旋が無い |
| **コヴェナント＝保護契約（保護失敗で契約解除）** | `ConsentRules` は「抑圧→不同意」（ガンジー型：悪い統治への抵抗）。ホッブズの契約は**「保護の提供→服従の義務」**：どれほど苛酷な統治でも安全を保証する限り服従が合理的だが、外敵から**守れなかった瞬間**に契約は消滅する |
| **主権の不可分性（二重請求→戦争と同値）** | `DiplomacyRules` は勢力間の外交状態。**同一空間の二重主権請求者の存在**が自然状態（戦争準備状態）と同値であるという圧力モデルが無い |

**結論**：ホッブズは当プロジェクトの政治純ロジックに**「秩序はなぜ存在するのか（自然状態の論理）」**という哲学的基底と、①**無政府宙域の動態コスト**　②**対称的軍拡スパイラル**　③**保護契約の閾値**　④**主権不可分圧力**という4つの欠落軸を与える。いずれも既存の「国家内部動態」や「非対称覇権競争」（THU）とは**直交する層**。

## 1. 役に立つ視点（要約）

リヴァイアサンの世界観を、**本システムに効く形**で1行ずつ：

1. **「自然状態＝万人の万人に対する闘争」**。統治なき宙域では、攻める意図がなくても防衛目的の武装がさらなる武装を呼び、全員が不安全な螺旋（安全保障ジレンマ）に陥る。→ 無政府宙域に**固有の不安定コスト**を与え、プレイヤーが「秩序を作る動機」を体感できる。
2. **「どんな暴君も無政府よりはマシ」**。自然状態の恐怖が極めて高いため、理性的な人間は悪い統治にも服従する（反直観）。→ 崩壊した統治の宙域で住民が**どんな新支配者でも受け入れる**（再統合が早まる）という数値的根拠を与える。
3. **「コヴェナント＝保護の交換」**。臣民が服従するのは保護を受けるためであり、守れない主権者への服従義務は消える。→ 既存の「抑圧→不同意」(`ConsentRules`) に**「保護失敗→契約解除」**という第二の離脱回路を加える。
4. **「分割された主権は主権ではない」**。二つの勢力が同一空間を同時に主権主張するとき、それは自然状態に等しい（どちらも相手を拘束できない）。→ 係争中の星系/回廊を「事実上の無政府」として**より高い不安定圧力**を与える。
5. **「死すべき神（Mortal God）」**。コモンウェルスは人工的に作られた「人格」（個人の集合的委任）であり、条件が崩れれば死ぬ。→ 国家崩壊のloreとして「秩序は自然でなく合意で成り立つ」を`DisclosureLedger`（秘史）に乗せる。
6. **「自然的平等＝最も強い者でも眠れば殺される」**。圧倒的強者も最低限の脅威フロアから逃れられない。→ 無政府宙域で最強勢力も安心できない**最小脅威フロア**として`AnarchyCostRules`に組み込む。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**ConsentRules/DynastyRules/DiplomacyRules/HegemonyRules/CoupRules を作り直さない**。LEVI はそれらに**欠落軸を足し、接続する**だけ（additive）。

### ★★★ 最優先（真の欠落・ホッブズの signature）

#### LEVI 無政府宙域の自然状態コスト（`AnarchyCostRules` / `AnarchyState`）
- **崩壊後状態のモデル化**：`FactionState.IsCollapsing` の後——勢力の崩壊で生じた無政府宙域（`AnarchyState`）に固有のコストを付与する。
  - 生産コスト：安定度0・`GovernanceRules.OutputFactor`=最低値（0.1程度）・造船停止
  - 拡散コスト：隣接支配勢力へ不安定圧力（`Province.stability` へのペナルティ係数）
  - 参入動機：無政府宙域の**「実効的再統合コスト」**が低い（誰が入っても歓迎される）＝自然状態の住民は「どんな統治者でも受け入れる」反直観の数値根拠
- **自然的平等フロア**：強勢力も無政府宙域では最小脅威0以下に下がれない（`MinThreatFloor`＝強さの逆数で脅威を保証→コアリション形成インセンティブ）。
- 接続：新 `AnarchyCostRules`（純ロジック・test-first）＋`AnarchyState`（純データ）。`FactionState.IsCollapsing` が true になった勢力のシステムに `AnarchyState` を発生させる（`GalaxyView` が配線）。`GovernanceRules.EquilibriumStability` に無政府ペナルティ係数を追加。

#### LEVI 安全保障ジレンマ（対称的軍拡スパイラル・`SecurityDilemmaRules`）
- **THU-1（覇権移行恐怖）との区別**：THU-1 は「台頭する勢力が引き起こす非対称的恐怖」。LEVI-2 は「誰も攻める意図がなくても防衛目的の武装が相互武装を呼ぶ対称スパイラル」。
- **アーミング圧力モデル**：
  - `ArmingPressure(faction, neighbors)` ＝隣接勢力の軍拡率（`FleetPool` の増分トレンド）の加重和→当勢力への軍拡圧力スコア
  - `SpiralIntensity(galaxyState)` ＝全勢力の平均アーミング圧力の上昇率。螺旋が深まるほど高くなる。
  - `SpiralWarRisk(intensity)` ＝螺旋強度が閾値を超えると「誰も望まない戦争」が確率的に発生（`DiplomacyRules.DeclareWar` の背景圧力として）
- **安全保障のジレンマ感**：防衛目的の建艦が隣接勢力の `ArmingPressure` を上げ、その勢力も防衛目的で建艦→元の勢力の圧力がさらに上がる自己強化ループ
- 接続：新 `SecurityDilemmaRules`（純ロジック・test-first）。`FleetPool` のトレンドデータを使う（新規フィールド）。`DiplomacyRules` に `SpiralWarRisk` を開戦判断の加算係数として追加。

### ★★ 高（既存への欠落軸の追加）

#### LEVI コヴェナント＝保護契約の閾値（`CovenantRules`）
- **ガンジー型(`ConsentRules`)との区別**：`ConsentRules` は「抑圧が大きいと協力を引き上げる（非服従・ボイコット）」——苦しいから抵抗する。ホッブズ型は「保護が失敗したとき、服従義務が消える」——守れないなら服従しない。
- **保護指数（ProtectionIndex）**：外敵からの防衛力・治安維持能力を 0..1 で表す。`FactionState`/`StrategicFleet` の防衛能力から算出。
- **コヴェナント解除閾値**：保護指数が `CovenantBreakThreshold`（既定0.2）を下回ると、その宙域の住民は「服従義務が消滅」→ `Polity.cooperation` へのショック低下（`ConsentRules.Withdraw` を呼ぶ）。重要：**抑圧が高くても保護があれば服従が続く**（`ConsentRules` と逆方向の補完）。
- **接続**：新 `CovenantRules`（純ロジック・test-first）。`GovernanceRules.Tick` / `FactionStateRules.Tick` からコールし、防衛失敗（辺境が占領された・補給線が断たれた・`SupplyRules.IsSupplied=false`）で `ProtectionIndex` を低下させ、閾値を切ると `ConsentRules.Withdraw` へ転送。

#### LEVI 主権不可分の圧力（`SovereigntyContestRules`）
- **二重請求の不安定化**：同一の星系/回廊に**二勢力が同時に主権主張**するとき（`FactionRelations.IsHostile` が true かつ占領状態が曖昧）、ホッブズ的意味で「どちらも相手を拘束できない＝自然状態と等値」→ 当該宙域の安定度ペナルティを追加。
- **不可分圧力スコア**：`ContestIntensity(systemId, factions)` ＝係争勢力数×勢力間力差の近似度（力が拮抗するほど強くなる）
- **エスカレーション増幅**：係争強度が高い星系で起きる小競り合い（`FleetRegistry` の戦闘）は `DiplomacyRules.opinion` への悪化が通常より速い（係争状態が普通の国境紛争より深刻）
- 接続：新 `SovereigntyContestRules`（純ロジック・test-first）。`StrategyRules.IsFtlBlocked` が true の回廊（前線）＋`GovernanceRules.Province.stability` へのペナルティ係数として機能。

### ★ 中（lore）

#### LEVI（lore）死すべき神・人工人格と秩序の起源
- 「コモンウェルスは自然ではなく、人々の合意で作られた人工的人格（Mortal God）であり、条件が崩れれば死ぬ」。
- 「どんな悪い統治も、秩序がないよりはマシ——それが自然状態の教訓」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）へのloreデータ入力。`DisclosureEntry` として「自然状態の開示」（無政府宙域が初めて生成された時に発火）＋「コモンウェルスの死」（勢力崩壊イベントの上位開示）。

### ❌ 不採用（重複・既存で十分）

| 不採用 | 理由 |
|---|---|
| 覇権移行恐怖・台頭勢力の先制戦 | **`HegemonyRules`（THU-1 #229）が既存でカバー**。LEVI は対称スパイラルのみ（非対称版は THU-1 で十分） |
| クーデター・反乱のロジック | **`CoupRules`/`ConsentRules` が既存でカバー**。LEVI は接続係数を足すだけ |
| 主権の委任・代表制・議会制 | **`ConstitutionRules`/`PartyRules` が既存でカバー**（ホッブズの権威主義的主権論はジョン・ロックへの否定として読める——採用しない） |
| 「自然権」としての自己保存権 | 個人スケールの権利モデルは既存 `Constitution.rightsProtection` 相当。新規追加は重複かつタイクン化 |
| ソブリンの無謬性（主権者の命令は常に正しい） | ゲームバランスを壊す。`CoupRules`/`DynastyRules.Reform` の既存機構で対処 |
| 宗教・教会との闘争（レヴィアタンvs教権） | **`ReligionRules`（#172-175）/SAW-7 が既存**。新規EPIC化しない |
