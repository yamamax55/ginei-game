# サピエンス全史 参考設計（EPIC #SAPI）

> 参照元：ユヴァル・ノア・ハラリ『サピエンス全史』。人類が他種を圧して地球を支配したのは、**実在しないもの（神・国家・貨幣・法人・人権）を集団で信じる能力＝虚構（共同主観的実在）**を獲得したからだ、という認知革命史観。**虚構を共有する者の数だけ協力できる**——150人（ダンバー数）の壁を越えた大規模協力は、すべて**想像上の秩序**の上に立つ。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に巨大な社会・政治・経済シミュ純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#SAPI` として issue 化する提案。
> 著作権注意：固有名・文章・図表・著者固有の用語は流用せず、**「虚構が大規模協力を生む」という構造パターン／メカニクスのみ**を参考にする。

## 0. なぜ「サピエンス全史」が本システムに役立つか — 各論はあるが「総論＝共通基盤」が無い

当プロジェクトは、正統性・宗教・貨幣・組織・法人格を**それぞれ別個の純ロジック**として大量に保有している：

| 既存（個別の制度・各論） | カバー範囲 |
|---|---|
| `DynastyRules`/`Regime`（#867） | 王朝の正統性・天命・腐敗 |
| `ReligionRules`/`Religion`（#172-175） | 改宗圧力・異端・聖戦・社会効果 |
| `FiscalRules`/`MarketRules`/`BankRules`（#161/#179/#186） | 為替・需給価格・信用創造（**貨幣の数量面**） |
| `Organization`/`SuccessionRules`（#812） | 英雄死後の組織存続（制度化 vs 個人カリスマ） |
| `Polity`/`ConsentRules`（#836） | 「権力は借り物」協力＝統治の前提 |
| `FactionState`/`CampaignRules`（統合層） | 国家状態の合成・実効安定度 |
| `DisclosureRules`/`DisclosureLedger`（FND-4 #495） | 秘史開示・世界観の連鎖開示 |
| `CultureRules`/`ReligionRules`/`PropheticAuthorityRules`（#194/#172/PRPH） | 文化同化・宗教発生・啓示的カリスマ |

**しかし、これらは「制度ごとに別々のスカラ」を持つ各論**であり、サピエンス全史が固有に描く「すべての大規模協力を支える**共通の認知基盤＝共有された虚構（imagined order）**」という**総論レイヤー**が**欠けている**：

| サピエンス全史が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **協力の規模は虚構の共有度で決まる**（ダンバー数150の壁→神話で突破） | 既存の協力（`Polity`協力/`Organization`結束）は**人数スケール上限を持たない**。「素では150人しか束ねられない／虚構を信じる者の数だけ協力が伸びる」回路が無い |
| **貨幣・国家・宗教・法人は同型の虚構**（信じられている限り実在する） | 各制度が**バラバラのスカラ**を持つ。「同じ"共有信念"という共通通貨で測れる／一つが崩れると同型に波及する」横断軸が無い |
| **貨幣は最も普遍的な相互信頼の制度**（価値は内在せず合意） | `FiscalRules.ExchangeRateFactor` は数量・財政の関数。**「貨幣の価値＝共同主観的信認そのもの」「信認崩壊＝ハイパーインフレ／物々交換回帰」**が無い |
| **虚構は信じられなくなった瞬間に消える**（崩壊の非連続性） | 既存の崩壊（`IsCollapsing`/`MandateLost`）は**個別制度の閾値割れ**。「**信念の臨界＝雪崩的に一気に虚構が無効化**」という共有信念の相転移が無い |
| **想像上の秩序は語り直しで維持・改変できる**（神話の再生産） | 開示`DisclosureLedger`は lore の解禁。**「為政者が物語を語り直して正統性を作る／プロパガンダ投資」**という能動的な虚構の生産が無い |
| **農業革命＝後戻りできない罠**（小麦が人を家畜化した／生活の質は下がった） | `ColonizationRules`は拡張＝善。**「人口は増えたが個々は不自由になった・もう戻れない不可逆な転換」**という両義的進歩が無い |
| **統一への力学＝貨幣・帝国・宗教の3つの普遍的秩序** | 統一は軍事征服中心。**「交易圏・宗教圏・帝国理念という非軍事の統合力が版図を一体化する」**回路が薄い |

**結論**：サピエンス全史は当プロジェクトに、点在する制度（正統性/宗教/貨幣/組織）を貫く**「共有された虚構＝大規模協力の共通基盤」という総論レイヤー**を与える。これは新制度の羅列ではなく、`CombatModifiers`#106（戦闘係数の一元化）・`CampaignState`（社会シミュの合成）の**信念版**＝**「想像上の秩序スタック」**。核心は4つ：①**虚構が協力の規模上限を決める**（ダンバー突破）②**貨幣/国家/宗教は同型の虚構**で共通の信認スカラで測れる③**信認は臨界で雪崩的に崩れる**（相転移）④**為政者は物語を語り直して虚構を生産できる**。既存の `FactionState`/`DynastyRules`/`ReligionRules`/`Polity`/`DisclosureLedger` を**作り直さず束ねて接続する**統合EPIC。

## 1. 役に立つ視点（要約）

サピエンス全史の世界観を、**本システムに効く形**で1行ずつ（既存issue/モジュールへの共鳴を明記）：

1. **大規模協力は共有された虚構の上にしか立たない**。素の信頼は小集団どまり、神話・国家・貨幣を信じる者の数だけ協力が伸びる。→ `Polity`協力#836・`Organization`結束#812 に**「人数スケールの天井と、虚構による突破」**を与える＝既存の協力に上限と梃子を導入。
2. **貨幣・国家・宗教・法人は同型の"想像上の秩序"**。価値は内在せず、信じられている限りだけ実在する。→ `DynastyRules`正統性#867・`ReligionRules`信仰#172・`FiscalRules`貨幣#161 を**同じ"共有信念"スカラで横断**＝制度間の共鳴（一つが崩れると同型へ波及）。
3. **貨幣は最も普遍的な相互信頼の機械**。信認が崩れれば紙くずに戻る（物々交換回帰）。→ `FiscalRules.ExchangeRateFactor`#161 に**信認次元**を足す＝財政健全性とは別の「信じられているか」軸。狼と香辛料SAW-1（正貨の品位）と一対＝**SAWは品位の物理、SAPIは信認の心理**。
4. **想像上の秩序は崩れるときは一気に崩れる**（信念の相転移）。→ `FactionStateRules.IsCollapsing`・`DynastyRules.MandateLost`・`BankRules`取付け#186 を**共有信念の臨界**として統一＝雪崩（関ヶ原#817 カスケードの信念版）。
5. **為政者は物語を語り直して虚構を再生産する**。神話・儀礼・教育・プロパガンダで正統性を能動生産。→ `DisclosureLedger`#495・`DynastyRules.Reform`#867・支持#113 に**「物語の生産」という高位の決断**を接続。
6. **進歩は不可逆な罠でもある**（農業革命＝人口増だが個は不自由・もう戻れない）。→ `ColonizationRules`#129・人口`DemographicsRules`#153 に**両義的転換（マクロ増・ミクロ後退・不可逆）**を足す＝開示`DisclosureLedger`のlore（進歩の代償）。
7. **統一の普遍力＝貨幣圏・帝国理念・宗教圏**（非軍事の統合）。→ `LogisticsRules`版図一体化#844・`DiplomacyRules`#189・`CampaignRules.EffectiveStability` に**非軍事の統合力（交易/理念/信仰で版図がまとまる）**を接続。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`DynastyRules`#867／`ReligionRules`#172／`FiscalRules`#161／`Polity`#836／`Organization`#812／`DisclosureLedger`#495 を作り直さない**。SAPI はそれらに**「共有された虚構という総論レイヤー」を足し、横断接続する**だけ（additive）。タイクン化回避＝物語の生産も虚構の崩壊も**高位の決断→エンジン駆動→創発帰結**（信念スカラのマイクロ管理はしない）。**PERF#1117 規律**＝信念は勢力/制度の集約スカラに留め（個人粒度に降りない）、評価は**暦境界Tick・変化時のみ再計算・差分収束**で回す。

### ★★★ 最優先（真の欠落・サピエンス全史の signature）

#### SAPI 共有された虚構と協力の規模上限（ダンバー突破）
- `SharedFictionRules`（static・test-first・純ロジック）＝勢力/組織の**協力可能規模**を「素の信頼（小集団上限）×虚構の共有度で梃子をかけた値」として算出。**信じる者が多く・虚構が強いほど大規模協力が解ける**。`ImaginedOrder`（共有信念の強度0..1＋浸透度＝信者比率）を data として持つ。
- 接続：`Polity`協力#836（協力の規模天井と梃子）・`Organization`結束#812（属人 vs 制度化＝虚構の制度性）・`FactionState`（合成へ）。**新メカニクスでなく既存協力にスケール上限と梃子を入れる**のが核。

#### SAPI 想像上の秩序スタック＝制度横断の信認（貨幣/国家/宗教の同型）
- `ImaginedOrderRules`（static）＝正統性（`DynastyRules`）・信仰（`ReligionRules`）・貨幣信認（`FiscalRules`）を**同一の"共有信念"スカラへ写像して合成**（`CombatModifiers`#106 が係数を一元化したのと同型の**信念版**）。一つの虚構の信認が落ちると**同型の制度へ部分波及**（共鳴）。**バラバラのスカラを束ねる横断窓口**。
- 接続：`DynastyRules`正統性#867・`ReligionRules`#172・`FiscalRules`#161・`FactionStateRules`（合成層・**新ルートでなく `FactionState` 配下に虚構スカラを足す**）。

#### SAPI 貨幣の信認次元とハイパーインフレ（信じられている限りの価値）
- `MoneyTrustRules`（static）＝貨幣価値を「財政健全性（既存）×**共同主観的信認**」へ分解。信認低下で**実効購買力が非線形に低下→閾値割れでハイパーインフレ／物々交換回帰**（数量でなく信頼の崩壊）。
- 接続：`FiscalRules.ExchangeRateFactor`#161（**信認次元を足す**・数量面は不変）・`BankRules`取付け#186（信認崩壊の銀行版と共鳴）。**狼と香辛料SAW-1（正貨の品位＝物理）と一対の"信認＝心理"**＝重複でなく相補。

### ★★ 高（信念の動学・物語の生産）

#### SAPI 信念の相転移（虚構は一気に崩れる）
- `BeliefCascadeRules`（static）＝共有信念が**閾値まではゆっくり減衰し、臨界で雪崩的に無効化**（連続でなく相転移）。関ヶ原#817 の寝返りカスケード（`LoyaltyRules.ResolveCascade`）を**「制度への信認」へ一般化**＝皆が「もう誰も信じていない」と気づいた瞬間に崩れる。
- 接続：`LoyaltyRules.ResolveCascade`#817（不動点反復を**信認カスケード**に流用・**ロジック委譲・重複新設しない**）・`FactionStateRules.IsCollapsing`・`DynastyRules.MandateLost`#867。崩壊の非連続性を統一窓口で。

#### SAPI 物語の生産（神話の語り直しで虚構を再生産する高位の決断）
- `MythMakingRules`（static）＝為政者が**儀礼/教育/プロパガンダ/記念で共有信念を能動的に生産・補修**する（`ImaginedOrder` の強度・浸透度を上げる）。効果は遅効・逓減・反発（露骨な捏造はバックファイア）＝タイクン回避の高位レバー。
- 接続：`DynastyRules.Reform`#867（正統性の更新）・支持#113・`DisclosureLedger`#495（語られた物語が開示・lore へ）・`NonviolenceRules`可視化#831（語りの届く範囲）。

### ★ 中（不可逆な進歩・非軍事の統一・lore）

#### SAPI 不可逆な転換と両義的進歩（農業革命の罠＝マクロ増・ミクロ後退）
- `IrreversibleTransitionRules`（static）＝大きな社会転換（定住/拡張/制度化）は**マクロ指標（人口/産出）を上げるが、ミクロ（個の自由度/生活の質）を下げ、後戻りできない**（ratchet）。`ColonizationRules`#129・`DemographicsRules`#153 に**両義性**を足す（拡張＝単純善でない）。
- 接続：`ColonizationRules`#129・`DemographicsRules`人口#153・`GovernanceRules`安定#109・`DisclosureLedger`（進歩の代償の lore）。

#### SAPI（lore）統一の普遍力と虚構の世界観データ
- 「貨幣圏・帝国理念・宗教圏という非軍事の統合力が版図を一体化する」「人類を束ねたのは虚構だった」「神は信じられている限り実在する」。**コード新設せず** `DisclosureLedger`（FND-4）への lore データ入力（CCX-6 世界観 codex 退避方針に一貫）＋`LogisticsRules`版図一体化#844 に**非軍事統合の係数**を薄く接続。
- 接続：`DisclosureLedger`#495・`LogisticsRules`#844・`DiplomacyRules`#189。コード新設は最小。

### ❌ 不採用（重複・既存で十分・タイクン化）

| 不採用 | 理由 |
|---|---|
| 宗教システムそのもの（改宗/異端/聖戦） | **`ReligionRules`#172-175 がカバー**。SAPI は宗教を「同型の虚構の一例」として横断接続するだけ・作り直さない |
| 正統性・王朝サイクルそのもの | **`DynastyRules`/`Regime`#867 がカバー**。SAPI は正統性を共有信念スカラへ写像して横断する |
| 貨幣の数量・需給・為替・信用創造 | **`FiscalRules`/`MarketRules`/`BankRules`#161/#179/#186 がカバー**。SAPI は「信認＝心理」次元のみ足す |
| 正貨の品位・改鋳・改鋳投機 | **狼と香辛料SAW-1/2 がカバー**（貨幣の物理面）。SAPI は信認の心理面で相補＝同一EPIC化しない |
| カリスマからの正統性創出・宗教の創始 | **PRPH-1/2 がカバー**（啓示的カリスマ）。SAPI は「虚構一般の生産」を横断レイヤーで足す・人物軸は不採用 |
| 文化同化・ナショナリズム・分離独立 | **`CultureRules`#194 がカバー**。SAPI は重複させず接続 |
| 認知バイアス・非合理の決断 | **行動経済学#BEH がカバー**。SAPI は「集団が共有する信念」＝別軸（個人のバイアスでなく共同主観） |
| 進化史・人類学の年表再現 | ゲームメカニクスでない・タイクン化＝lore（`DisclosureLedger`）に退避 |

> **取捨選択の明示**：SAPI は「全部入り」にしない。signature は**①虚構が協力規模を決める ②制度横断の信認スタック ③貨幣の信認次元 ④信念の相転移**の4つに集中し、宗教/正統性/貨幣量/品位/カリスマ/文化/認知バイアスは**既存（ReligionRules/DynastyRules/FiscalRules/SAW/PRPH/CultureRules/BEH）へ接続のみ**＝最大の価値は「点在する制度を貫く総論レイヤー」を一本入れることにある。

### 推奨着手順
`SAPI-1`（共有虚構と協力規模＝最も固有で欠落の大きい signature・既存協力に上限と梃子）→ `SAPI-2`（想像上の秩序スタック＝制度横断の信認＝総論レイヤーの核・各制度を束ねる）→ `SAPI-3`（貨幣の信認＝SAWと相補）→ `SAPI-4`（信念の相転移＝崩壊の統一）→ `SAPI-5`（物語の生産＝高位レバー）→ 残り（SAPI-6 不可逆な進歩／SAPI-7 lore）。

> いずれも既存（宗教/正統性/貨幣/協力/組織/開示）を**後退させず横断接続**する additive 設計。SAPI の収穫は、点在する制度ロジックを貫く**「共有された虚構＝大規模協力の共通基盤」という総論レイヤー**を一本入れること——`CombatModifiers`#106 が戦闘係数を、`CampaignState` が社会シミュを束ねたのと同型の、**信念の統合層**。武力でなく**共有された物語で銀河を束ねる**という王道路線の理論的背骨になる。
