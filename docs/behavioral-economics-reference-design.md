# 行動経済学（人は必ずしも合理的でない） 参考設計（EPIC #BEH）

> 参照元：行動経済学（カーネマン＆トヴェルスキー、セイラー他）。古典経済学が前提する**合理的経済人(ホモ・エコノミクス)は虚構**であり、人間の意思決定は**系統的なバイアス**（損失回避・アンカリング・サンクコスト・群集行動…）に支配される、という知見群。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略）にとって**役に立つ視点**だけを抽出し、EPIC `#BEH` として issue 化する提案。
> 著作権注意：学術分野の概念であり、固有名・人物・実験は流用せず、**意思決定バイアスの構造パターンのみ**を参考にする。

## 0. なぜ「行動経済学」が本システムに役立つか — 非合理の"総論"

本作の意思決定主体（AI将帥・人物・市場・群衆・有権者・敵勢力）は、暗黙に**「合理的エージェント」**として振る舞いがちだ。だが本作の魅力は銀英伝的な**「名将でも判断を誤る」「民衆は扇動される」「同盟は雪崩を打つ」**——つまり**非合理**にこそある。

実は、既存・着手済みのEPID群の多くは、**非合理の"各論"**だった：

| 既存EPIC（非合理の各論） | 扱う非合理 |
|---|---|
| OTH-3 疑心の自己強化（#1187） | 確証バイアス |
| SARA-6 誰も望まなかった戦争（#1224） | 合成の誤謬 |
| PRL-3 眠れる獅子（#1235） | 戦術的成功への過信・近視眼 |
| （バックログ）カーネマン『ファスト&スロー』 | 二重過程・損失回避・計画錯誤 |
| （バックログ）ル・ボン『群衆心理』 | 群集の非合理・感染 |
| `HopeRules`#852 末人・`LoyaltyRules`#817 | 希望/恐怖が合理を曲げる |

**しかし、これらを貫く"総論"＝意思決定バイアスの統一基盤が無い**。各EPICが個別にバイアスを実装すると重複・不統一になる。行動経済学は、`CombatModifiers`#106（戦闘の係数を一元化）の**意思決定版**＝**「決断バイアス・スタック」**を与える。あらゆる決断モジュール（決断#502・AI将帥・市場#179・選挙#159・投資`FiscalRules`・群衆士気）が、同じバイアス窓口を通して非合理になる。

| 行動経済学が固有に与えるもの | 当プロジェクトでの欠落 |
|---|---|
| **プロスペクト理論**＝損失を利得より重く見る・参照点依存 | 期待値で動くAI。**損失回避で撤退をためらう/勝ち逃げを焦る**非対称が無い |
| **ヒューリスティクスとバイアス**＝アンカリング/利用可能性/代表性/確証 | 情報を正しく評価するAI。**直近の出来事に引きずられる/印象で誤判断**が無い |
| **サンクコスト**＝投じた費用に縛られ撤退できない | 損切りできるAI。**「ここまで犠牲を払ったのに引けない」**泥沼化が無い |
| **双曲割引**＝目先を過大評価し長期を過小評価 | 一貫した時間選好。**近視眼的な財政/研究軽視**が無い |
| **群集行動・ハーディング**＝他者の模倣が連鎖（バブル/取付け/敗走） | 個別に判断する主体。**模倣の連鎖が暴走する**統一回路が無い |
| **フレーミング/ナッジ**＝提示の枠で同じ事実でも選択が変わる | 内容で判断。**見せ方が決断を動かす**回路が無い |

**結論**：行動経済学は当プロジェクトに**「決断バイアスの統一レイヤー」**を与える。これは新メカニクスの羅列ではなく、**既存の非合理EPID群を束ね、あらゆる意思決定に"人間らしい愚かさ"を一元的に注入する横断基盤**。`no-outcome-preview`（結果を予想で見せない）とも深く響く——バイアスは「主体が結果を読み違える」装置だから。

## 1. 役に立つ視点（要約）

1. **損は得の倍重い**。同じ大きさでも損失は利得より痛く、人は損を避けるために不合理な賭けに出る。
2. **最初の数字に縛られる**。アンカー(参照点)が判断を歪め、直近の鮮烈な出来事を過大評価する。
3. **沈んだ費用に縛られる**。引き返せば損が確定するから引き返せない——犠牲が大きいほど泥沼に残る。
4. **目先が大きく見える**。遠い利益を割り引きすぎ、長期投資(制度/研究/財政再建)を先送りする。
5. **皆がやるからやる**。他者の行動の模倣が連鎖し、バブル・取付け・敗走・流行を生む。
6. **見せ方が中身を上書きする**。同じ事実でも「9割生存」と「1割死亡」で選択が変わる。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`CombatModifiers`#106・決断#502・市場#179・`FiscalRules`・既存の非合理EPID(OTH-3/SARA-6/PRL-3) を作り直さない**。BEH は**バイアスの統一窓口**を新設し、既存の決断点がそれを"読む"形で接続する（additive）。タイクン化回避＝バイアスはAI/創発の質を上げるもので、プレイヤーにマイクロ管理させない。基準値非破壊（実効値パターン＝決断の修正子）。

### ★★★ 最優先（統一基盤・BEH の signature）

#### BEH 決断バイアス・スタック（意思決定の係数一元化）
- `DecisionBias`(struct)＋`BehavioralRules`(static・test-first)＝意思決定の期待値に**バイアス修正子を積む**統一窓口（`ModifierStack`#106 の決断版）。各バイアス(損失回避/アンカー/サンクコスト/割引/同調)を関数として提供し、AI・人物・市場が同じ窓口を通す。
- 接続：`CombatModifiers`#106（同じ設計思想）・決断#502・AI将帥・MKV 情報品質。**新規バイアスはここに足す**（各所にインライン実装しない）。

#### BEH プロスペクト理論（損失回避・参照点・確率加重）
- `ProspectRules`＝利得と損失を**参照点**からの差で評価し、損失を係数(既定2倍)で重く見る。確率は加重(低確率を過大・高確率を過小)。＝**撤退をためらう/勝ち逃げを焦る/低確率の博打に乗る**非対称な行動。PRL-2 窮鼠の博打の心理的基盤。
- 接続：決断#502・`WarGoalRules`厭戦・FleetAI 撤退判断・PRL-2 窮鼠・`FiscalRules`。

### ★★ 高（バイアスの各論）

#### BEH ヒューリスティクスと認知バイアス（アンカリング/利用可能性/代表性/確証）
- 判断の系統的歪み：直近・鮮烈な出来事を過大評価(利用可能性)、最初の情報に固着(アンカリング)、信じたい証拠を集める(確証＝OTH-3 と統合)。AIの情報評価・敵戦力見積もり・諜報解釈に歪みを入れる。
- 接続：MKV 情報品質・OTH-3 疑心（確証バイアスをここに集約）・諜報`EspionageRules`・VOY-3 過剰解釈（統合）。

#### BEH サンクコストとコミットメントのエスカレーション
- 投じた犠牲(兵力/資源/威信)が大きいほど**撤退の心理的コスト**が上がり、合理的には引くべき局面で泥沼に残る。敗勢の戦争継続・PRL-4 速戦への固執・失敗事業の延命。
- 接続：`WarGoalRules`厭戦（撤退閾値にサンクコスト係数）・PRL-4・メガプロジェクトPIL・FleetAI撤退。

#### BEH 双曲割引と時間非整合（近視眼）
- 目先の利益を過大評価し長期を割り引きすぎる＝制度化#812・研究投資・財政再建・インフラを先送りし、危機が来てから後悔。`TimeFlowRules` の暦圧縮とは別＝**主体の選好の歪み**。
- 接続：`FiscalRules`（近視眼的財政）・研究#123・`Organization`#812制度化投資・暦Tick。

#### BEH 群集行動とハーディング（模倣の連鎖）
- 他者の行動の模倣が連鎖＝バブル(SAW-5)・取付け(BankRun)・敗走の伝播・流行・世論の雪崩(ル・ボン)。個別判断が同調圧力で上書きされる統一回路。SARA-2 同盟連鎖・士気カスケードの心理基盤。
- 接続：SAW-5 コーナリング/バブル・`BankRules`取付け・士気`FleetMorale`・群衆心理(ル・ボン控え)・選挙#159世論。

### ★ 中（操作・lore）

#### BEH フレーミングとナッジ（見せ方が決断を動かす）
- 同じ事実でも**提示の枠**で選択が変わる(「9割生存」vs「1割死亡」)。プロパガンダ・名分(ZUO-3)・対話(DLG)・世論操作の効果がフレーミングで増幅。デフォルト選択(ナッジ)が行動を誘導。
- 接続：DLG 対話（#1156 選択肢の提示）・ZUO-3 名分・SAW-3 風説・選挙#159・プロパガンダ。

#### BEH（lore）合理的経済人の神話
- 「人は必ずしも合理的でない」「ホモ・エコノミクスは虚構」＝古典経済学の前提崩壊の開示データ。経済思想(スミス→行動経済学)の系譜・世界観codex。
- 接続：`DisclosureLedger`（FND-4）・経済思想。コード新設は最小。

### ❌ 不採用（重複・別ジャンル・タイクン化）

| 不採用 | 理由 |
|---|---|
| 係数スタックの再実装 | **`CombatModifiers`#106 の設計を流用**。BEH は決断版を新設し思想を共有 |
| 確証バイアスの二重実装 | **OTH-3/VOY-3 を BEH-3 に集約**（横断統一が目的） |
| バイアスをプレイヤーが手動設定 | タイクン化。バイアスはAI/創発の質＝自動で効く背景 |
| 心理状態の精密シミュ | マズロー#403/希望#852 の枠で足りる。BEH は決断の歪みのみ |
| 実在の実験・人物の再現 | 構造のみ。固有名は使わない |
