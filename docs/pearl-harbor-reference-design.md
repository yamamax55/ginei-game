# 真珠湾攻撃 参考設計（EPIC #PRL）

> 参照元：真珠湾攻撃（1941年・太平洋戦争の開戦）。経済封鎖で追い詰められた側が、**宣戦布告前の戦略的奇襲**で敵主力に一撃を加えたが、**戦術的大成功が戦略的大失敗**（相手国民を団結させ総力戦に引きずり込む＝「眠れる獅子を起こす」）に転じた史的構造。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋戦争・経済・諜報の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#PRL` として issue 化する提案。
> 著作権注意：固有名・実在人物・国名は流用せず、**「追い詰められた側の先制奇襲」と「戦術的勝利が戦略的破滅を招く」逆説の構造パターンのみ**を参考にする。

## 0. なぜ「真珠湾攻撃」が本システムに役立つか・SARA #1218 との対照

直前の**サラエボ事件 EPIC #1218（SARA）**は「**偶発の一撃が同盟連鎖で誰も望まない戦争へ暴走する**」（受動・無意図・連鎖）を描いた。真珠湾はその**対極**＝「**追い詰められた側が、意図的に・合理的に先制を選び、しかし戦略的に破滅する**」（能動・意図的・自滅）。両者で「**戦争はどう始まり、なぜ破滅するか**」の二類型が揃う。

| | SARA（サラエボ #1218） | PRL（真珠湾・本EPIC） |
|---|---|---|
| 開戦の性質 | 偶発・誰も望まない・連鎖暴走 | 意図的・追い詰められた末の賭け |
| 主体 | 制御不能なシステム | 合理的に先制を選ぶ弱者 |
| 破滅の理由 | 封じ込めの窓を逃す | 戦術的勝利が戦略的敗北を招く |

当プロジェクトの既存資産：

| 既存（戦争・経済・諜報） | カバー範囲 |
|---|---|
| `WarGoalRules`（DIP-3 #192） | casus belli・厭戦・講和 |
| SARA（#1218） | フラッシュポイント・最後通牒・動員圧力 |
| PRC-8 経済制裁・SAW 経済戦 | 禁輸・制裁・通商破壊 |
| （バックログ）孫子 | 詭道・奇襲・謀攻 |
| ZUO-3 名分（#1196） | 征伐/侵略の正当性ラベル |
| OTH 諜報・`EspionageRules` | 情報入手・工作 |

**しかし、真珠湾固有の以下が欠けている**：

| 真珠湾が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **戦略的奇襲**＝宣戦前の先制一撃（奇襲ボーナス×名分喪失） | 開戦は宣戦後。**宣戦前の奇襲＝大戦果と引き換えに大義を失う**トレードオフが無い |
| **窮鼠の開戦**＝封鎖で追い詰められた弱者の「座して死を待つより」博打 | 開戦は強者の選択。**追い詰められた弱者が破滅的賭けに踏み切る**決断モデルが無い |
| **戦術的勝利と戦略的敗北の乖離**＝一撃の成功が長期破滅を招く | 戦果＝利益。**短期の勝利が長期の破滅に転じる逆説（眠れる獅子）**が無い |
| **短期決戦 vs 総力戦**＝速戦で国力差を覆そうとして消耗戦に沈む | 国力差はある。**速戦ドクトリンが総力戦の現実に敗れる**動学が無い |
| **警告の軽視**＝情報はあったのに組織が動けず奇襲が成立 | 諜報は入手/発覚。**情報があっても組織が動けない（奇襲が成立する組織的理由）**が無い |

**結論**：真珠湾は当プロジェクトに**「弱者の窮鼠的先制」と「戦術的勝利が戦略的破滅を招く逆説」**を与える。経済戦（PRC-8/SAW）が**戦争の引き金**になり（封鎖→窮鼠）、奇襲が**短期勝利と引き換えに大義と長期を失う**。これはメモリ `no-outcome-preview`（結果を予想で見せない）と完全に響く——**奇襲の短期戦果は即座に見えるが、それが招く長期破滅は事後にしか分からない**。プレイヤーに「勝ったはずの一撃が、なぜ破滅へ繋がったのか」を体験させる。

## 1. 役に立つ視点（要約）

1. **追い詰められた者は博打を打つ**。封鎖で座して死を待つより、勝率の低い先制に賭ける（窮鼠）。
2. **先制は大義と引き換え**。宣戦前の奇襲は戦果を最大化するが、名分を失い相手に大義を与える。
3. **戦術的勝利は戦略的敗北を隠す**。鮮やかな一撃が、起こしてはならない相手を本気にさせる。
4. **速戦は総力戦に沈む**。国力で劣る側の短期決戦ドクトリンは、長期化すれば必ず消耗で負ける。
5. **情報があっても防げない**。警告は届いていても、組織が動けなければ奇襲は成立する。
6. **勝敗は事後にしか分からない**。一撃の戦果は即座に見えるが、その代償は時間が経って現れる。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`WarGoalRules`・SARA・PRC-8/SAW 経済戦・ZUO-3 名分・`EspionageRules` を作り直さない**。PRL はそれらに**「窮鼠的先制奇襲」と「戦術的勝利→戦略的破滅の逆説」**を足し、接続するだけ（additive）。タイクン化回避＝先制の決断は高位の選択、破滅は創発（事後判明）。

### ★★★ 最優先（真の欠落・PRL の signature）

#### PRL 戦略的奇襲（宣戦前の先制一撃＝戦果×名分のトレードオフ）
- `SurpriseAttackRules`（static・test-first）＝宣戦布告前の先制は**奇襲ボーナス**（初撃の戦果が大幅増・準備不足の敵）を得るが、**名分を失う**（ZUO-3 名分が負・第三国の対立感情・相手の戦意団結）。奇襲の成否は諜報優位×相手の警戒度。
- 接続：孫子 詭道（バックログ）・ZUO-3 名分（#1196）・`WarGoalRules`・SARA-3 最後通牒（決裂後の奇襲）・OTH 諜報。

#### PRL 窮鼠の開戦決断（封鎖で追い詰められた弱者の博打）
- `DesperationWarRules`（static）＝経済封鎖/禁輸（PRC-8/SAW）で**国力が逓減**する弱者が、「座して死を待つ」か「勝率の低い先制に賭ける」かの**窮鼠の決断**。追い詰められ度（封鎖の深さ×残存国力×時間）が閾値を超えると、合理的に**破滅的開戦**を選びうる。
- 接続：PRC-8 経済制裁・SAW 経済戦・`WarGoalRules`・`FiscalRules`（国力逓減）・SARA（危機段階）。

### ★★ 高（逆説・ドクトリン）

#### PRL 戦術的勝利と戦略的敗北の乖離（眠れる獅子を起こす）
- `PyrrhicSurpriseRules`（static）＝奇襲の**戦術的戦果**（即座に可視）が、相手の**国民団結・総力戦動員**（後から顕在化）を招く＝短期勝利が長期破滅に転じる逆説。戦果が大きいほど相手の結束も強まる非線形。`no-outcome-preview` 方針＝長期の代償は事後にしか見えない。
- 接続：`HopeRules`/支持#113（相手の団結）・`ReligionRules`/動員・`DisclosureLedger`（事後評価）・総力戦。

#### PRL 短期決戦ドクトリン vs 総力戦
- `DecisiveBattleDoctrineRules`（static）＝国力で劣る側の**速戦ドクトリン**（早期講和を狙う一点突破）vs **総力戦の現実**（長期化で国力差が顕在化）。速戦が決まれば講和（`WarGoalRules`）、決まらず長期化すれば消耗で敗北。賭けの構造。
- 接続：`WarGoalRules`（厭戦/講和）・国力（`FiscalRules`/兵站#92）・SGZ・八月の砲声（総力戦）。

### ★ 中（組織・lore）

#### PRL 警告の軽視と奇襲の成立（組織が動けない）
- `WarningNeglectRules`（static）＝奇襲の警告情報が**届いていても組織が動けない**（情報の埋没・縦割り・正常性バイアス）と奇襲が成立。情報優位（OTH/SAW-3）があっても**組織の対応力**が低いと防げない＝『失敗の本質』型の組織病理。
- 接続：OTH 諜報・省庁#158 縦割り・MKV 情報品質・`EventEngine`。

#### PRL（lore）だまし討ちの代償・眠れる獅子
- 「鮮やかな勝利が破滅の始まり」「だまし討ちは大義を相手に渡す」＝奇襲の逆説の開示データ。後世の評価（戦術的天才・戦略的愚挙）として `DisclosureLedger`。
- 接続：`DisclosureLedger`（FND-4）・ZUO-3 名分。コード新設は最小。

### ❌ 不採用（重複・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 経済制裁・禁輸そのもの | **PRC-8/SAW がカバー**。PRL は封鎖→窮鼠の決断を足す |
| 奇襲戦術の戦闘実装 | 会戦は既存戦術コア。PRL は戦略層の奇襲ボーナス係数のみ |
| 危機エスカレーションそのもの | **SARA #1218 がカバー**。PRL は決裂後の先制を接続 |
| 実在の作戦・人物・国名の再現 | 構造のみ。固有名は使わない |
| 諜報エンジンの再実装 | **`EspionageRules`/OTH がカバー**。警告の軽視は組織対応力の係数 |

> いずれも既存（戦争/経済/諜報）を**後退させず接続**する additive 設計。SARA #1218（偶発の暴走）と対をなし、**「追い詰められた側の合理的だが破滅的な先制」**という開戦の第二類型を足す。経済戦（PRC-8/SAW）が戦争の引き金になり、奇襲が短期勝利と引き換えに大義と長期を失う——**`no-outcome-preview`（結果を予想で見せない）が最も活きる**＝勝ったはずの一撃の代償が、時間を経て破滅として現れるのが最大の収穫。
