# 銃・病原菌・鉄 参考設計（EPIC #GGS）

> 参照元：ジャレド・ダイアモンド『銃・病原菌・鉄』。**なぜ大陸ごとに文明の進度が違ったか**を、人種でなく**地理（環境）の不均等**で説明する地理決定論。
> 大陸の**軸の向き（東西 vs 南北）**が技術・作物・家畜・病原菌の**伝播速度**を決め、**初期賦存（栽培化可能な動植物の偶然）**が出発点の格差を生み、**集住が病原菌を育て**、それが征服の武器になった——という「**地理→賦存→伝播→帰結**」の構造。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋手続き銀河 G-1 #118・群像/政治/経済の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#GGS` として issue 化する提案。
> 著作権注意：固有名・文章・特定文明の固有設定は流用せず、**地理→伝播の構造パターン／メカニクスのみ**を参考にする（特定の人種・民族・実在文明を「優劣」として扱わない＝本書の主旨そのものに沿う）。

## 0. なぜ「銃・病原菌・鉄」が本システムに役立つか

当プロジェクトは星系・回廊・内政・伝播の**純ロジックを既に保有**している（[CLAUDE.md] 参照）：

| 既存（関連する土台） | カバー範囲 |
|---|---|
| `GalaxyMap`/`StarSystem`/`Corridor`（C-1） | 星系ノード＋回廊エッジのグラフ。`systemType`(工業/農業/鉱業/居住)・`owner` |
| 創発キャンペーン G-1 #118（設計のみ） | `GalaxyGenerator` 構想＝チョーク保証の手続き生成（**生成原理は未確定**） |
| `ResearchRules`（#123-127） | 研究出力・政体偏り（**抽象的な研究点・単一勢力内で閉じる**） |
| `LogisticsRules`（GEO-3 #844） | 所有星系の回廊連結＝版図一体化度（**伝播でなく到達性**） |
| `CultureRules`/`Culture`（#194） | 同化圧力・分離独立（**勢力内の被支配民族の同化**） |
| `GovernanceRules`/`Province`（#109） | 安定度・統合度・産出（賦存を**消費する**側） |
| `ColonizationRules`（#129） | 入植＝1惑星→銀河へ拡張（**人の拡散だが伝播の異方性は無い**） |
| `PIL-3`（#1092・技術は人に宿る） | ネームド技術者が工法を保持・移籍/亡命で伝播（**人スケールの伝播**） |
| `DisclosureRules`/`DisclosureLedger`（FND-4） | 秘史・世界観の開示データ入力 |

**しかし、これらは「在る盤面の上で」動くロジックであり、銃・病原菌・鉄が固有に描く「盤面そのものがなぜ不均等か／不均等がどう伝播を決めるか」が欠けている**：

| 銃・病原菌・鉄が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **初期賦存の偶然＝出発点が地理で決まる**（栽培化可能種・家畜の有無） | G-1 #118 は「チョーク保証」止まり。**星系賦存（生産性ポテンシャル）の非対称を地理から決定論的に生成**する原理が無い。賦存は手置き |
| **大陸の軸の向きが伝播速度を決める（東西は速い・南北は遅い）** | 伝播の**異方性**が無い。`CultureRules` 同化も `ResearchRules` も「速さは地理に依らない」。回廊は等質 |
| **技術/作物は適地が似た区域を伝う（緯度＝気候帯）** | 星系類型（`systemType`）が**伝播の媒質**になっていない。同類星系間は速く、異類を跨ぐと遅い、という回路が無い |
| **病原菌は集住が育て、征服の非対称武器になる**（免疫の差） | 疫病が無い。**集住度→疫病発生→未接触集団への壊滅的非対称ダメージ**（戦わずに人口が崩れる）が無い |
| **地理が文明の進度を決める（運の格差）が、人種の優劣ではない** | 勢力間の**初期国力差を地理＝環境から正当化**する説明軸が無い（賦存の偶然＝運） |
| **障壁（砂漠・山脈）が伝播を分断＝孤立が遅滞を生む** | 回廊型（通商/要衝）はあるが**伝播障壁としての地形**（拡散を減衰させる回廊）が無い |
| **賦存の差が拡散ポテンシャル（人口・余剰）を決め、それが征服力に転化** | 賦存→余剰→拡散→征服 の**創発連鎖**が一本に繋がっていない |

**結論**：銃・病原菌・鉄は当プロジェクトに**「盤面の不均等を地理から生成する原理」と「不均等が伝播を方向づける異方性」**という2つの欠落軸を与える。これは①G-1 #118 手続き銀河の**生成原理**（賦存の地理的非対称）と②伝播の**異方性エンジン**（軸の向き・媒質・障壁）を新設し、既存の `ResearchRules`/`CultureRules`/`ColonizationRules`/`GovernanceRules` を**勢力内で閉じた値から勢力をまたぐ拡散場へ接続する結節点**になる。PIL-3（人スケールの伝播）とは直交＝こちらは**構造スケール（地形が決める拡散場）**。

## 1. 役に立つ視点（要約）

銃・病原菌・鉄の構造を、**本システムに効く形**で1行ずつ：

1. **地理が出発点を決める（運であって優劣でない）**。栽培化可能種の偶然が初期賦存を分けた。→ G-1 #118 の銀河生成に**賦存の地理的非対称**を与える生成原理。初期国力差を環境＝運で正当化（[no-outcome-preview] と整合＝宿命でなく偶然）。
2. **軸の向きが伝播速度を決める**。東西（同緯度＝同気候）は速く、南北は遅い。→ 回廊に**伝播の異方性**（方向で拡散係数が変わる）を入れ、回廊の戦略価値を「通行」から「伝播」へ拡張。
3. **技術は適地が似た区域を伝う**。作物は気候帯に沿って広がる。→ 星系類型（`systemType`）を**伝播の媒質**にし、同類間は速く・異類跨ぎは遅い拡散へ。
4. **病原菌は集住が育て、未接触集団を壊滅させる**。免疫の非対称が征服を決めた。→ **戦わずに人口が崩れる**非対称ショック＝疫病を `GovernanceRules`/`Population` へ。通商破壊#95・経済戦と並ぶ「第四の非武力崩壊」。
5. **障壁が伝播を分断し孤立が遅滞を生む**。砂漠・山脈が拡散を止めた。→ 回廊型に**伝播障壁**を足し、孤立星系の技術遅滞＝賦存の不利を増幅。
6. **賦存→余剰→拡散→征服の連鎖**。豊かな出発点が人口余剰を生み、それが拡張の燃料になった。→ 賦存（地理）から `ColonizationRules`（拡散）・国力（征服）までを**一本の創発連鎖**で結ぶ。
7. **地理は宿命ではなく初期条件**。後の制度・政体（創発キャンペーン I-1）が地理の与件を活かすか殺すか。→ #117 創発（地理→政体）と共鳴。地理決定論を**決定論で終わらせず**創発の入力にする。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**手続き銀河 G-1 #118・`ResearchRules`・`CultureRules`・`ColonizationRules`・`GovernanceRules` を作り直さない**。GGS はそれらに**地理の非対称（生成）と伝播の異方性（拡散場）を足し、接続する**だけ（additive）。
> タイクン回避：プレイヤー操作は増やさない。賦存はシード固定の**生成時に一度決まる**与件、伝播・疫病は**暦境界 Tick のエンジン駆動**、帰結は創発（事前確率を見せない）。

### ★★★ 最優先（真の欠落・GGS の signature）

#### GGS 賦存の地理的非対称（手続き銀河の生成原理）
- **`Endowment`/`EndowmentRules`**（純ロジック・test-first）：星系に**賦存ポテンシャル**（生産性・栽培化可能性・資源豊度）を持たせ、G-1 #118 の生成時に**位置・類型・近傍構造から決定論的に**割り振る（シード固定＝再現可能）。賦存は**偶然の格差**＝特定星系が「たまたま豊か」。
- 高位の決断不要＝**生成時に一度決まる与件**。`GovernanceRules.OutputFactor` の上限ポテンシャルとして効き、`Province` の産出を地理で底上げ/抑制（基準非破壊＝実効値パターン）。
- 接続：G-1 #118（`GalaxyGenerator` の生成原理）×`StarSystem.systemType`×`GovernanceRules`。**初期国力差を環境＝運で説明**する軸（人種優劣でない＝本書の主旨）。

#### GGS 伝播の異方性（軸の向き・媒質・障壁＝拡散場）
- **`DiffusionRules`/`DiffusionField`**（純ロジック・test-first）：技術・作物（賦存）・文化が回廊グラフ上を**異方的に拡散**する。拡散係数＝f(回廊の軸方向, 端点の `systemType` 一致, 障壁係数)。**同類星系・同軸方向は速く、異類跨ぎ・障壁回廊は遅い**。`GalaxyMap` 上で `CalendarDispatcher` の月次/年次 Tick で拡散を進める。
- `ResearchRules` の研究点を**勢力内で閉じた値から、隣接星系へ漏出・流入する拡散場へ**接続（先進星系の隣は速く追いつく＝賦存の不利を地理で部分補償）。
- 接続：`GalaxyMap`/`Corridor`×`ResearchRules`#123×`CultureRules`#194（同化も拡散場で）×`CalendarDispatcher`（TIME-6）。回廊の戦略価値を「通行」から「**伝播の動脈**」へ拡張。

### ★★ 高（非武力の征服・賦存の連鎖）

#### GGS 疫病と免疫の非対称（戦わずに人口が崩れる）
- **`Epidemic`/`EpidemicRules`**（純ロジック・test-first）：**集住度（人口密度・交易接続）が疫病を育て**、保有集団は免疫を得る。未接触集団（孤立・低交易）への**接触で非対称ショック**＝人口（`Population`/`Province.population`）が崩落（戦闘なしの占領容易化）。免疫差は接触履歴から決まる。
- 高位の決断は無し（接触＝交易/侵攻の副作用として創発）。`DiffusionRules` と同じ拡散場を**負の伝播**として共有（病原菌も軸・媒質で広がる）。
- 接続：`DiffusionRules`（GGS-2 の拡散場を流用）×`DemographicsRules`/`Population`（LIFE-3）×`GovernanceRules`（安定度ショック）。通商破壊#95・制裁・経済戦と並ぶ**第四の非武力崩壊**。

#### GGS 賦存→余剰→拡散の連鎖（拡散ポテンシャル）
- 賦存（GGS-1）が高い星系ほど**人口余剰**を生み、それが `ColonizationRules`（#129 入植）の**拡散燃料**になる＝豊かな出発点が拡張力を持つ。賦存の不利な勢力は拡散が遅い（地理の不利が拡張力の不利へ転化）。
- 接続：GGS-1×`ColonizationRules`#129×`DemographicsRules`（人口ボーナス→拡散）。賦存→征服までを**一本の創発連鎖**に。

### ★ 中（障壁地形・lore）

#### GGS 伝播障壁としての地形（孤立が遅滞を生む）
- `Corridor` に**伝播障壁係数**を足す（既存の `CorridorType{通商,要衝}` を拡張・既定=障壁なし＝後方互換）：障壁回廊は通行できても**拡散を強く減衰**する＝砂漠/山脈の星間版。孤立星系は技術が伝わらず賦存の不利が増幅（GGS-2 の異方性に統合）。
- 接続：`Corridor`×`DiffusionRules`（GGS-2 の係数の一項）×G-1 #118（生成時に障壁回廊を配置）。**新ルール新設でなく GGS-2 の拡散係数へ統合**（最小実装）。

#### GGS（lore）地理決定論の開示データ
- 「**強さは血でなく地理＝運の格差**」「孤立は遅滞を生む」「賦存の与件を制度が活かすか殺すか（#117 創発）」型の世界観 lore。地理決定論を**宿命でなく初期条件**として提示（決定論の罠を避ける開示）。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。創発キャンペーン I-1 #120（地理→政体）と連動。

### ❌ 不採用（重複・既存で十分・タイクン化）

| 不採用 | 理由 |
|---|---|
| ネームド技術者が工法を保持・人の移動で伝播 | **PIL-3（#1092）がカバー**。GGS は人スケールでなく**構造スケール（地形の拡散場）**＝直交。重複新設しない |
| 研究ツリー・研究点の産出そのもの | **`ResearchRules` #123-127 がカバー**。GGS は研究点に**拡散場（勢力をまたぐ漏出）を接続するだけ** |
| 同化・分離独立・ナショナリズム | **`CultureRules` #194 がカバー**。GGS は同化を**異方拡散場へ接続するだけ**（係数を地理で変える） |
| 入植・植民の成立進捗 | **`ColonizationRules` #129 がカバー**。GGS は賦存→拡散燃料の**接続のみ** |
| 版図の到達性・連結成分 | **`LogisticsRules` #844 がカバー**。GGS は「到達」でなく「**伝播速度（異方性）**」＝別軸 |
| 個別作物/家畜/栽培化の細目シミュ | **タイクン化**（マイクロ管理）。賦存は**1つのポテンシャル値**に集約（個体粒度へ降りない＝PERF #1117） |
| 人種・民族の優劣付け | **本書の主旨に真っ向から反する**。賦存差は**地理＝運**として扱い、特定実在文明を優劣化しない（著作権・倫理） |
| 大砲/製鉄/銃の固有技術ツリー | 技術の中身は `ResearchRules`/`ResearchField` の抽象点で十分。GGS は**伝播の地理**のみ |

### 推奨着手順
`GGS-1`（賦存生成＝最も固有で欠落の大きい signature・盤面の不均等の出所）→ `GGS-2`（伝播の異方性＝拡散場の基盤）→ `GGS-3`（研究/文化を拡散場へ接続＝既存モジュールの活用）→ `GGS-6`（障壁＝GGS-2 の係数拡張で最小）→ `GGS-4`（疫病＝拡散場の負伝播応用）→ `GGS-5`（賦存→拡散の連鎖＝創発の結節）→ `GGS-7`（lore）。
