# 参考EPIC 探索軸（multi-modal 発掘の定義）

> 自動補充ループ（`worldview-backlog-refill.yml`）が、**この表の各軸を並列に**掘って [`reference-epic-backlog.md`](./reference-epic-backlog.md) へネタを供給する。
> 各軸のエージェントは**互いの守備範囲を知らないまま**（blind）発掘する＝[`parallel-core-fanout.md`](./parallel-core-fanout.md) の multi-modal sweep。だから「名著」1軸では出てこない、自分の概念外のネタが入る。
> **拡張点**：軸を増やしたければこの表に1行足すだけ。次の補充サイクルから新軸が並列に掘り始める（例「建築」「料理」「疫病」「音楽」を足す等）。
> バックログの名称列に `[軸名] 名称` の形でプレフィックスを付ける（無印＝名著・後方互換）。

| 軸 | 探索指示（このカテゴリで「本システム＝銀英伝風の星間国家戦略＋思想・歴史・経済・群像シミュ」の欠落軸に効くものを探す） |
|---|---|
| 名著・作品 | 古典・小説・歴史書・思想書・SF・ゲーム。構造パターン/メカニクスを供給するもの |
| 国家・文明 | 興亡・統治形態・制度の歴史的実例（ローマ/モンゴル/ヴェネツィア共和国/オスマン/ハンザ同盟/江戸幕府…）。政体・経済・版図の構造 |
| 言語・文字 | 表語/表音・敬語体系・外交言語・失われた言語・記録媒体。コミュニケーション・文化・記録の構造 |
| 軍事ドクトリン | 電撃戦/縦深防御/海上封鎖/ゲリラ戦/制空権/任務指揮/ランチェスター。戦術・作戦術・戦略思想 |
| 宗教・神話 | 宇宙論・宗派分裂・儀礼・聖職位階・神話体系（#305系の「風」モチーフ＝固有名/教義不使用・構造のみ） |
| 政治思想・イデオロギー | 自由主義/社会主義/保守主義/無政府主義/ナショナリズム。正統性・動員・対立軸 |
| 制度・組織 | 官僚制/ギルド/株式会社/大学/常備軍/議会。組織の発生・運営・腐敗・継承 |
| 技術・発明 | 印刷/火薬/鉄道/電信/暗号/標準化。社会を変えた技術と軍事・経済・政治への波及 |
| 歴史的事件・転換点 | サラエボ/真珠湾型＝単一事件が構造連鎖で大変動を起こす。革命・恐慌・疫病・遷都 |
| 人物・指導者の類型 | カリスマ/改革者/反逆者/簒奪者/殉教者。固有名でなく「型」として（正統性・継承・群像） |
| 経済現象 | バブル/恐慌/ハイパーインフレ/交易網/独占。市場・金融・兵站の動学 |
| 自然・地理 | 地政学/チョークポイント/環境収奪/気候変動/災害。盤面と版図の構造 |
