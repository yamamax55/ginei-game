# サラエボ事件 参考設計（EPIC #SARA）

> 参照元：サラエボ事件（1914年・第一次世界大戦の引き金）。一発の銃弾による暗殺という**単一の偶発事件**が、最後通牒・同盟網の自動連鎖・動員計画の硬直性を通じて、**誰も望まなかった世界大戦**へ数週間で拡大した史的構造（七月危機）。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋外交・同盟・戦争の純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#SARA` として issue 化する提案。
> 著作権注意：固有名・実在人物・国名は流用せず、**「局地的偶発事件が構造連鎖で大戦に拡大する」危機エスカレーションの構造パターンのみ**を参考にする。

---

## 0. なぜ「サラエボ事件」が本システムに役立つか

当プロジェクトは外交・同盟・戦争の純ロジックを保有している（[CLAUDE.md] 参照）：

| 既存（外交・同盟・戦争） | カバー範囲 |
|---|---|
| `DiplomacyRules`/`TreatyRules`（DIP-1/2 #190/#191） | 二国間の外交状態・条約・同盟 |
| `WarGoalRules`（DIP-3 #192） | casus belli・厭戦・講和受諾 |
| SGZ-1 多極均衡（#1103） | 最強台頭→弱小連衡 |
| ZUO-2 会盟（#1195） | 多国間盟約・違背→諸侯連合の討伐 |
| OTH 失脚工作・`EspionageRules` | 諜報・人物標的工作 |
| `EventEngine`#116 | 条件発火イベント |
| （バックログ）タックマン『八月の砲声』 | 動員計画の自動性・止められない動員 |

**しかし、これらが扱わないサラエボ事件固有の「危機の暴走」**がある：

| サラエボ事件が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **フラッシュポイント**＝単一の偶発事件（暗殺/テロ/国境衝突）が危機の引き金 | 戦争は意図的開戦（`WarGoalRules`）のみ。**偶発の一撃が危機を点火する**回路が無い |
| **最後通牒**＝受諾不可能な要求を突きつける意図的エスカレーション | 宣戦/講和はあるが、**「呑めない要求で開戦を正当化する」外交手段**が無い |
| **同盟網の自動連鎖参戦**＝一国の参戦が条約で芋づる式に大国を巻き込む | 同盟・会盟はあるが、**局地戦→世界大戦へのカスケード（巻き込み）**が自動で走らない |
| **動員の先制圧力**＝先に動員しないと負ける→誰も止められない動員競争 | 動員はコスト。**「動員したら後戻りできない・先手必勝が全員を縛る」**先制圧力が無い |
| **エスカレーション・ラダー**＝封じ込め可能な窓と、越えると不可逆な閾値 | 危機の段階管理が無い。**外交で止められる窓と、過ぎたら止まらない閾値**が無い |
| **誰も望まなかった戦争**＝個々は合理的なのにシステムが暴走 | 戦争は主体の意志の産物。**創発的な制御不能（合成の誤謬）**が無い |

**結論**：サラエボ事件は当プロジェクトに**「局地的な偶発事件が、同盟と動員の構造を通じて、誰も望まないのに世界大戦へ暴走する」連鎖カスケード**を与える。SGZ-1（多極均衡）・ZUO-2（会盟）・DIP（外交）は**秩序が保たれる**側を描くが、SARA は**その秩序が一撃で雪崩を打って崩れる**側＝危機管理の失敗を描く。バックログのタックマン『八月の砲声』（動員計画の自動性）の**引き金側**を担い、両者で「七月危機」の全体が動く。プレイヤーに**「封じ込められたはずの小さな火種を見逃すと、構造が勝手に大戦を起こす」**緊張を与える。

---

## 1. 役に立つ視点（要約）

1. **大戦は小さな火種から始まる**。意図的開戦でなく、偶発の一撃が危機を点火し、構造が拡大させる。
2. **外交は戦争の道具にもなる**。呑めない最後通牒は「相手が拒否したから正当」という開戦の儀式。
3. **同盟は安全保障であり連鎖爆弾でもある**。守ってくれる条約が、他人の戦争へ自分を引きずり込む。
4. **動員は後戻りできない**。先に動かないと負ける構造が、全員を「止めれば負け」に縛る（先制圧力）。
5. **危機には窓がある**。封じ込め可能な段階と、越えると不可逆な閾値＝外交が間に合うかの競争。
6. **誰も望まなくても起きる**。個々の合理的選択の合成がシステムを暴走させる（合成の誤謬）。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**DIP#190/191/192・SGZ-1・ZUO-2・`EventEngine` を作り直さない**。SARA はそれらに**「危機エスカレーションの連鎖」**を足し、接続するだけ（additive）。タイクン化回避＝フラッシュポイントへの対応（封じ込め/エスカレート）は高位の決断、連鎖の暴走は創発。

### ★★★ 最優先（真の欠落・SARA の signature）

#### SARA フラッシュポイントと危機の点火（偶発の一撃）
- `FlashpointRules`（static・test-first）＝暗殺/テロ/国境衝突/挑発などの**偶発事件**（`EventEngine` 発火・諜報OTH 由来も可）が、関係勢力間の**危機度（0..1）**を点火・上昇させる。危機度は対応（封じ込め/譲歩/強硬）で上下し、閾値で次段階（最後通牒・動員）へ。
- 接続：`EventEngine`#116・`DiplomacyRules`（opinion）・OTH 失脚工作（人為的フラッシュポイント）・`NotificationCenter`。

#### SARA 同盟網の自動連鎖参戦（局地戦→世界大戦のカスケード）
- `AllianceChainRules`（static）＝一勢力が参戦すると、その**同盟/会盟の参戦義務**（DIP-2 `TreatyRules`/ZUO-2 `CovenantRules`）が連鎖発火し、芋づる式に大国を巻き込む＝**局地戦が世界大戦化**するカスケード。連鎖の深さは条約網の密度で決まり、中立宣言・条約不履行（信用低下と引き換え）で止められる。
- 接続：`TreatyRules`（DIP-2）・ZUO-2 会盟・SGZ-1（多極の巻き込み）・`WarGoalRules`。

### ★★ 高（外交手段・動員圧力）

#### SARA 最後通牒（呑めない要求＝意図的エスカレーション）
- `UltimatumRules`（static）＝受諾の難易度を**意図的に上げた要求**を突きつける外交手段。受諾＝相手の屈服（威信・領土の獲得）／拒否＝**正当化された開戦**（名分 ZUO-3 が立つ）。第三国の世論・参戦判断にも影響。
- 接続：`WarGoalRules`（DIP-3 casus belli）・ZUO-3 名分（#1196）・VEN 文言主義（要求条項）・支持#113。

#### SARA 動員の先制圧力（止められない動員競争）
- `MobilizationRules`（static）＝危機度が閾値を超えると動員が始まり、**先に動員した側が優位・遅れると不利**（先制圧力）＝双方が「止めれば負け」に縛られる。動員は**後戻りコストが高く**（解除すると無防備の窓）、自動エスカレーションを生む。バックログ『八月の砲声』（動員計画の自動性）の引き金側。
- 接続：`WarGoalRules`・兵站#92（動員＝戦力展開）・SGZ・（後続）八月の砲声EPIC。

### ★ 中（管理・lore）

#### SARA エスカレーション・ラダーと危機管理
- 危機を**段階（ラダー）**で表現：外交抗議→最後通牒→部分動員→総動員→開戦。各段に**封じ込めの窓**（首脳会談・第三国仲裁・譲歩）と、越えると**不可逆な閾値**。プレイヤーの危機管理（どこで降りるか）が試される。
- 接続：`DiplomacyRules`・`EventEngine`・ZUO 会盟（仲裁）・`NotificationCenter`（段階通知）。

#### SARA（lore）誰も望まなかった戦争（合成の誤謬）
- 「個々は合理的なのにシステムが暴走する」「封じ込めの窓を逃した代償」＝危機管理の開示データ。後世の評価（避けられたか）として `DisclosureLedger`。
- 接続：`DisclosureLedger`（FND-4）・`EventEngine`。コード新設は最小。

### ❌ 不採用（重複・別ジャンル）

| 不採用 | 理由 |
|---|---|
| 外交状態・条約そのもの | **DIP-1/2 がカバー**。SARA は連鎖と危機段階を足す |
| casus belli・講和そのもの | **DIP-3 がカバー**。最後通牒はその上の手段 |
| 動員計画の詳細（鉄道時刻表型） | **バックログ『八月の砲声』が担う**。SARA は先制圧力の点火のみ |
| 実在の国名・人物・事件の再現 | 構造のみ。固有名は使わない |
| 暗殺の実行ミニゲーム | OTH 失脚工作/諜報の枠で抽象化（フラッシュポイントの一入力） |

`SARA-1 → SARA-2`（点火＋連鎖＝カスケードの核）→ `SARA-3`（最後通牒）→ `SARA-4`（動員圧力）→ `SARA-5`（危機管理）→ `SARA-6`。

> いずれも既存（外交/同盟/戦争）を**後退させず接続**する additive 設計。SGZ-1/ZUO-2/DIP が描く**秩序の維持**に対し、SARA は**その秩序が一撃で雪崩を打つ崩壊**を足す＝危機管理の失敗。バックログ『八月の砲声』（動員の自動性）の**引き金側**を担い、両者で七月危機の全体が動く。**「封じ込められたはずの小さな火種を見逃すと、構造が勝手に大戦を起こす」**——プレイヤーが望まなくても戦争が起きうる緊張が、戦略レイヤーに入るのが最大の収穫。
