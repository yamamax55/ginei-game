# 近代世界システム 参考設計（EPIC #WSYS）

> 参照元：イマニュエル・ウォーラーステイン『近代世界システム』。世界を**単一の分業体制（世界＝経済）**として捉え、
> **中核（core）・半周辺（semi-periphery）・周辺（periphery）**の三層が**不等価交換**で結びつき、剰余が周辺から中核へ吸い上げられる構造論。
> さらに**ヘゲモニー（覇権国）が生成→絶頂→衰退する長期循環**と、半周辺の**上昇/転落の流動**を描く。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋巨大な社会・経済純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#WSYS` として issue 化する提案。
> **著作権注意：固有名・文章・固有設定は流用せず、世界システム論の構造パターン／メカニクスのみを参考にする。** 用語（中核/半周辺/周辺/ヘゲモニー等）は学術一般語として概念のみ使用。

## 0. なぜ「近代世界システム」が本システムに役立つか

当プロジェクトは社会・政治・経済・地理の**マクロ純ロジックを大量に保有**している（[CLAUDE.md]「テスト基盤」節 参照）：

| 既存（カバー範囲） | 何を持っているか |
|---|---|
| `GovernanceRules`/`Province`（#109） | 星系・惑星の内政（安定度・統合度・思想・産出）。単一星系の均衡 |
| `CampaignState`/`CampaignRules`/`FactionState`（統合層） | 勢力レベルの社会シミュ合成＋盤面進行。`EffectiveStability`/`LeadingFaction` |
| `LogisticsRules`（GEO-3 #844） | 所有星系の**連結成分→一体化度**（散在版図は国力割引） |
| `SupplyRules`/`CommerceRaidingRules`（#94/#95） | 補給線・通商破壊（補給源から所有回廊で到達・遮断で枯渇） |
| `MarketRules`/`Market`/`Good`（#179-182） | **単一市場**の需給均衡価格・生活水準→支持 |
| `FiscalRules`/`RedistributionRules`（#161-163） | 国債/金利/為替・**階級別**税負担・累進/逆進・階級対立 |
| `ResourceProductionRules`（#93） | 星系類型×安定度比例の資源産出 |
| `DiplomacyRules`（#189） | 外交状態×opinion→`FactionRelations.IsHostile` |
| `ResearchRules`（#123-127） | 研究産出・政体偏り |
| `DisclosureRules`/`DisclosureLedger`（FND-4 #495） | 世界観lore の条件開示エンジン |

**しかし、これらは「単一星系の内政」「単一市場の均衡」「勢力ごとの社会合成」**であり、世界システム論が固有に描く以下が**欠けている**：

| 近代世界システムが固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **中核/半周辺/周辺の三層構造的地位** | 星系は内政値を持つが、**世界分業の中での「地位」**（高利潤の中核産業か、低利潤の周辺産業か）を持たない |
| **不等価交換＝周辺の剰余が中核へ流出** | `MarketRules` は対等な需給均衡。**構造的に周辺が損し中核が得る**剰余移転の回路が無い |
| **ヘゲモニー循環（覇権国の生成→絶頂→衰退）** | `LeadingFaction` は瞬間の安定度トップ。**生産→商業→金融へ優位が移り衰退する長期サイクル**が無い |
| **半周辺の上昇/転落（地位の流動）** | 星系の地位は固定。**周辺→半周辺→中核へ這い上がる／中核が転落する**動学が無い |
| **単一世界システム＝分業は国境を越える一体** | `LogisticsRules` は自勢力内の連結のみ。**敵味方をまたいだ一つの分業体制**という視座が無い |
| **依存的発展＝周辺は中核に従属する形でしか発展しない** | 内政は自律的に安定化。**中核への従属が周辺の発展経路を歪める**回路が無い |

**結論**：近代世界システムは当プロジェクトに**「星系を世界分業の三層構造で位置づける」という構造的地位の軸**と、**①不等価交換による剰余移転 ②ヘゲモニー循環 ③半周辺の地位流動**という欠落軸を与える。これは既存の `GovernanceRules`（単一星系）・`MarketRules`（単一市場）・`LogisticsRules`（自勢力内）を**作り直さず、その上に「世界＝経済」という一段上の構造レイヤーを被せる**ものであり、銀河経済の構造論として `CampaignState` 統合層に最も効く。

## 1. 役に立つ視点（要約）

世界システム論を、**本システムに効く形**で1行ずつ（既存issue/モジュールへの共鳴を明記）：

1. **世界は単一の分業体制であり、富は層の間を不等価交換で移動する**。＝個々の星系内政（`GovernanceRules`）の総和でなく、**星系間の構造的関係**が国力を決める。`LogisticsRules`（版図一体化）の発展形。
2. **星系には「内政値」とは別の「構造的地位」（中核/半周辺/周辺）がある**。地位は高利潤産業・剰余の受け取り側か、低利潤・剰余の供出側かを決める。→ `Province`/`ResourceProductionRules` に**地位レイヤー**を被せる。
3. **覇権は永続しない＝生産優位→商業優位→金融優位と移り、やがて衰退する**。＝`CampaignState.LeadingFaction` を**瞬間値から長期サイクル**へ。王朝サイクル`DynastyRules`の経済版。
4. **半周辺は緩衝帯であり、上昇も転落もする流動層**。固定階層でなく、政策と条件で地位が動く。→ 地位流動が**戦略目標**になる（中核入りを目指す/周辺を維持して搾取する）。
5. **周辺の発展は中核への従属の形でしか進まない（依存的発展）**。搾取は短期に中核を富ませるが、周辺の不安定化・反システム運動を生む。→ `GovernanceRules`の反乱・`CampaignState.IsCollapsing` と接続＝搾取の長期コスト。
6. **（lore）「単一世界システム」「中核の覇権交代」「反システム運動」**＝世界観の構造的洞察。→ `DisclosureLedger`（FND-4）への lore データ入力（コード新設なし）。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`GovernanceRules`/`MarketRules`/`LogisticsRules`/`CampaignState` を作り直さない**。WSYS はそれらに**「世界分業の構造レイヤー」を被せ、接続する**だけ（additive）。マイクロ操作は増やさず、**高位の決断（産業政策・搾取の度合い）→エンジン駆動→創発帰結（地位流動・覇権交代）**で回す。

### ★★★ 最優先（真の欠落・世界システム論の signature）

#### WSYS 構造的地位（中核/半周辺/周辺）と不等価交換による剰余移転
- **構造的地位**：各星系（または各勢力）に `WorldRole{中核,半周辺,周辺}` を持たせる純データ。地位は**産業の利潤水準**（中核＝高付加価値・周辺＝低付加価値の一次産品）を表す。
- **不等価交換**：星系間の交易/版図内の産出配分で、**周辺の剰余が中核へ構造的に移転**する（対等な`MarketRules`均衡に、地位差に比例した剰余移転項を上乗せ）。
- 接続：新 `WorldSystemRules`/`WorldPosition`（純ロジック・test-first）。`LogisticsRules`（版図一体化）の上に分業構造を被せ、`ResourceProductionRules`/`GovernanceRules.OutputFactor` の産出を**地位で再配分**する。`MarketRules` は不変＝裁定均衡の上に構造移転を加える。

#### WSYS ヘゲモニー循環（覇権の生成→絶頂→衰退の長期サイクル）
- **覇権段階**：覇権国が**生産優位→商業優位→金融優位**と段階を移り、金融化＝衰退の兆候として次の覇権へ交代する長期サイクル。
- 接続：新 `HegemonyRules`/`HegemonyCycle`（純ロジック・test-first）。`CampaignState.LeadingFaction` を**瞬間の安定度トップから段階つき長期サイクル**へ拡張（既存 API は不変・上に被せる）。`DynastyRules`（王朝サイクル）の経済覇権版＝同型エンジンの再利用思想。

### ★★ 高（地位の流動・搾取の創発帰結）

#### WSYS 半周辺の上昇/転落（構造的地位の流動）
- 地位は固定でなく、**産業政策・蓄積・覇権の波及**で `周辺→半周辺→中核`／逆向きに動く。半周辺は緩衝帯（上昇も転落もする）。
- 接続：`WorldSystemRules` に地位遷移ルール（目標地位へ収束＝`GovernanceRules` の安定度収束と同流儀）。地位上昇を**戦略目標**化（高位の決断＝産業投資の方向）。`CampaignRules.Tick`（暦境界Tick）に相乗り。

#### WSYS 依存的発展と反システム運動（搾取の長期コスト）
- 周辺は中核への**従属の形でしか発展しない**（依存的発展）。搾取は短期に中核を富ませるが、周辺の**不安定化・反システム運動**（反乱圧力）を蓄積する。
- 接続：`WorldSystemRules` の搾取度→`GovernanceRules.RebelPressure`/`CampaignState.IsCollapsing` へ係数で波及。`JusticeRules`（不満な正義観）/`NonviolenceRules`（反システム運動）と共鳴。搾取の**創発的帰結**＝面白さの核。

### ★ 中（世界観lore）

#### WSYS（lore）世界システムの構造的洞察
- 「世界は単一の分業体制」「覇権は交代する」「周辺の発展は中核に従属する」「反システム運動」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への lore データ入力。世界観EPIC（啓蒙/秘史）に一貫。

### ❌ 不採用（重複・既存で十分・タイクン化・全部入り回避）

> 「全部入り」は失敗。世界システム論の**全構成要素を実装しない**。既存カバー分・粒度過剰は捨てる。

| 不採用 | 理由 |
|---|---|
| 単一市場の需給均衡・価格・裁定 | **`MarketRules`/`Market`（#179-182）＋SAW裁定が既にカバー**。WSYS は均衡の上に構造移転を被せるだけ |
| 通貨・為替・通貨改鋳 | **`FiscalRules.ExchangeRateFactor`＋SAW `CoinageRules` がカバー** |
| 版図の物流・連結・補給線 | **`LogisticsRules`（#844）/`SupplyRules`（#94）がカバー**。WSYS は一体化度の上に分業を被せ接続のみ |
| 階級別の税・再分配・格差 | **`RedistributionRules`/`CapitalRules`（ピケティ#917）がカバー**。WSYS は星系間（空間）の搾取に限定 |
| 覇権「国」の軍事興亡そのもの | **戦略レイヤー（`StrategyRules`/会戦）が既にカバー**。WSYS は経済覇権の段階循環のみ |
| 多段加工・商品ごとの細かい生産ツリー | **タイクン化回避**＝個別商品でなく地位（利潤水準）の抽象で扱う |
| 個々の企業/商人スケールの取引 | **FRM#1022・SAW ネームド商人がカバー**。WSYS は構造（マクロ）専任 |
