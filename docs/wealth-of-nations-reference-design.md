# アダム・スミス『国富論』参考設計（EPIC #WLT）

> 参照元：アダム・スミス『国富論』（An Inquiry into the Nature and Causes of the Wealth of Nations, 1776）。
> **分業・市場規模・比較優位・資本蓄積・独占の害**を軸に、国家の富がなぜ生まれ、なぜ衰えるかを論じた近代経済学の原典。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風星間国家戦略＋既存経済純ロジック層）にとって**役に立つ視点だけを抽出**し、EPIC `#WLT` として issue 化する提案。
> 著作権注意：固有名・文章・キャラクター・固有設定は流用せず、**経済メカニクス／世界観の構造パターンのみ**を参考にする。

## 0. なぜ『国富論』が本システムに役立つか

当プロジェクトは経済の**マクロ純ロジックを大量に保有**している：

| 既存（マクロ・抽象） | カバー範囲 |
|---|---|
| `MarketRules`/`Good`/`Market`（#179-182） | 需給均衡価格・生活水準→支持（「見えざる手」の基本形） |
| `ResourceProductionRules`（L-1 #93） | 星系類型（工業/農業/鉱業/居住）×安定度比例の産出 |
| `LogisticsRules`（#844） | 版図一体化度（`CohesionFactor`）・分断による国力損耗 |
| `FiscalRules`/`FiscalState`（#161/162） | 財政・税率・国債・為替 |
| `CommerceRaidingRules`（L-3 #95） | 通商破壊・補給線遮断 |
| `DiplomacyRules`/`DiplomacyState`（#189） | 外交状態・条約の枠組み |
| `ShipyardRules`（#884） | 生産力×安定度の産出係数 |
| `Province`/`GovernanceRules`（#109） | 安定度→産出係数（`OutputFactor`） |

**しかし、これらは「均衡した市場と安定した生産」の静態モデル**であり、国富論が固有に描く以下が**欠けている**：

| 国富論が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **分業と専門化→生産性の飛躍** | `SystemType` は類型を持つが「専門化の深さ」が無い。特化度が高まると産出が飛躍するループが無い |
| **市場規模が分業の深さを決める** | `LogisticsRules.CohesionFactor` は接続を評価するが「版図が広い＝専門化が深まる＝産出増」の因果がない |
| **比較優位と自由貿易の相互利得** | 各星系/勢力が得意産業に特化して貿易する利得が無い。保護関税（重商主義）の逆効果も無い |
| **資本蓄積→生産性の自己強化ループ** | `FiscalState` は収支を追うが「剰余→再投資→産出増→更なる剰余」のフィードバックが無い |
| **独占の社会損失** | `MarketRules` は競争均衡のみ。独占による価格釣り上げ・産出制限・死重損失が無い |

**結論**：国富論は当プロジェクトのマクロ経済に**「なぜ連結すると強くなるか」「なぜ孤立・保護・独占で衰えるか」の動的因果**を与える。
具体的には①専門化ループ ②市場規模制約 ③比較優位・自由貿易政策 ④資本蓄積サイクル ⑤独占害の5軸が欠落しており、これらが採用候補となる。

## 1. 役に立つ視点（要約）

国富論の世界観を、**本システムに効く形**で1行ずつ：

1. **分業で生産性は飛躍する**——単純作業の反復が熟練を生み、1人の仕事が10人分になる。星系が一産業に特化するほど産出係数が上がる。→ `ResourceProductionRules`（L-1）に専門化ボーナスの軸を足す。
2. **市場の広さが分業の深さを決める**（スミスの核心命題）——孤立した村は分業できない。銀河規模の市場があるから超深度の専門化が成立する。→ `LogisticsRules.CohesionFactor`（#844）を専門化上限の入力として使う。
3. **比較優位のある産業に特化して交易するだけで双方が得をする**——得意を交換するのが自由貿易の本質。保護関税は生産者を守るが消費者（星系全体）を貧しくする。→ `DiplomacyRules`（#189）の通商条約に実質を与え、重商主義政策の逆効果を可視化する。
4. **剰余を消費せず資本に変えることで未来の産出が増える**——資本（機械・設備・艦艇建造力）の蓄積が長期成長の源泉。→ `FiscalState`（#161）の黒字が再投資に回り生産性が上がるサイクルを作る。
5. **独占は競争を殺し社会を貧しくする**——商社・独占特許・国家管理の弊害。フェザーン的商社国家が銀河を独占すると価格は上がり産出は下がる。→ `MarketRules`（#179）に独占プレミアム＆死重損失を足す。
6. **「見えざる手」は失敗する——公共財・外部性の問題**——灯台（国防/治安）は市場が供給しない。基礎インフラは国家の責務。→ `GovernanceRules`（#109）と`Province`の安定度支出正当化のlore。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`MarketRules`/`ResourceProductionRules`/`LogisticsRules`/`FiscalRules` を作り直さない**。WLT はそれらに**欠落軸を足し、接続する**だけ（additive）。
> タイクン化回避：建設マイクロ・マルチ段階製造ツリー・個別工場管理は持たない。高位の決断→エンジン駆動→創発帰結。

### ★★★ 最優先（真の欠落・国富論の signature）

#### WLT 分業と専門化（`SpecializationRules` 新設）
- **専門化度（0..1）** を`Province`/星系に追加。単一産業に集中するほど度が上昇→ `ResourceProductionRules.Produce` の産出係数にボーナス乗算。
- 複数産業を持つ（多角化）と度が薄まる——「何でもやろうとする星系は何も得意でない」。
- 専門化度の変化：投資と時間で上昇、孤立（補給切れ・封鎖）で減衰。
- 接続：`Province`×`ResourceProductionRules`×`GovernanceRules.OutputFactor`。新 `SpecializationRules`（純ロジック・test-first）。

#### WLT 市場規模が専門化を制約する（`LogisticsRules.CohesionFactor` × 専門化上限）
- スミスの核心命題を純ロジックで実装：**専門化度の理論上限 = f(CohesionFactor)**。
- 孤立した星系はローカル需要しかないので深く特化できない。版図が広がり接続が良くなるほど上限が上がる。
- これにより「征服・回廊確保→市場統合→分業深化→産出増」の戦略的意味が生まれる（軍事と経済の連結）。
- 接続：WLT-1 × `LogisticsRules`。

### ★★ 高（自由貿易政策と資本サイクル）

#### WLT 比較優位と自由貿易vs重商主義（`TradeGainRules` 新設）
- **比較優位**：各星系が機会費用の小さい産業に特化する利得を計算する `TradeGainRules`（純ロジック）。
- **政策オプション**：
  - 自由貿易（関税0）→ 交易利得最大・双方の総産出↑。
  - 重商主義（保護関税）→ 短期に関税収入（`FiscalRules`）を得るが交易量↓→分業↓→長期で`LogisticsRules.CohesionFactor`低下→専門化上限↓。
- **重商主義の罠**（ブーメラン効果）：プレイヤーが貿易相手を関税で守れば、相手も関税で報復→全体産出がジリジリ下がる負のスパイラル。
- 接続：`DiplomacyRules`（通商協定/貿易戦争）×`FiscalRules`（関税収入）×WLT-1/2。

#### WLT 資本蓄積と生産性再投資サイクル（`CapitalAccumulationRules` 新設）
- `CapitalAccumulationRules`（純ロジック）：勢力/星系が財政黒字（`FiscalState.PrimaryBalance`>0）のとき一部を「資本ストック」として積み上げ、産出係数を強化する。
- **減耗**：再投資が止まると資本が老朽化→産出係数が落ちる（戦時の消耗・占領でリセット）。
- **収穫逓減**：資本が増えるにつれて限界ボーナスが小さくなる（過剰投資の無意味化・タイクン化回避）。
- 接続：`FiscalState`×`ResourceProductionRules`×`ShipyardRules`×WLT-1。

### ★ 中（独占の害）

#### WLT 独占の社会損失と競争政策（`MonopolyRules` 新設）
- `MonopolyRules`（純ロジック）：ある勢力が特定商品の供給を独占すると価格上昇・産出制限・「死重損失（deadweight loss）」が発生する。
- **独占判定**：`GalaxyMap` 上で、接続された星系群の特定`GoodType`の供給シェアが閾値を超えたら独占状態。
- **独占プレミアム**：`MarketRules.ClearingPrice` に割増乗算→生活水準(`SoLToSupport`)経由で民心悪化。
- **競争政策**（独占解体・自由化）→ プレミアム解除・産出回復。フェザーン的商社国家のjitter要因に。
- 接続：`MarketRules`×`DiplomacyRules`×`LogisticsRules`。

### ❌ 不採用（重複・既存で十分・マルクスへ移管）

| 不採用 | 理由 |
|---|---|
| 労働価値論（労働が価値の源泉） | マルクス『資本論』が別途バックログにあり範囲を移管する |
| 通貨・金本位・品位 | SAW-1/2（`CoinageRules`）が既にカバー。WLT から重複新設しない |
| 多段加工・製造BOM | `SCM#982` がカバー。WLT は産出係数ボーナスのみ |
| 個別工場の工程管理・生産スケジュール | タイクン化回避。係数として背景的に効かせる |
| 国際収支・経常勘定の詳細会計 | `FiscalRules` に統合済み。新モジュール不要 |
| 農業・重農主義（ケネー批判） | `SystemType.農業` が既にある。理論の内部事情は不要 |
| 人口と労働力市場の詳細 | `DemographicsRules`（#153）がカバー |

### 推奨着手順
`WLT-1`（分業/専門化＝全ての土台）→ `WLT-2`（市場規模制約＝WLT-1に上限を与える）→ `WLT-3`（貿易政策＝WLT-1/2の帰結として政策選択肢を作る）→ `WLT-4`（資本蓄積＝剰余の使いみちを財政に接続）→ `WLT-5`（独占＝市場支配力の失敗ケース）。

> いずれも既存マクロ経済を**後退させず接続**する additive 設計。`LogisticsRules`（版図連結）・`DiplomacyRules`（通商政策）・`FiscalRules`（財政）の三軸が軍事戦略と経済の接合点として機能する。
