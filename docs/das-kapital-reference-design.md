# 資本論 参考設計（EPIC #CAPI）

> 参照元：カール・マルクス『資本論（Das Kapital）』。資本主義経済の**運動法則**を解剖する経済学批判。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略＋既に巨大な経済純ロジック層）にとって**役に立つ視点**だけを抽出し、EPIC `#CAPI` として issue 化する提案。
> 着眼点：**恐慌の周期性・資本の集積/集中・階級の生成**（共産勢力#17／Pop階級#110 の理論的裏打ち）。
> 著作権注意：本作は学術書だが、**特定の文章・図表・固有の言い回しは流用せず、経済の構造パターン／運動法則のメカニクスのみ**を参考にする。

## 0. なぜ「資本論」が本システムに役立つか

当プロジェクトは経済・階級・格差の**マクロ純ロジックを大量に保有**している（[CLAUDE.md] 参照）：

| 既存（カバー範囲） | 何を見ているか |
|---|---|
| `CapitalRules`/`CapitalState`（ピケティ #917） | **r>g の資本集中ドリフト**・累進資本課税・格差→反乱（`InequalityUnrest`） |
| `RedistributionRules`/`FiscalClass`/`TaxStructure`（#162/163） | 富裕/中間/貧困の税負担・累進/逆進・階級対立（`ClassTension`）・階級別支持 |
| `MarketRules`/`Good`/`Market`（#179-182） | 需給均衡価格・生活水準（SoL）→支持（**「経済が革命を生む」**） |
| `StockMarketRules`/`Company`（#185） | 株価・配当・**暴落リスク**（`CrashRisk`） |
| `BankRules`/`Bank`（#186） | 信用創造・取付け・債務超過 |
| `FiscalRules`/`FiscalState`（#161/162） | 国債/金利/為替・債務スパイラル |
| `DynastyRules`/`Regime`（#867） | 腐敗→正統性低下→天命喪失→易姓革命（体制転換の周期） |
| `GovernanceRules`/`Province`（#109） | 安定度・反乱圧力・思想一致 |

**しかし、これらは「格差は溜まる／市場は均衡へ向かう／暴落はランダムに起きる」という静的・外生的な扱い**であり、資本論が固有に描く以下が**欠けている**：

| 資本論が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **剰余価値＝搾取率（労働が生む価値と賃金の差）** | `MarketRules` は需給で価格を出すが、**価値が誰の労働から/どれだけ吸い上げられるか**という生産点の搾取率が無い。富の起点が外生 |
| **利潤率の傾向的低下→過剰蓄積→恐慌の<u>周期性</u>** | `StockMarketRules.CrashRisk` は確率イベント。**蓄積が進むほど内生的に利潤率が下がり、周期的に恐慌が来る**という景気循環エンジンが無い |
| **資本の集積（再投資で大きく）と集中（吸収合併で少数へ）の区別** | `CapitalRules.ConcentrationDrift` は r>g の一様ドリフト。**自己蓄積（集積）と他者吸収（集中＝大資本が小資本を呑む）の二経路**が無い |
| **産業予備軍（失業者プール）＝景気で伸縮し賃金を抑える** | `DemographicsRules` は人口三コホート。**好況で雇われ不況で吐き出される失業バッファが賃金・支持を規律する**回路が無い |
| **階級の<u>生成</u>＝経済構造から階級が立ち上がり意識化→組織化→革命** | `FiscalClass` は固定3層。`InequalityUnrest` は格差→反乱の即値。**搾取の累積が階級意識を醸成し、組織化を経て体制転換に至るプロセス**が無い（#17 共産勢力／#110 Pop階級の理論的裏打ちが空白） |
| **本源的蓄積（囲い込み／収奪が資本の出発点）** | 占領は `OnOccupied` で統合度リセットのみ。**征服・収奪が生産手段を私的資本へ転化する起点**という視点が無い |
| **商品物神性／疎外（lore）** | 世界観の開示データに「物が人を支配する」近代の主題が無い |

**結論**：資本論は当プロジェクトの「**格差は溜まる**」静的モデルに、**①剰余価値＝搾取の生産点、②利潤率低下→恐慌の内生的周期、③集積と集中の二経路、④産業予備軍、⑤階級の生成プロセス（意識化→組織化→革命）**という運動法則を与える。これにより**共産勢力#17 と Pop階級#110 が「なぜ生まれ、なぜ革命に向かうか」の理論的背骨**を得る。既存（ピケティ/再分配/市場/株式）は**作り直さず接続**＝additive。

## 1. 役に立つ視点（要約）

資本論の運動法則を、**本システムに効く形**で1行ずつ：

1. **富は生産点の搾取から生まれる（剰余価値）**。市場の需給より手前で「労働が生んだ価値−賃金」が資本に吸われる。→ `MarketRules`/`CapitalState` に**富の起点（搾取率）**を与え、外生だった蓄積を内生化。
2. **蓄積は内生的に恐慌を生む（利潤率の傾向的低下）**。蓄積が進むほど利潤率が下がり、過剰蓄積が周期的に弾ける。→ `StockMarketRules.CrashRisk` のランダム暴落を**景気循環エンジン**に格上げ（恐慌の周期性＝着眼点の核）。
3. **資本は集積し集中する（二経路）**。自己再投資で大きくなり（集積）、恐慌のたびに大資本が小資本を呑む（集中）。→ `CapitalRules.ConcentrationDrift` を**二経路**へ分解（集積/集中＝着眼点の核）。
4. **失業の予備軍が賃金を規律する**。好況で雇い不況で吐き出すバッファが賃金上昇を抑え搾取を維持。→ `DemographicsRules`×`GovernanceRules` に景気連動の失業プール。
5. **階級は経済構造から生成し、意識化→組織化→革命へ進む**。格差の即時反乱でなく、搾取の累積が階級意識を醸成しストライキ/政党化を経て体制転換に至る。→ #17 共産勢力／#110 Pop階級／`CoupRules`(革命)/`PartyRules`(階級政党) の発生源（階級の生成＝着眼点の核）。
6. **資本の出発点は収奪（本源的蓄積）**。征服・囲い込みが共有地を私的資本へ転化する。→ 占領 `OnOccupied`／入植 `ColonizationRules` に収奪→資本化の一回的ショック。
7. **（lore）商品物神性・疎外＝近代の主題**。→ `DisclosureLedger` への lore データ入力（コード新設なし）。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**ピケティ`CapitalRules`#917／再分配`RedistributionRules`#162／市場`MarketRules`#179／株式`StockMarketRules`#185／銀行`BankRules`#186 を作り直さない**。資本論はそれらに**運動法則（搾取・周期・集中・予備軍・階級生成）を足し、接続する**だけ（additive）。タイクン化回避＝高位の決断（賃金水準/再投資率/弾圧 等）→エンジン→創発帰結（恐慌・革命）。

### ★★★ 最優先（真の欠落・資本論の signature・着眼点に直結）

#### CAPI 剰余価値と搾取率（富の起点を内生化）
- **生産点モデル**：星系/勢力の産出から「労働が生む価値」を立て、**賃金分を控除した剰余＝資本の取り分**を出す。搾取率（剰余/賃金）が高いほど短期の蓄積は速いが、生活水準（SoL）が下がり階級意識を醸成。
- 接続：新 `SurplusValueRules`/`ProductionRelation`（純ロジック・test-first）。`MarketRules.StandardOfLiving`（賃金側）×`CapitalState`（資本側）の**橋**。`GovernanceRules.OutputFactor` の産出を価値配分へ分解（既存産出は非破壊・read-only）。

#### CAPI 利潤率の傾向的低下と恐慌の周期性（景気循環エンジン）
- **内生周期**：蓄積（資本の有機的構成の高度化）が進むほど利潤率が逓減→過剰蓄積→**周期的に恐慌**（生産破壊→出直し）。確率の単発暴落でなく、**蓄積状態から決まる周期**。
- 接続：新 `AccumulationCycleRules`/`BusinessCycle`（純ロジック・test-first）。`StockMarketRules.CrashRisk` を**内生周期の出力**として駆動（既存 CrashRisk は接続＝置換しない）。`FiscalRules` の債務・`MarketRules` の価格へ波及。**恐慌の周期性＝着眼点①**。

#### CAPI 資本の集積と集中（二経路の分解）
- **集積**＝剰余の再投資で資本が自己増殖。**集中**＝恐慌のたびに大資本が破綻小資本を吸収（少数へ）。両者でジニ的集中が進み、独占化が利潤率低下を一時緩和するが格差は極大化。
- 接続：`CapitalRules.ConcentrationDrift`（ピケティ#917 の一様ドリフト）を**集積/集中の二経路**へ拡張（既存 API は維持し、二経路を加算する additive）。CAPI 恐慌（集中の契機）と直結。**資本の集積/集中＝着眼点②**。

### ★★ 高（階級の生成・予備軍）

#### CAPI 産業予備軍（失業プールが賃金を規律）
- **景気連動の失業バッファ**：好況で雇用吸収し不況で放出。予備軍が厚いほど賃金上昇を抑え搾取率を維持＝恐慌が搾取を回復させる皮肉。
- 接続：新 `LaborReserveRules`（純ロジック・test-first）。`DemographicsRules.Population`（生産年齢コホート）×`AccumulationCycleRules`（景気）×賃金（SurplusValue）。失業→`GovernanceRules` 安定度/支持へ。

#### CAPI 階級の生成と階級意識（搾取→意識化→組織化→革命）
- **生成プロセス**：搾取率・SoL低下・予備軍の累積が**階級意識**を醸成（即時反乱でなく蓄積変数）→閾値で**組織化**（ストライキ/階級政党）→さらに進めば**革命（体制転換）**。`InequalityUnrest` の即値を**段階プロセス**へ。
- 接続：新 `ClassFormationRules`/`ClassConsciousness`（純ロジック・test-first）。**#17 共産勢力／#110 Pop階級の発生源**。`PartyRules`（階級政党）/`CoupRules`(革命)/`GovernanceRules`(反乱) へ出力。`FiscalClass` の固定3層を**動的に立ち上げる**（既存3層は read-only 参照、生成は別変数）。**階級の生成＝着眼点③**。

### ★ 中（本源的蓄積・lore）

#### CAPI 本源的蓄積（収奪→資本化の起点）
- 征服・囲い込みが共有資源を私的資本へ転化する**一回的ショック**：占領/入植で資本ストックが立ち上がるが、被収奪側に階級的怨恨（後の意識化の種）を残す。
- 接続：`GovernanceRules.OnOccupied`／`ColonizationRules.Establish` に**収奪→`CapitalState` 初期化＋階級意識の種**を付与（既存フローへ薄く接続・新エンジンは作らない）。

#### CAPI（lore）商品物神性・疎外の開示データ
- 「物が人を支配する（物神性）」「労働の疎外」「資本の自己増殖運動」＝近代の主題。世界観EPIC（啓蒙/ニーチェ/秘史）に接続。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への**lore データ入力**。

### ❌ 不採用（重複・既存で十分・additive で接続のみ）

| 不採用 | 理由 |
|---|---|
| 格差の集中ドリフト（r>g 一様） | **`CapitalRules`/`CapitalState`#917 が既にカバー**。CAPI は集積/集中の二経路を**足すだけ**（作り直さない） |
| 階級別の税負担・累進/逆進・階級対立 | **`RedistributionRules`/`FiscalClass`/`TaxStructure`#162 がカバー**。CAPI は固定3層を読むだけ |
| 需給均衡価格・生活水準→支持 | **`MarketRules`#179 がカバー**（「経済が革命を生む」）。CAPI は生産点の搾取率を上流に足すのみ |
| 株価暴落そのもの | **`StockMarketRules.CrashRisk`#185 がカバー**。CAPI は内生周期で**駆動する**だけ（置換しない） |
| 信用創造・取付け・銀行恐慌 | **`BankRules`#186 がカバー**。恐慌の金融面はここへ接続 |
| 体制転換の周期（腐敗→正統性→易姓革命） | **`DynastyRules`/`Regime`#867 がカバー**。革命の出力はここ/`CoupRules` へ流す |
| 通貨/多段生産ツリー/個別企業会計の精密化 | タイクン化＝マイクロ操作の増殖。CAPI は集約変数（搾取率/利潤率/予備軍/階級意識）に留める |

### 推奨着手順
`CAPI-1`（剰余価値＝富の起点を内生化＝全ての上流）→ `CAPI-2`（利潤率低下→恐慌の周期＝着眼点①の核）→ `CAPI-3`（集積/集中＝着眼点②、恐慌と連動）→ `CAPI-4`（産業予備軍＝賃金規律）→ `CAPI-5`（階級の生成＝着眼点③、#17/#110 の背骨）→ `CAPI-6`（本源的蓄積＝起点接続）→ `CAPI-7`（lore）。

> いずれも既存マクロ経済・階級ロジックを**後退させず接続**する additive 設計。**搾取率→利潤率低下→恐慌→集中→予備軍→階級意識→革命**という一本の運動法則で、#17 共産勢力と #110 Pop階級に「なぜ生まれ革命に向かうか」の理論的背骨を与える。
