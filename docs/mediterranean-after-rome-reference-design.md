# ローマ亡き後の地中海世界 参考設計（EPIC #PIRA）

> 参照元：塩野七生『ローマ亡き後の地中海世界』。**ローマという普遍的秩序が消えた後**の地中海が、海賊（非国家アクター）の跋扈する**無秩序海域**へ転落し、通商が萎縮し、沿岸が略奪と人身売買に怯え、やがて私掠（国家公認の海賊）と海賊討伐の集団行動でゆるやかに再秩序化されていく——**「秩序の真空に何が湧くか」**を海賊を主役に描いた歴史叙述。
> 本ドキュメントは、当プロジェクト（Ginei＝銀英伝風の星間国家戦略・銀河グラフ＝海図／回廊＝海上交通路）にとって**役に立つ視点**だけを抽出し、EPIC `#PIRA` として issue 化する提案。
> **着眼点**：海賊と通商・無秩序海域の経済（通商破壊 #95 の海賊版＝**非国家アクター**）。
> **著作権注意**：固有名・文章・キャラクター・固有の歴史記述・図表は流用せず、**メカニクス／世界観の構造パターンのみ**を参考にする。

## 0. なぜ「ローマ亡き後の地中海世界」が本システムに役立つか

当プロジェクトは、海図に相当する盤面（銀河グラフ）と、**国家どうしの**海戦・通商・制海の周辺要素を**既に大量に保有**している（[CLAUDE.md] 参照）：

| 既存（カバー範囲） | 該当モジュール |
|---|---|
| 海図＝ノード/エッジ・経路探索 | `GalaxyMap`/`Corridor`/`GalaxyPathfinder` |
| **通商破壊（国家の襲撃 vs 護衛・補給遮断）** | `CommerceRaidingRules`（L-3 #95）＝raider/escort/船団の数値解決 |
| 補給線（補給源→所有回廊で到達・遮断で枯渇） | `SupplyRules`（L-2 #94） |
| 資源生産・備蓄・通商の流れ | `ResourceProductionRules`/`ResourceStockpile`（L-1 #93） |
| 制海権グラデーション・チョークポイント・シーパワー国力（**国家視点**） | `SeaControlRules`/`ChokepointRules`/`SeaPowerRules`（MHAN #MHAN・国家海軍の理論） |
| 敵対判定・外交状態（**国家間**） | `FactionRelations`/`DiplomacyRules`（#189 DIP-1） |
| 諜報（情報収集・破壊工作） | `EspionageRules`（決定論 roll） |
| 内政・占領統合・反乱圧力 | `GovernanceRules`/`Province`（#109/#767） |
| 捕虜化・解放・処断・登用 | `CaptivityRules`（LIFE-4 #154・**人物単位**） |
| 私的暴力主体・レント分配で暴力を抑える秩序類型 | （backlog ノース『暴力と社会秩序』＝未） |

**しかし、これらはすべて「国家という主体」を前提にしたマクロ均衡・国家間動学**であり、本作が固有に描く**「ローマ亡き後＝秩序の真空」に湧く非国家アクター（海賊）**の以下が**欠けている**：

| 本作が固有に持つ視点 | 当プロジェクトでの欠落 |
|---|---|
| **海賊＝非国家アクター**が、勢力の支配が及ばぬ海域に**自然発生**する | `CommerceRaidingRules` は**国家の艦隊**が襲う。`FleetRegistry`/`StrategicFleet` は勢力に属す。**どの勢力にも属さない襲撃主体**が盤面に存在しない |
| **秩序の真空が海賊を生む**（制海権・統治の空白＝海賊の温床） | 制海権 `SeaControlRules`・統治 `GovernanceRules` はあるが、**「空白度→海賊の湧き」という生成回路**が無い。空白は「誰も得しない」だけで脅威を生まない |
| **略奪・身代金（ランサム）経済**＝海賊は壊すのでなく**奪って稼ぐ**（捕虜の身代金/人身売買） | `CommerceRaidingRules` は「撃破＝補給0」の破壊モデル。**奪取して海賊側に富が移る**収奪経済・**身代金収入**が無い。`CaptivityRules` は人物単位で経済化されていない |
| **沿岸襲撃（コースト・レイディング）＝海から陸を脅かす** | 通商破壊は回廊（海路）上の船団のみ。**沿岸星系/惑星が海賊に略奪され安定度・人口が削られる**陸への波及が無い |
| **海賊討伐＝集団行動問題**（誰もが望むが誰も費用を払いたがらない公共財） | 敵対は二国間。**複数勢力が費用を出し合って共通の脅威を討つ／フリーライドする**多者協調の動学が無い（静観 #820 は会戦内・国家間の集団行動でない） |
| **私掠＝国家公認の海賊**（海賊を取り込み敵に差し向ける／非対称戦の外注） | `CommerceRaidingRules` は正規艦隊。**勢力が非国家アクターに認可を与えて敵通商を襲わせる**（黙認・スポンサー）回路が無い |
| **海賊国家化＝無法者が拠点を持ち準国家へ**（バルバリア海賊型） | 海賊は流動的脅威のまま。**拠点を固定し徴税・常設化して準勢力へ昇格**する経路が無い |
| **（lore）普遍的秩序の喪失と再秩序化** | 王道路線・崩壊系世界観はあるが、「**ローマ＝秩序が消えると海が荒れ、再び秩序が組まれるまで人は怯える**」という秩序の真空テーマの開示データが無い |

**結論**：本作は当プロジェクトの**国家中心の海洋マクロ層**に、**①秩序の真空に湧く非国家アクター（海賊）の生成 ②略奪・身代金の収奪経済 ③海賊討伐＝多者集団行動問題 ④私掠と海賊国家化（非対称戦の外注と昇格）**という4本柱の欠落軸を与える。いずれも既存の `CommerceRaidingRules`/`SeaControlRules`/`GovernanceRules`/`CaptivityRules`/`FactionRelations` に**接続するだけ**で、**「強い国どうしの戦い」から一段降りて、秩序の空白そのものが脅威になる**創発層を完成させる。MHAN（国家の制海権）の**裏面＝制海できない海に何が起きるか**を埋める。

## 1. 役に立つ視点（要約）

本作の歴史叙述を、**本システムに効く形**で1行ずつ（既存issue/モジュールへの共鳴を明記）：

1. **普遍的秩序（ローマ）が消えると海は無法地帯になる**＝制海権・統治の空白そのものが脅威を生む。→ MHAN の制海権 `SeaControlRules`・`GovernanceRules` の**裏返し**＝空白度を入力に非国家脅威を創発させる。
2. **海賊は壊すためでなく奪うために襲う**＝略奪と身代金で稼ぐ収奪経済。→ `CommerceRaidingRules`（破壊モデル #95）に**富の移転と身代金収入**という収奪面を足す。`CaptivityRules`（#154）に経済の歯を入れる。
3. **海から陸が脅かされる**＝沿岸襲撃が交易だけでなく人口・安定度を削る。→ `GovernanceRules`/`Province`（#109）へ**海賊の陸への波及**を接続。`SupplyRules`（#94）の遮断とは別系統の略奪。
4. **海賊討伐は典型的な集団行動問題**＝誰もが平穏を望むが討伐費用は払いたがらない（フリーライド）。→ backlog オルソン『集合行為論』・静観 #820 の**国家間版**。`FactionRelations`×多者協調の純ロジック。
5. **海賊は外注できる**＝私掠（公認の海賊）で敵通商を非対称に襲わせる。→ `DiplomacyRules`（#189）×`CommerceRaidingRules`＝**勢力が非国家アクターをスポンサーする**戦わぬ経済戦（通商破壊 #95・制裁 PRC-8 と三層）。
6. **無法者は準国家へ昇格しうる**＝拠点を固定し徴税して海賊国家になる（やがて討伐対象 or 取り込み対象）。→ `Polity`/`ConsentRules`・`Organization`（#812 制度化）の**最小自治体としての海賊拠点**。
7. **（lore）秩序の真空は人を怯えさせ、再び秩序が組まれるまで終わらない**。→ `DisclosureLedger`（FND-4）への lore データ入力（コード新設なし）。崩壊系世界観・王道路線（武力でなく秩序で海を鎮める）と共鳴。

## 2. 取り入れるべきメカニクス（優先度つき・既存への接続）

> 大原則：**`CommerceRaidingRules`/`SeaControlRules`/`SupplyRules`/`GovernanceRules`/`CaptivityRules`/`FactionRelations` を作り直さない**。PIRA はそれらに**欠落軸（非国家アクター）を足し、接続する**だけ（additive）。タイクン化回避＝高位の決断（討伐に費用を出す/私掠を認可する/拠点を叩く）→エンジン→創発帰結（海賊の増減・身代金の流出・フリーライドの蔓延・海賊国家の興亡）。海賊個体のミクロ操作は作らない。

### ★★★ 最優先（真の欠落・本作の signature）

#### PIRA 海賊の発生・消長（秩序の真空→非国家アクターの湧き）
- **海賊圧（piracy pressure）**：回廊網/海域ごとに、**制海権の空白度**（`SeaControlRules` の無支配度）・**沿岸統治の弱さ**（`GovernanceRules` 低安定度）・**通商の濃さ**（獲物の多さ）から、海賊勢力の**自然発生・成長・消滅**を導く純関数。制海すれば減り、空白が続けば自己増殖する（収穫逓減の獲物＝荒らしすぎると枯れる自己抑制）。
- **どの勢力にも属さない襲撃主体**＝`FactionRelations` で全勢力に敵対する中立アクターとして抽象（個別の `StrategicFleet` を量産しない＝海域の状態量として持つ＝終盤ラグ回避 PERF #1117）。
- 接続：`SeaControlRules`/`SeaControlState`（MHAN・空白度）×`GovernanceRules`（沿岸安定）を read-only で読む新 `PiracyRules`/`PiracyState`（純ロジック・test-first）。`Corridor.type`（要衝/通商）で獲物の濃さを重み付け。
- 完了条件にできる単位：空白度→海賊圧の収束（`GovernanceRules` 流儀の目標値収束）・制海回復で減衰・獲物枯渇の自己抑制テスト・盤面接続（観測層追従）。

#### PIRA 略奪・身代金の収奪経済（壊すのでなく奪う＝富の移転）
- **収奪モデル**：海賊が船団/沿岸を襲うと、撃破（#95）と違い**富が海賊側へ移転**する＝被害勢力の `ResourceStockpile` から流出し海賊圧の蓄財（再投資＝さらなる成長）になる。捕虜（`CaptivityRules`）には**身代金（ランサム）**が付き、払えば解放・払えねば人身売買で海賊の収入になる。
- 接続：`CommerceRaidingRules`（#95 迎撃解決）に**収奪分岐**を足す薄い拡張（破壊モデルは後方互換で残す）＝新 `PlunderRules`（純ロジック・test-first）が「撃破/拿捕（奪取）/身代金」を解決し、`ResourceStockpile`・`CaptivityRules` へ read-only で接続。
- 完了条件にできる単位：奪取で攻守の富が保存的に移転するテスト・身代金の支払い/不払い分岐・護衛で守り切るテスト（#95 と整合）。

### ★★ 高（無秩序海域の社会動学を足す）

#### PIRA 沿岸襲撃（海から陸への略奪＝安定度・人口への波及）
- **沿岸の脆弱性**：海賊圧の高い海域に面した星系/惑星 `Province` は、海路の船団被害だけでなく**沿岸が略奪される**＝安定度低下・人口減・統合度後退。守るには制海（MHAN）か沿岸防衛が要る。「海が荒れると陸が痩せる」。
- 接続：`PiracyRules`（海賊圧）×`GovernanceRules`/`Province`（#109 安定度・人口）＝新 `CoastalRaidRules`（純ロジック・test-first）。`SupplyRules`（#94 補給遮断）とは別系統の**直接略奪**。
- 完了条件にできる単位：海賊圧→沿岸 `Province` の安定度/人口ペナルティ・制海回復で停止・内陸星系は無影響のテスト。

#### PIRA 海賊討伐の集団行動問題（公共財・フリーライダー）
- **討伐の公共財性**：海賊鎮圧は全勢力が便益を受けるが**費用を出すのは討伐に参加した勢力だけ**＝強者にタダ乗りするフリーライダーが湧き、誰も十分に出さず海賊が栄える（or 覇権国が単独供給する）。共同討伐が組めれば一気に鎮圧。
- 接続：backlog オルソン『集合行為論』・静観 #820（フリーライド）の**国家間版**＝`FactionRelations`/`DiplomacyRules`（#189）を read-only で読む新 `AntiPiracyRules`（純ロジック・test-first）＝各勢力の拠出と海賊圧の減衰、フリーライド均衡・覇権安定（単独供給）の判定。
- 完了条件にできる単位：拠出合計→鎮圧の閾値・フリーライダー均衡（誰も出さず脅威残存）・覇権国単独供給のテスト・盤面接続（観測層追従）。

#### PIRA 私掠（国家公認の海賊＝非対称戦の外注）
- **私掠認可**：勢力が海賊（非国家アクター）に**認可（黙認/スポンサー）**を与え、敵勢力の通商へ差し向ける＝自国艦隊を出さずに敵を蝕む。平時は敵対のはずの海賊が、認可勢力には牙を向けない（が制御は不完全＝寝返り・暴走リスク）。
- 接続：`DiplomacyRules`（#189 外交状態）×`PiracyRules`（海賊圧）×`CommerceRaidingRules`（#95）＝新 `PrivateeringRules`（純ロジック・test-first）＝認可で海賊圧を敵海域へ誘導、`FactionRelations.IsHostile` の海賊に対する向きを認可勢力だけ反転。通商破壊 #95・制裁 PRC-8 と並ぶ**第三の経済戦**。
- 完了条件にできる単位：認可で敵通商被害が増え自通商は守られる・制御失敗で認可勢力にも牙を向く（暴走）・認可解除のテスト。

### ★ 中（海賊国家化・lore）

#### PIRA 海賊国家化（無法者が拠点を持ち準国家へ昇格）
- **拠点の固定化**：海賊圧が高止まりした海域で、海賊が**拠点（巣）を固定**し徴税・常設化すると、流動的脅威から**準勢力（海賊国家）**へ昇格する＝討伐対象 or 外交・取り込み（招安）対象になる。拠点を叩けば瓦解、放置すれば制度化（`Organization` #812）して恒久化。
- 接続：`PiracyRules`（海賊圧）×`Polity`/`ConsentRules`（最小自治体）×`Organization`（#812 制度化）＝新 `PirateHavenRules`（純ロジック・test-first）。招安（賊→取り込み）は backlog『水滸伝』と共鳴。拠点星系は既存 `StarSystem`/`Province` を流用（新レジストリ非新設）。
- 完了条件にできる単位：海賊圧→拠点固定の閾値・拠点制圧で瓦解・放置で制度化（恒久勢力）のテスト・盤面接続（観測層追従）。

#### PIRA（lore）秩序の真空の世界観開示データ
- 「普遍的秩序が消えると海が荒れる」「海賊は秩序の不在が生む」「奪う経済は富を生まず移すだけ」「討伐は皆が望み誰も払わない」「無法者も秩序を欲して国になる」。
- 接続：**コード新設せず** `DisclosureLedger`（FND-4）への lore データ入力。CCX-6（世界観 codex 退避）方針に一貫。崩壊系世界観・王道路線（武力でなく秩序で海を鎮める）・フェザーン #160（交易の保護者）と共鳴。

### 推奨着手順
`PIRA-1`（海賊の発生・消長＝最も固有で全柱の土台・空白度を読む生成エンジン）→ `PIRA-2`（略奪・身代金＝海賊の signature「奪う経済」・#95 に収奪面）→ `PIRA-3`（沿岸襲撃＝海から陸への波及・脅威が実害になる）→ `PIRA-4`（討伐の集団行動問題＝なぜ海賊が栄えるかの社会動学）→ `PIRA-5`（私掠＝海賊を武器化する高位の決断）→ `PIRA-6`（海賊国家化＝無法者の昇格・招安）→ `PIRA-7`（lore・最後にまとめて開示データ化）。

> いずれも既存の海洋・経済・社会マクロ（`CommerceRaidingRules`/`SeaControlRules`/`SupplyRules`/`GovernanceRules`/`CaptivityRules`/`FactionRelations`）を**後退させず接続**する additive 設計。三本柱＝**非国家アクターの生成（PIRA-1）／奪う経済＝略奪・身代金・沿岸襲撃（PIRA-2-3）／無秩序海域の社会動学＝討伐の集団行動・私掠・海賊国家化（PIRA-4-6）**。MHAN（国家の制海権）の**裏面＝制海できない海に何が湧くか**を埋め、王道路線（武力でなく秩序で海を鎮める）・フェザーン #160（交易の保護者）に効く。海賊個体のミクロ操作を作らず**海域の状態量**として抽象する＝終盤ラグ回避（PERF #1117）。
